## ① 腸活えのきスープ

**ソース:** [Instagram（Pinterest 経由）](https://uk.pinterest.com/pin/on-instagram-kana0716t-225-1-12-in-2024--843862048962140522/)
**YouTube:** —
**食数:** 2〜2.5食分 | **kcal/食:** ~230 kcal

### 材料

| 食材 | 分量 |
|------|------|
| えのき | 1袋 |
| もやし | 1/2袋 |
| 長ネギ | 1本 |
| 絹豆腐 | 180g |
| 卵 | 2個 |
| 水 | 500ml |
| 鶏ガラスープの素 | 小さじ2 |
| 麺つゆ（2倍濃縮） | 大さじ2 ※醤油大さじ1+みりん大さじ0.5+ほんだし少々で代替可 |
| 生姜 | 小さじ1 |
| 片栗粉 | 小さじ2 |
| 水（とろみ用） | 大さじ2 |
| ごま油 | 小さじ1 |
| 塩・黒胡椒 | 適量 |

### 作り方

1. えのきを根元を切って3等分、ネギを小口切り
2. 鍋に水・鶏ガラ・麺つゆ・生姜を入れ沸騰させる
3. えのき・もやし・ネギ・豆腐（崩して）を加え3分煮る
4. 水溶き片栗粉でとろみをつける
5. 溶き卵を回し入れてかき混ぜる
6. ごま油・塩・黒胡椒で仕上げ

## ② 具沢山スープ（白菜・鶏ひき肉・春雨）

**ソース:** [Threads @kana.0716t](https://www.threads.com/@kana.0716t/post/DVXy7lfE0Jf/)
**YouTube:** —
**食数:** 2〜3食分 | **kcal/食:** ~250 kcal

### 材料

| 食材 | 分量 |
|------|------|
| 白菜 | 1/8個 |
| えのき | 1袋 |
| 鶏ひき肉 | 150g |
| 春雨 | 30g |
| 水 | 500ml |
| 鶏ガラスープの素 | 大さじ1 |
| 醤油 | 小さじ1 |
| ごま油 | 小さじ1 |
| 生姜 | 小さじ1 |
| 黒胡椒 | 適量 |

### 作り方

1. 白菜をざく切り、えのきを3等分
2. 鍋に水・鶏ガラ・生姜を入れ沸騰させる
3. 鶏ひき肉を加えほぐしながら火を通す
4. 白菜・えのき・春雨を加え中火で5分煮る
5. 醤油・ごま油・黒胡椒で味を整える

## ③ 酸辣スープ（鶏もも・もやし・しめじ）

**ソース:** [Instagram reel @kana.0716t](https://www.instagram.com/reel/CxPqO1Oy7v6/)
**YouTube:** —
**食数:** 2〜3食分 | **kcal/食:** ~280 kcal

### 材料

| 食材 | 分量 |
|------|------|
| 鶏もも肉（皮なし） | 1枚 |
| 人参 | 1/4本 |
| もやし | 1/2袋 |
| 長ネギ | 1本 |
| しめじ | 1袋 |
| 卵 | 1個 |
| 水 | 500ml |
| 鶏ガラスープの素 | 大さじ1 |
| 減塩醤油 | 大さじ2 |
| 料理酒 | 大さじ1.5 |
| 米酢 | 大さじ2 |
| 豆板醤 | 小さじ1 |
| 片栗粉 | 大さじ1 |
| 水（とろみ用） | 大さじ2 |

### 作り方

1. 鶏もも肉を一口大に切り、人参を千切り
2. 鍋に水・鶏ガラ・酒を入れ沸騰させ鶏肉を入れる
3. 野菜を加え中火で5分煮る
4. 醤油・酢・豆板醤で味付け
5. 水溶き片栗粉でとろみをつける
6. 溶き卵を回し入れて仕上げ
