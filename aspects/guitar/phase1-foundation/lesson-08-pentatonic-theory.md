## 1. ペンタトニックスケールって何？（10分）

### 5つの音のスケール

Lesson 5 で学んだメジャースケールは **7つの音** でできていたよね。

```
C メジャースケール（7音）:
C  D  E  F  G  A  B
1  2  3  4  5  6  7
```

**ペンタトニックスケール** は、ここから **2つの音を抜いて5つにしたもの**。

「ペンタ（penta）」はギリシャ語で「5」、「トニック（tonic）」は「音」。
つまり **「5音のスケール」** という意味。

### なぜ2つ抜くの？

メジャースケールの7音のうち、**半音の関係にある2つの音** を抜く。

Lesson 5 で学んだことを思い出して:

```
C メジャースケール:
C → D → E → F → G → A → B → C
  全  全  半  全  全  全  半
        ↑                 ↑
     E→Fが半音          B→Cが半音
```

この半音を作っている音のうち、**F（4番目）と B（7番目）** を抜く。

```
C メジャースケール:  C  D  E  F  G  A  B
                                ×        ×
C メジャーペンタ:    C  D  E     G  A
                     1  2  3     5  6
```

5つの音が残る。これが **C メジャーペンタトニックスケール**。

### なぜ半音を抜くと「最強」になるのか

ここが面白いところ。

半音（フレット1個分の距離）は「緊張感」を生む音程だったよね（Lesson 2）。
短2度の響きを思い出してみて --- ちょっと不安定で、ピリッとした感じ。

ペンタトニックは、その **半音の関係を全部取り除いた** スケール。

```
C メジャーペンタ: C  D  E  G  A
音の距離:        全音 全音 短3度 全音

→ 半音が1つもない！
```

半音がないということは:
- **どの音を弾いても「外れた」感じがしない**
- **どんなコードの上で弾いても大きくぶつからない**
- **適当に弾いてもそれなりに音楽っぽく聴こえる**

これが「最強」と言われる理由。

初心者が即興（アドリブ）を始めるとき、ペンタトニックを使えば「音を外す恐怖」がほぼなくなる。
プロのミュージシャンも、ペンタトニックをベースにソロを組み立てることが非常に多い。

例え話で言うと:

```
メジャースケール（7音）= フルコースの料理
  → 材料が多い。組み合わせ次第で最高にもなるし、失敗もある

ペンタトニック（5音）= 定食
  → 材料は少ないけど、どう組み合わせてもハズレがない
```

> **黒田メモ:** 「ペンタトニックは初心者用」って思われがちだけど、それは大きな誤解。John Mayer、B.B. King、Jimi Hendrix、Eric Clapton --- 世界のトップギタリストがペンタトニックをベースにしたソロを弾いてる。「シンプルだからこそ奥が深い」のがペンタトニックの魅力。

## 2. マイナーペンタトニックとメジャーペンタトニック（15分）

### 2種類のペンタトニック

ペンタトニックには **メジャー（明るい）** と **マイナー（暗い）** の2種類がある。

さっき見た C メジャーペンタトニックは「明るい方」。
ロック、ブルース、John Mayer のソロでよく使われるのは **マイナーペンタトニック（暗い方）**。

### A マイナーペンタトニック

マイナーペンタトニックの構造を見てみよう。

```
A ナチュラルマイナースケール（7音）:
A  B  C  D  E  F  G
1  2  3  4  5  6  7

ここから2番目（B）と6番目（F）を抜く:

A マイナーペンタ: A  C  D  E  G
                  1  ♭3  4  5  ♭7
```

> **用語メモ:** 「♭3（フラットスリー）」「♭7（フラットセブン）」は、メジャースケールの3番目・7番目の音から半音下がった音。マイナーの響きを作る音。Lesson 2 で学んだ「短3度」がこの♭3。

### 大発見: C メジャーペンタ = A マイナーペンタ

ここで驚きの事実。

```
C メジャーペンタ: C  D  E  G  A
A マイナーペンタ: A  C  D  E  G
```

**使ってる音が全く同じ！**

ただ、スタートする音（ルート）が違うだけ。

これは Lesson 2 の「平行調」と同じ関係。C メジャーと A マイナーは「親戚」みたいなもの。

### じゃあ何が違うの？

音は同じでも、**どの音を「ホーム」にするか** で響きが変わる。

```
C メジャーペンタとして弾く:
→ C を中心にする → 明るい響き

A マイナーペンタとして弾く:
→ A を中心にする → 暗い・ブルージーな響き
```

実際のロックやブルースでは **A マイナーペンタとして弾くことがほとんど**。
だから今日は A マイナーペンタに集中しよう。

### A マイナーペンタの5音を覚えよう

```
A マイナーペンタトニック:

音名:      A    C    D    E    G    (A)
度数:      1   ♭3    4    5   ♭7   (8/オクターブ)
半音の数:   0    3    5    7   10    12

各音の性格:
A（ルート）  → 家。ここに戻ると安定する
C（♭3）     → マイナーの「暗さ」を決める音。切ない感じ
D（4度）    → 中立的。上にも下にも行ける「分岐点」
E（5度）    → 安定。パワフル。ルートの次に安定する音
G（♭7）    → ブルージー。「もうちょっと続きがありそう」な響き
```

> **黒田メモ:** 5音しかないけど、それぞれに個性がある。特に ♭3（C）と ♭7（G）がマイナーペンタの「味」を決めてる。この2つの音がブルースやロックの「あの感じ」を作ってるんだ。

## 3. ギターの指板上で見る A マイナーペンタ（10分）

### ポジション1（5フレット付近）

これがギターで一番最初に覚えるペンタトニックの形。
世界中のギタリストがここから始める。

```
【指板図】A マイナーペンタトニック ポジション1

フレット:  5     6     7     8
     e |--●---|------|------|--●---|  A           C
     B |--●---|------|------|--●---|  E           G
     G |--●---|------|--●---|------|  C           D
     D |--●---|------|--●---|------|  G           A
     A |--●---|------|--●---|------|  D           E
     E |--●---|------|------|--●---|  A           C
```

TAB 譜で見ると:

```
【TAB譜】A マイナーペンタトニック ポジション1（上昇）

e |---------------------------5--8--|
B |---------------------5--8-------|
G |-----------------5--7-----------|
D |-----------5--7-----------------|
A |-----5--7-----------------------|
E |--5--8--------------------------|

音名:
e |---------------------------A--C--|
B |---------------------E--G-------|
G |-----------------C--D-----------|
D |-----------G--A-----------------|
A |-----D--E-----------------------|
E |--A--C--------------------------|

→ 使ってる音は A, C, D, E, G の5種類だけ
→ A マイナーペンタ（A, C, D, E, G）と同じ音
```

### この「形」が最強な理由

1. **人差し指は5フレット、薬指は7フレット、小指は8フレット** --- 手をほとんど動かさなくていい
2. **1本の弦につき2音ずつ** --- シンプルで覚えやすい
3. **この形をそのまま別のフレットに移動すれば、別のキーのマイナーペンタになる**

```
この形を移動するだけ:
5フレット → A マイナーペンタ（ルート A）
3フレット → G マイナーペンタ（ルート G）
7フレット → B マイナーペンタ（ルート B）
8フレット → C マイナーペンタ（ルート C）

6弦の音がルートになる。ランドマークを覚えてれば即対応できる。
```

### フィンガリング

```
【TAB譜】フィンガリング

e |---------------------------1--4--|
B |---------------------1--4-------|
G |-----------------1--3-----------|
D |-----------1--3-----------------|
A |-----1--3-----------------------|
E |--1--4--------------------------|

1 = 人差し指（5フレット担当）
3 = 薬指（7フレット担当）
4 = 小指（8フレット担当）
```

> **黒田メモ:** メジャースケール（Lesson 5）のポジション1は3弦で手の位置がずれたけど、ペンタトニックは「人差し指と薬指/小指」の2本で全部弾ける。この「シンプルさ」がペンタトニックの武器。形を覚えるのが早いから、すぐに「音楽的に使う」段階に進める。

## 4. なぜペンタトニックで「ソロが弾ける」のか（10分）

### コードの上で安全な理由

ここが理論的に一番面白いところ。

Key=A マイナー（または Key=C メジャー）の曲で使われるコードは:

```
Am, Dm, Em, C, F, G ...
```

A マイナーペンタの5音（A, C, D, E, G）がこれらのコードとどう関係するか見てみよう:

```
Am = A, C, E  → ペンタの音が3つ入ってる
Dm = D, F, A  → ペンタの音が2つ入ってる
Em = E, G, B  → ペンタの音が2つ入ってる
C  = C, E, G  → ペンタの音が3つ入ってる
F  = F, A, C  → ペンタの音が2つ入ってる
G  = G, B, D  → ペンタの音が2つ入ってる
```

**どのコードの上でも、ペンタの音が2〜3個はコードの構成音と一致する。**

だから「外れた」感じがしない。

メジャースケールだと7音あるから、コードと「ぶつかる」音が出てくることがある（例: F の音は C コードの上でちょっと緊張する）。
でもペンタは「ぶつかりやすい音」（F と B）を最初から抜いてあるから、安心して弾ける。

### ブルースとの深い関係

ペンタトニックスケールはブルースの根幹。

ブルースは「7thコード（セブンスコード）」をたくさん使う音楽。
7thコードには ♭7 の音が入ってる。
ペンタトニックにも ♭7（G）が入ってる。

```
A7 コード = A, C#, E, G
A マイナーペンタ = A, C, D, E, G

共通の音: A, E, G（3つも一致）
```

さらに、マイナーペンタの ♭3（C）と A7 コードの 3（C#）は半音違い。
この「半音のぶつかり」が **ブルースの独特な響き** を生み出す。

これは「間違い」じゃなくて「味」。料理でいう「ちょっとした酸味」みたいなもので、これがないとブルースにならない。

> **黒田メモ:** ブルースという音楽は、理論的には「矛盾」を含んでいる。メジャーとマイナーが混ざり合ってる。でもその矛盾こそが人間の感情のリアルさを表現してる。嬉しいのに悲しい、笑ってるのに泣きたい --- そういう複雑な感情をたった5つの音で表現できるのが、ペンタトニックのすごさなんだ。

## 5. John Mayer とペンタトニック（10分）

### John Mayer のソロの秘密

John Mayer は「ブルースギタリスト」としても非常に高く評価されている。
彼のソロを分析すると、ベースになっているのはほぼ全てペンタトニック。

### Gravity のソロ冒頭

```
Gravity のソロ（冒頭部分を簡略化）:

使われている音:
G, A, C, D, E ...

→ すべて A マイナーペンタトニックの音
```

John Mayer のすごいところは、ペンタトニックという「5音の素材」だけで、あれほど豊かな表現ができること。

秘密は:
1. **音の選び方**: 5音のうちどれを選ぶか、どの順番で弾くか
2. **リズム**: 同じ音でもリズムを変えると全然違うフレーズになる
3. **テクニック**: チョーキング（弦を押し上げて音程を変える）、スライド、ハンマリングで音に表情をつける
4. **ダイナミクス**: 強弱の幅が広い（Lesson 4 で学んだこと）

### 「5音しかない」のではなく「5音で十分」

```
全12音    → 可能性は無限だけど、迷子になりやすい
7音（スケール）→ 道筋がある。でもまだ選択肢が多い
5音（ペンタ） → 最小限の選択肢。だからこそ「どう弾くか」に集中できる

John Mayer のアプローチ:
「何を弾くか」よりも「どう弾くか」を大事にする
→ ペンタトニックはその哲学にぴったりのツール
```

### 聴いてみよう

もし時間があれば、YouTubeで「John Mayer Gravity solo」を検索して聴いてみて。
今日学んだ A マイナーペンタの5音（A, C, D, E, G）を意識しながら聴くと、
「あ、本当にこの5つの音が中心なんだ」って気づくはず。

> **黒田メモ:** ペンタトニックを「制限」と捉えるか「自由」と捉えるか。5音しかないから「足りない」のか、5音だから「迷わない」のか。John Mayer は明らかに後者。少ない材料で最大限の表現をする --- それがペンタトニックの哲学。これから瀬戸さんと橋本さんと一緒に、その「自由」を体験していくよ。

## 6. メジャースケールとの関係を整理（5分）

### 全体像

ここまで学んできたスケールの関係を整理しよう。

```
C メジャースケール（7音）: C  D  E  F  G  A  B
                                    ×        ×
C メジャーペンタ（5音）:   C  D  E     G  A

A マイナーペンタ（5音）:   A  C  D  E  G
↑ C メジャーペンタと同じ音。スタート地点が違うだけ
```

覚えておくポイント:

1. **ペンタトニック = メジャースケールから2音抜いたもの**
2. **抜くのは半音を作る音（4番目と7番目）**
3. **メジャーペンタとマイナーペンタは同じ音。ルートが違うだけ**
4. **ペンタトニックはメジャースケールの「サブセット（部分集合）」**

つまり、**メジャースケールを知っていれば、ペンタトニックはすでに知っている**ということ。
ポジション1で弾いた C メジャースケールの中から、F と B を弾かなければ、それがペンタトニック。

### なぜわざわざ分けて学ぶのか

メジャースケールの中からペンタの音だけを意識して弾くことで:
- **ソロの骨格（＝ペンタ）** と **装飾（＝残りの2音）** を区別できるようになる
- 最初はペンタだけで弾いて、慣れたら残りの2音を「スパイス」として足していく
- John Mayer もこのアプローチでソロを組み立てている

---

## まとめ

今日学んだことを整理しよう。

1. **ペンタトニック = 5音のスケール**。メジャースケールから半音を作る2音を抜いたもの。
2. **半音がないから「外さない」**。どの音を弾いてもコードと大きくぶつからない。
3. **マイナーペンタとメジャーペンタは同じ音**。ルートが違うだけ（A マイナーペンタ = C メジャーペンタ）。
4. **A マイナーペンタの5音: A, C, D, E, G**。それぞれに個性がある。
5. **ポジション1（5フレット付近）は世界共通の「最初のペンタの形」**。
6. **John Mayer のソロはペンタトニックがベース**。5音で十分豊かな表現ができる。

次は瀬戸さんと一緒に、ポジション1で実際にソロを弾く練習をするよ。
バッキングトラックに合わせて、ペンタトニックだけで即興する。

### 練習のポイント

- **「形」を手に覚えさせることが今週の最優先。** 指板を見なくても弾けるようになるまで繰り返す。

- **John Mayer を聴く時間も練習。** ソロの中でペンタの音を追いかけてみて。「あ、今の音は A だ」「今のは G だ」って分かるようになったら大きな進歩。
- **焦らなくていい。** ペンタの形は来週の実践（Lesson 9）でたっぷり使うから、今週は「覚えること」に集中しよう。
