# Phase 3: ブルースリック大全

## ゴール

- ペンタトニック・ボックス2-3を習得してポジション移動できるようにする
- ブルーノート＆クロマチックで「渋い」フレーズを弾く
- ダブルストップ＆コードトーンの引き出しを増やす
- BPM 別のリック使い分けを身につける

## このフェーズを終えると

- ペンタ3ポジションを行き来しながらリックを弾ける
- ブルーノート・クロマチックで「ペンタだけじゃ出せない渋さ」を表現できる
- ダブルストップで Chuck Berry / Hendrix 風のフレーズが弾ける
- スロー・ミディアム・アップテンポそれぞれに合ったリックを選べる
