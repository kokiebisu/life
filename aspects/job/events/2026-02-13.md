# 最終出社日

## 会社への確認・依頼

- [ ] 離職票の発行を依頼する（失業保険申請に必要）
- [ ] 健康保険資格喪失証明書の発行を依頼する（国保加入に必要）
- [ ] 源泉徴収票の発行を依頼する（確定申告に必要）
- [ ] 退職証明書の発行を依頼する（離職票が届くまでの間に国保加入等で使える）
- [ ] 雇用保険被保険者証を受け取る（会社保管の場合。ハローワークで必要）
- [ ] 源泉徴収票は新住所に送付してほしいと人事・総務に伝える
- [ ] 退職後の連絡先・住所を人事に伝える（離職票等の書類郵送先として）
- [ ] 年金手帳が会社預かりか確認（預けていれば返却してもらう）
- [ ] 退職金の有無・支給時期を確認
- [ ] 最終給与の支払日・未消化有給の扱いを確認
- [ ] 住民税の残額の精算方法を確認（一括徴収 or 普通徴収への切替）
- [ ] 通勤定期券の精算方法を確認（払い戻し手続きが必要か）

## 返却物

- [ ] 健康保険証
- [ ] 社員証・入館証
- [ ] 会社PC・備品
- [ ] 名刺（自分の名刺の残り。受け取った名刺の扱いも確認）
- [ ] その他貸与物

## 最終出社日にやること

- [ ] 業務引き継ぎの最終確認（漏れがないか上司・後任と確認）
- [ ] デスク・ロッカーの片付け（私物の持ち帰り）
- [ ] 会社PC内の個人データを削除する
- [ ] 同僚・関係者への挨拶
