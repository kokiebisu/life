# ジムメニュー

## 方針

- デカくしない。体脂肪を落として体力をつける
- メイン: 有酸素（インクライン・ウォーキング）
- サブ: 軽い筋トレ（姿勢改善・基礎体力維持）
- 週3回、1回50分以内（+ 往復徒歩40分）

## 場所

- FIT PLACE24 みなとみらい
- 片道徒歩20分（往復40分 = ウォームアップ＋クールダウン）
- 合計所要時間: 約90分（徒歩20分 + ワークアウト50分 + 徒歩20分）

## メニューローテーション

A日とB日を交互に実施。週3回なら A→B→A の繰り返し。

---

### A日: ウォーキング + 筋トレ（50分）

---

#### 1. インクライン・ウォーキング（30分）

> トレッドミル（ランニングマシン）の傾斜を上げて歩くだけ。
> 坂道を早歩きするイメージ。走らなくていい。

| 項目 | 設定 |
|------|------|
| 傾斜 | 10〜12% |
| 速度 | 5〜6 km/h |
| 時間 | 30分 |

**やり方:**
1. トレッドミルに乗って「START」を押す
2. 速度を 5 km/h に設定する（ボタンで調整）
3. 傾斜（INCLINE）を 10% に設定する
4. そのまま 30分歩く。走らなくていい
5. 終わったら速度を落として 2分くらいゆっくり歩いてからSTOP

**注意:**
- 手すりに掴まらない（効果が半減する）
- 息が上がりすぎたら傾斜を下げる。無理しない
- 心拍数 120〜140bpm が目安（マシンに表示される）

---

#### 2. 腕立て伏せ（3セット × 15回）

> 床に手をついて体を上げ下げする運動。胸と腕を鍛える。

**やり方:**
1. 床にうつ伏せになる
2. 両手を肩幅より少し広めに床につく（胸の横あたり）
3. つま先と手で体を支えて、体をまっすぐ持ち上げる
4. 肘を曲げてゆっくり体を下ろす（胸が床につくギリギリまで）
5. 肘を伸ばして体を押し上げる → これで1回

**きつかったら:**
- 膝をついてやる（膝つき腕立て伏せ）。これでも十分効く
- 15回できなければ、できる回数でOK

**よくある間違い:**
- お尻が上がる or 腰が反る → 頭からかかとまで一直線を意識
- 腕だけで上げようとする → 胸の筋肉を使う意識で

| 設定 | 値 |
|------|------|
| セット数 | 3セット |
| 回数 | 15回（きつければ減らしてOK） |
| セット間の休憩 | 60秒 |
| 鍛える場所 | 胸・腕・肩 |

---

#### 3. ダンベルロウ（3セット × 15回）

> ベンチに片手をついて、もう片方の手でダンベルを引き上げる運動。
> 背中を鍛えて猫背を改善する。

**やり方:**
1. フラットベンチ（平らなベンチ台）の横に立つ
2. 片方の手と同じ側の膝をベンチに乗せる（右手+右膝 or 左手+左膝）
3. もう片方の手でダンベルを持つ（最初は 3〜5kg で十分）
4. 背中をまっすぐにしたまま、ダンベルを脇腹に向かって引き上げる
5. ゆっくり下ろす → これで1回
6. 片側15回やったら、反対側も15回

**コツ:**
- 腕で引くのではなく、肩甲骨（背中の骨）を寄せるイメージ
- 背中が丸まらないように。目線は少し前方
- 重すぎると腕だけで引いてしまうので、軽めで始める

**ダンベルの選び方（FIT PLACE24）:**
- ダンベル置き場から 3kg or 5kg を取る
- 楽にできるなら次回から重くする

| 設定 | 値 |
|------|------|
| セット数 | 3セット（左右それぞれ） |
| 回数 | 15回 |
| セット間の休憩 | 60秒 |
| ダンベル重量 | 3〜5kg（最初は軽めで） |
| 鍛える場所 | 背中（広背筋）・猫背改善 |

---

#### 4. スクワット（3セット × 15回）

> 立った状態からお尻を落として立ち上がる運動。
> 下半身全体を鍛える。器具なし（自重）でOK。

**やり方:**
1. 足を肩幅に開いて立つ。つま先はやや外向き（30度くらい）
2. 両手を前に伸ばす or 胸の前で組む（バランス取りやすい方で）
3. お尻を後ろに引きながら、椅子に座るようにしゃがむ
4. 太ももが床と平行になるくらいまで下ろす
5. かかとで地面を押して立ち上がる → これで1回

**注意:**
- 膝がつま先より前に出すぎない（膝を痛める）
- かかとが浮かない（浮くなら浅めでOK）
- 背中が丸まらない。胸を張る
- 深くしゃがめなければ、できる範囲で

| 設定 | 値 |
|------|------|
| セット数 | 3セット |
| 回数 | 15回 |
| セット間の休憩 | 60秒 |
| 鍛える場所 | 太もも・お尻・体幹 |

---

#### 5. プランク（3セット × 30秒）

> うつ伏せの状態で、前腕（肘から手首）とつま先だけで体を支えてキープする運動。
> 腹筋・体幹を鍛えてお腹を引き締める。じっとしてるだけ。

**やり方:**
1. 床にうつ伏せになる
2. 肘を肩の真下について、前腕（肘〜手首）を床につける
3. つま先を立てて、体全体を持ち上げる
4. 頭からかかとまで一直線の姿勢をキープ
5. そのまま 30秒間じっとする。呼吸は止めない

**注意:**
- お尻が上がったり、腰が沈んだりしない → 一直線を意識
- 呼吸を止めない（自然に呼吸する）
- 30秒きつかったら 15秒でもOK。少しずつ伸ばす
- プルプルしてきたら効いてる証拠

| 設定 | 値 |
|------|------|
| セット数 | 3セット |
| 時間 | 30秒（きつければ15秒から） |
| セット間の休憩 | 60秒 |
| 鍛える場所 | お腹・体幹（腹直筋・腹横筋） |

---

### B日: ウォーキングのみ（40分）

> A日の筋トレの疲労を回復しながら、脂肪を燃やす日。

| 項目 | 設定 |
|------|------|
| 傾斜 | 10〜12% |
| 速度 | 5〜6 km/h |
| 時間 | 40分 |

やり方は A日のインクライン・ウォーキングと同じ。時間が10分長いだけ。

心拍数を脂肪燃焼ゾーン（120〜140bpm 目安）に維持。

---

## 導入期（最初の2週間）

- 傾斜8%・速度5km/h・20分から始める
- 筋トレは自重のみ（ダンベルなし）
- フォームを覚えることが目的。追い込まない
- 参考動画: Jeff Nippard、山本義徳

## 段階的な進め方

| 時期 | ウォーキング | 筋トレ |
|------|-------------|--------|
| 1〜2週目 | 傾斜8%/速度5/20分 | 自重のみ |
| 3〜4週目 | 傾斜10%/速度5/30分 | 軽いダンベル追加 |
| 5週目〜 | 傾斜12%/速度6/40分 | 回数・セット数を増やす |

速度は6km/h を上限にして、それ以上は傾斜で負荷を調整する。

## 持ち物

- 着替え（Tシャツ + 短パン）
- タオル
- 水（500ml以上）
- イヤホン
- ジム用シューズ

## 習慣化のコツ

- トリガー: 昼食後 → そのままジムへ（毎回判断しない）
- 朝の時点でバッグを準備しておく
- 帰宅後シャワー → 午後の開発へ（リフレッシュ効果で集中力回復）
