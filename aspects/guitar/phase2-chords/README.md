# Phase 2: コードの色彩

## ゴール

- Jazz / Neo-Soul で使われるテンションコードを習得する
- コード上でメロディを動かす技術を身につける
- 定番進行（ii-V-I）を実践レベルで弾けるようにする

## このフェーズを終えると

- 9th / 13th などテンションを自在に加えられる
- コードフォームの中でメロディラインを作れる
- ベースラインを維持しながらコードチェンジできる
- ii-V-I 進行をスムーズに弾ける
