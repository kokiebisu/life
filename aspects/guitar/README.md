# guitar

ギター練習の記録と機材管理。アコギ（Martin Custom Shop）とエレキ（Fender Custom Shop '62 Strat）の二刀流、教会ワーシップでの担当を見据えた学習を進めている。

## Overview

コードの「なぜこの押さえ方になるか」の理解を重視する学習スタイル。基本コード・7th コードは習得済み、テンションコードとブルーススケールが現在のテーマ。
