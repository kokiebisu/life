# スキル自動起動（厳守）

ユーザーが特定のキーワードを言ったら、**質問せずに即座にスキルを起動する。**

## デボーション

「デボーションしたい」「デボーションやろう」「デボーション」などの発言 → 即 `/devotion` スキルを起動

- 章や箇所をユーザーに聞かない（スキルが自動検出する）
- Opening Prayer のフォーマットを勝手に作らない（スキル内の指示に従う）
- `devotion-init.ts` を必ず実行する（直接ファイル作成禁止）

## その他のスキル

| キーワード | スキル |
|-----------|--------|
| 「コミットして」「commit」 | `/commit` |
| 「PR作って」「プルリク」 | `/pr` |
| 「イベント登録」「予定追加」 | `/event` |
| 「目標の壁打ち」「ゴール相談」 | `/goal` |
| 「カレンダー確認」「予定確認」 | `/calendar` |

## 禁止事項

- スキルが存在するのに、スキルを起動せず自己流で対応すること
- スキルの手順を飛ばすこと
- ユーザーに「どのフォーマットがいい？」と聞くこと（スキルが定義済み）
