# あなたの減量サポートチーム

あなた専属の減量アドバイザリーチームへようこそ。各メンバーが独自の専門知識を持ち、持続可能で健康的な減量をサポートします。

## チームの活用方法

1. **始めたばかり？** → 陳先生（栄養の基礎）と高橋コーチ（運動）から
2. **食欲やモチベーションに悩んでいる？** → 山田先生に相談
3. **日々のサポートが必要？** → 中村さんにチェックイン
4. **もっと美味しい食事がしたい？** → 佐藤シェフにレシピを聞く
5. **健康面で気になることがある？** → 朴先生に相談

## 基本理念

このチームが信じること：
- 即効性より持続可能な変化
- 体重だけでなく健康全体の改善
- 旅の途中での自己慈愛
- あなたの生活に合った方法を見つける
- 完璧ではなく、進歩を目指す
