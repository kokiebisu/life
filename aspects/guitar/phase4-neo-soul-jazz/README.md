# Phase 4: Neo Soul & Jazz（応用）

目指す音楽スタイルを本格的に学ぶフェーズ。

## Goals

このフェーズを終えると:
- 複雑なコードヴォイシングが弾ける
- ii-V-Iを様々なキーで弾ける
- Neo Soulのグルーヴを体得している
- King Gnuの曲を分析・演奏できる

## Topics

### 1. 複雑なコードヴォイシング
- Shell Voicing（3度と7度）
- Drop 2, Drop 3 ヴォイシング
- Upper Structure Triads
- 同じコードの複数のヴォイシング

### 2. ii-V-I マスター
- 全12キーでii-V-I
- ii-V-Iのリック集
- ターンアラウンド（I-vi-ii-V）
- マイナーii-V-i

### 3. Neo Soul グルーヴ
- 16分音符のグルーヴ
- ゴーストノート
- コードのリズミックな刻み
- D'Angelo, Erykah Badu的アプローチ

### 4. 分析と実践
- King Gnuの楽曲分析
- コード進行の借用（Modal Interchange）
- セカンダリードミナント
- サブスティテューション

## Study Artists

### Neo Soul
- D'Angelo
- Erykah Badu
- Tom Misch
- FKJ
- Cory Henry

### Jazz Guitar
- Julian Lage
- Kurt Rosenwinkel
- Pat Metheny

### King Gnu 重点曲
- 「白日」- 複雑な転調を分析
- 「三文小説」- ジャズ的なアプローチ
- 「一途」- グルーヴと和声の融合
- 「Teenager Forever」- ポップとジャズの境界

## Practice

### Daily
- ii-V-I を2-3キーで練習
- ヴォイシングのバリエーション練習

### Weekly
- 1曲を深く分析
- 新しいリック/ヴォイシングを習得

## Notes

ジャズリック、ヴォイシングの発見を記録。
