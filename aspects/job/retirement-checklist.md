# 退職手続きチェックリスト

最終出社日: 2026年2月13日（金）
退職日: 2026年2月28日（土）

## 退職前（〜2/13 最終出社日まで）

### 会社への確認・依頼

- [ ] 離職票の発行を依頼する（失業保険申請に必要）
- [ ] 健康保険資格喪失証明書の発行を依頼する（国保加入に必要）
- [ ] 源泉徴収票の発行を依頼する（確定申告に必要）
- [ ] 年金手帳が会社預かりか確認（預けていれば返却してもらう）
- [ ] 退職金の有無・支給時期を確認
- [ ] 最終給与の支払日・未消化有給の扱いを確認
- [ ] 住民税の残額の精算方法を確認（一括徴収 or 普通徴収への切替）

### 会社への返却物

- [ ] 健康保険証を返却（退職日以降は使えない）
- [ ] 社員証・入館証の返却
- [ ] 会社PC・備品の返却
- [ ] その他貸与物の返却

### 個人対応

- [ ] 名刺・業務データなど会社資産の整理
- [ ] 必要な個人データのバックアップ（私物のみ）

## 退職後すぐ（3月1日〜）

### 健康保険の切り替え（期限厳守）

3つの選択肢がある。**どれか1つを選ぶ必要がある。**

| 選択肢 | 期限 | 特徴 |
|--------|------|------|
| ① 国民健康保険 | 退職翌日から **14日以内**（3/14まで） | 前年所得ベースで保険料が決まる |
| ② 任意継続 | 退職翌日から **20日以内**（3/20まで） | 在職中の保険を最大2年継続。保険料は全額自己負担（約2倍） |
| ③ 家族の扶養に入る | 退職後速やかに | 年収130万円未満が条件。保険料負担なし |

**やること:**

- [ ] 国保と任意継続の保険料を比較する
  - 国保: 役所の窓口 or 電話で試算してもらえる
  - 任意継続: 現在の健康保険組合に確認
- [ ] どちらにするか決定する
- [ ] 選んだ方の加入手続きをする

**国民健康保険の場合:**
- 持ち物: 健康保険資格喪失証明書、マイナンバーカード、印鑑
- 場所: 住所地の市区町村役所

**任意継続の場合:**
- 退職前の健康保険組合に申請書を提出

### 年金の切り替え（期限: 退職翌日から14日以内 → 3/14まで）

- [ ] 国民年金（第1号被保険者）への切り替え手続き
  - 場所: 住所地の市区町村役所
  - 持ち物: 年金手帳 or 基礎年金番号通知書、退職日が確認できる書類、マイナンバーカード
- [ ] 付加年金（月400円）に加入するか検討（任意）
- [ ] 経済的に厳しい場合は免除・猶予制度を確認

### 失業保険（雇用保険）

- [ ] 離職票が届いたらハローワークへ行く（退職後2週間程度で届く）
- [ ] 求職申込み・失業保険の受給手続き
  - 持ち物: 離職票、マイナンバーカード、写真2枚（3×2.4cm）、印鑑、通帳
  - 場所: 住所地管轄のハローワーク
- [ ] 自己都合退職の場合、給付制限期間（2ヶ月）があることを理解する

> **注意:** 失業保険は「就職する意思がある」ことが条件。個人事業を始める場合は受給できない場合がある。tsumugiとの関係を確認すること。

## 3月中にやること

### 住民税

- [ ] 住民税の納付方法を確認する
  - 1月〜5月退職の場合、残額は最終給与から一括徴収されることが多い
  - 普通徴収に切り替わった場合は自分で納付（年4回）

### その他

- [ ] iDeCo / 企業型DCがあれば移換手続き（退職後6ヶ月以内）
- [ ] 退職後の収入見込みを整理し、資金計画を立てる

## 2027年の確定申告（2027年2月16日〜3月15日）

- [ ] 源泉徴収票を保管しておく
- [ ] 2026年中に再就職しなければ確定申告が必要
- [ ] 国民年金・国保の保険料は社会保険料控除の対象（払った証明を保管）

## 参考: 手続きスケジュール

```
3月中  離職票届き次第 → ハローワークへ
```
