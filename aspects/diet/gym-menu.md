# ジムメニュー

## 方針

- デカくしない。体脂肪を落として体力をつける
- メイン: マシン筋トレ（全身をバランスよく）
- サブ: 有酸素（インクライン・ウォーキング）
- 週3回、1回50分以内（+ 往復徒歩40分）

## 場所

- FIT PLACE24 みなとみらい
- 片道徒歩20分（往復40分 = ウォームアップ＋クールダウン）
- 合計所要時間: 約90分（徒歩20分 + ワークアウト50分 + 徒歩20分）

## メニューローテーション

A日とB日を交互に実施。週3回なら A→B→A の繰り返し。

---

### A日: マシン筋トレ + ウォーキング（50分）

---

#### 1. インクライン・ウォーキング（20分）

> トレッドミル（ランニングマシン）の傾斜を上げて歩くだけ。
> ウォームアップ兼有酸素。走らなくていい。

| 項目 | 設定 |
|------|------|
| 傾斜 | 10〜12% |
| 速度 | 5〜6 km/h |
| 時間 | 20分 |

**やり方:**
1. トレッドミルに乗って「START」を押す
2. 速度を 5 km/h に設定する（ボタンで調整）
3. 傾斜（INCLINE）を 10% に設定する
4. そのまま 20分歩く。走らなくていい
5. 終わったら速度を落として 2分くらいゆっくり歩いてからSTOP

**注意:**
- 手すりに掴まらない（効果が半減する）
- 息が上がりすぎたら傾斜を下げる。無理しない
- 心拍数 120〜140bpm が目安（マシンに表示される）

---

#### 2. ベンチプレス（3セット × 15回）

> バーベルを胸の上で上げ下げするマシン。胸・腕・肩を鍛える。
> FIT PLACE24 にはスミスマシン（バーの軌道が固定されたマシン）がある。

| 設定 | 値 |
|------|------|
| セット数 | 3セット |
| 回数 | 15回 |
| セット間の休憩 | 60秒 |
| 重量 | バーのみ（20kg）→ 慣れたら +2.5kg ずつ |
| 鍛える場所 | 胸（大胸筋）・腕（三頭筋）・肩 |

**やり方:**
1. ベンチに仰向けに寝る。足は床にしっかりつける
2. バーを肩幅より少し広めに握る
3. ラックからバーを外し、胸の上に構える
4. ゆっくりバーを胸に下ろす（胸に軽く触れるくらい）
5. 胸から押し上げてスタートポジションに戻す → これで1回

**注意:**
- **必ずセーフティバーを設定する**（一人で潰れると危険）
- 手首を返さない（バーの真下に手首がくるように）
- 肩甲骨を寄せて胸を張る（肩ではなく胸で押す意識）
- お尻をベンチから浮かせない

---

#### 3. ラットプルダウン（3セット × 15回）

> 頭上からバーを引き下ろすマシン。背中を鍛えて姿勢を改善する。

| 設定 | 値 |
|------|------|
| セット数 | 3セット |
| 回数 | 15回 |
| セット間の休憩 | 60秒 |
| 重量 | 15kg → 慣れたら +5kg ずつ |
| 鍛える場所 | 背中（広背筋）・姿勢改善 |

**やり方:**
1. シートに座り、太ももをパッドで固定する
2. バーを肩幅より少し広めに握る（順手）
3. 胸を張り、少し後ろに傾く（15度くらい）
4. バーを鎖骨あたりまで引き下ろす
5. ゆっくり戻す → これで1回

**コツ:**
- 腕で引くのではなく、肘を脇腹に向かって引くイメージ
- 肩甲骨（背中の骨）を寄せる感覚で
- 体を大きく反らさない。軽い後傾のみ
- 重すぎると腕だけで引いてしまうので、軽めで始める

---

#### 4. レッグプレス（3セット × 15回）

> 座った状態で足でプレートを押すマシン。下半身全体を鍛える。

| 設定 | 値 |
|------|------|
| セット数 | 3セット |
| 回数 | 15回 |
| セット間の休憩 | 60秒 |
| 重量 | 30kg → 慣れたら +10kg ずつ |
| 鍛える場所 | 太もも（大腿四頭筋）・お尻（大臀筋）|

**やり方:**
1. シートに座り、背中をしっかりパッドにつける
2. 足を肩幅でプレートに置く（つま先はやや外向き）
3. ロックを外し、足でプレートを押し出す
4. 膝を曲げてゆっくり戻す（90度くらいまで）
5. 再び押し出す → これで1回

**注意:**
- **膝をロックアウト（完全に伸ばしきる）しない**（膝を痛める）
- お尻がシートから浮かないように
- 足の裏全体で押す（つま先だけで押さない）

---

#### 5. アブドミナルマシン（3セット × 15回）

> 座った状態で体を丸めるマシン。腹筋を鍛えてお腹を引き締める。

| 設定 | 値 |
|------|------|
| セット数 | 3セット |
| 回数 | 15回 |
| セット間の休憩 | 60秒 |
| 重量 | マシンの最低重量 → 慣れたら +5kg ずつ |
| 鍛える場所 | お腹（腹直筋）・体幹 |

**やり方:**
1. シートに座り、足をパッドで固定する
2. ハンドルを握る（胸の前あたり）
3. 息を吐きながら、おへそを覗き込むように体を丸める
4. ゆっくり戻す → これで1回

**コツ:**
- 腕の力で引っ張らない。お腹で丸める意識
- 反動を使わない。ゆっくりコントロールする
- 息を吐きながら丸める、吸いながら戻す

---

### B日: ウォーキングのみ（40分）

> A日の筋トレの疲労を回復しながら、脂肪を燃やす日。

| 項目 | 設定 |
|------|------|
| 傾斜 | 10〜12% |
| 速度 | 5〜6 km/h |
| 時間 | 40分 |

やり方は A日のインクライン・ウォーキングと同じ。時間が長いだけ。

心拍数を脂肪燃焼ゾーン（120〜140bpm 目安）に維持。

---

## 重量の進め方（プログレッシブ・オーバーロード）

「15回3セットが楽にできる」ようになったら、次回から重量を上げる。

| 種目 | 初期重量 | 増加幅 |
|------|---------|--------|
| ベンチプレス | バーのみ（20kg） | +2.5kg |
| ラットプルダウン | 15kg | +5kg |
| レッグプレス | 30kg | +10kg |
| アブドミナル | マシン最低重量 | +5kg |

- 上げて15回できなくなったら、その重量で3セット15回できるまで続ける
- 焦らない。フォームが崩れるくらいなら重量を戻す

## 持ち物

- 着替え（Tシャツ + 短パン）
- タオル
- 水（500ml以上）
- イヤホン
- ジム用シューズ

## 習慣化のコツ

- トリガー: 昼食後 → そのままジムへ（毎回判断しない）
- 朝の時点でバッグを準備しておく
- 帰宅後シャワー → 午後の開発へ（リフレッシュ効果で集中力回復）
