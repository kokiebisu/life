# Phase 1: 手を作る

## ゴール

- 右手のフィンガーピッキングとパーカッシブ奏法の基盤を作る
- 左手の7thコードフォームとカッティングを身につける
- 「弾ける手」の土台を固め、以降のフェーズに備える

## このフェーズを終えると

- 指弾きでメロディ+伴奏を同時に弾ける
- ボディヒットやゴーストノートでリズムに立体感を出せる
- Maj7 / m7 / 7 のコードを自然に押さえられる
- 16分のカッティングでグルーヴを作れる

## レッスン一覧

| # | テーマ | 素材 |
|---|--------|------|
| L1 | フィンガーピッキング基礎 | 藤井風「きらり」 |
| L2 | パーカッシブ奏法 | John Mayer「Neon」 |
| L3 | 7thコードフォーム | 「Fly Me to the Moon」 |
| L4 | 16分カッティング | Neo-Soul グルーヴ |

## Notes

Phase 1 は全フェーズの土台。各レッスンの基礎パターンを体に染み込ませてから次へ進む。
