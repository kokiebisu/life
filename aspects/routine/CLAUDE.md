# 理想の1日テンプレート

退職後の生活リズム（2026年2月16日〜）

> **2/14(土):** デート日 🎉
> **2/15(日):** 教会 → ゆっくり過ごす

## 基本スケジュール

| 時間帯 | 内容 |
|--------|------|
| 朝（8:00-8:30） | 朝シャワー |
| 午前（9:00-12:00） | sumitsugi開発（集中タイム） |
| 昼（12:00-14:00） | 昼食 + ジム or 運動 |
| 午後（14:00-17:00） | sumitsugi開発（続き）or 営業活動 |
| 夕方（17:00-18:00） | ギター練習（1時間） |
| 夜（18:00-20:00） | study / 読書 / 投資リサーチ / 自由時間 |
| 就寝前 | 日記 |

## ポイント

- **午前は最も集中力が高い時間。** sumitsugi開発に充てる
- **運動は昼に挟む。** 午後の集中力回復にもなる
- **ギターは毎日1時間。** 夕方に固定して習慣化
- **夜は柔軟に。** その日の気分や進捗に合わせて調整

## 曜日別の調整例

- **月曜:** 週次プラン作成（朝30分）→ 通常スケジュール
- **水曜・金曜:** ジムの日。昼の運動を重めに
- **土曜:** sumitsugi開発は午前のみ。午後は自由時間・福岡リサーチなど
- **日曜:** 教会 → ゆっくり過ごす日。ギターと読書中心

## 注意

これはテンプレートであり、厳守するルールではない。曜日や体調、状況で柔軟に調整すること。大事なのは「だいたいこの流れ」が体に染み付くこと。
