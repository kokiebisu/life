# Lesson 8: 「満ちていく」通し演奏 --- 全コードを繋げて1曲弾く

Phase 2 / L8
担当: 瀬戸 涼介 + 黒田 奏
所要時間: 約75分
前提: L5-L7 完了（13コードすべて習得済み）

---

## はじめに

> 瀬戸: Phase 2 の集大成。L5 で m7 と M7(9)、L6 で dim とオルタード、L7 で分数コードを覚えた。合計13個。今日はそのすべてを繋げて「満ちていく」を1曲通して弾く。ただ順番に弾くだけじゃなくて、**ヴォイスリーディング**（声部誘導 = コード間を最小限の指移動で繋ぐ技術）を意識する。コードチェンジの「つなぎ目」が滑らかになると、バラバラだったコードが1つの音楽になる。最初はゆっくり BPM 50 でいくよ。

> 黒田: 理論パートでは、この曲のハーモニーがどういう仕組みで動いているか、鍵（Key）と各コードの役割を見ていくよ。藤井風の曲はジャズの語彙をポップスに落とし込んでいて、そこが面白い。「なぜこの並びで気持ちいいのか」を一緒に読み解こう。

## ① TABで弾く

### STEP 1: コード進行マップ

まず、「満ちていく」のコード進行を全体で把握しよう。4つのセクションに分かれている。

**使うコード一覧（13個）:**

```
C#m7:      x-4-6-4-5-4
F#m7:      2-4-2-2-2-2
DM7(9):    x-x-0-2-2-0
Fdim:      x-x-3-1-0-1
F#dim:     x-x-4-2-1-2
C7(-5):    x-3-2-3-1-2
E7(♭9):   0-2-0-1-0-1
F#7(♭9):  2-4-2-3-2-3
A7(13):    x-0-2-0-2-2
A7(♭13):  x-0-2-0-2-1
Bm7onE:    0-2-0-2-0-2
Gm6onA:    x-0-5-3-3-0
EonF#:     2-2-2-1-0-0
```

**コード進行マップ:**

```
--- Intro（4小節）---
| DM7(9)    | A7(13)    | Bm7onE    | C#m7      |

--- Verse（6小節）---
| DM7(9)    | A7(♭13)   | F#m7      | C#m7      |
| Bm7onE    | EonF#     |

--- Chorus（7小節）---
| DM7(9)    | Fdim      | F#m7      | E7(♭9)    |
| A7(13)    | Gm6onA    | F#7(♭9)  |

--- Bridge（4小節）---
| C7(-5)    | F#dim     | DM7(9)    | A7(♭13)   |
```

> 瀬戸: 全部で21小節。1小節4拍として、BPM 50 だと約100秒。2分弱で1周できる計算。最初はセクションごとにバラして練習するから安心して。

---

### STEP 2: セクションごとに練習

セクション単位で、まずはコードチェンジだけに集中する。ストロークは全拍ダウンでOK。

**Intro の練習:**

```
BPM 50 / ストレート / 全拍ダウンストローク

Intro:
| DM7(9)    | A7(13)    | Bm7onE    | C#m7      |

DM7(9):     x-x-0-2-2-0    → 開放弦が多いので楽
A7(13):     x-0-2-0-2-2    → 5弦開放がルートA
Bm7onE:     0-2-0-2-0-2    → 6弦開放がベースE
C#m7:       x-4-6-4-5-4    → ここだけハイポジション
```

> 瀬戸: Intro は DM7(9) から始まる。開放弦が3つ入ってるから、押さえるのは4弦2フレットと3弦2フレットだけ。そこから A7(13) → Bm7onE までは5フレット以下のローポジション（低い位置）で済む。問題は4小節目の C#m7。一気に4-6フレット帯に飛ぶ。ここは「3小節目の最後の拍で左手を準備し始める」のがコツ。

**やること:** Intro を8回繰り返す（BPM 50）

---

**Verse の練習:**

```
BPM 50 / ストレート

Verse:
| DM7(9)    | A7(♭13)   | F#m7      | C#m7      |
| Bm7onE    | EonF#     |

DM7(9):     x-x-0-2-2-0
A7(♭13):   x-0-2-0-2-1    → A7(13) から1弦を1フレット下げただけ
F#m7:       2-4-2-2-2-2    → バレーコード。人差し指で1-5弦2フレットをセーハ
C#m7:       x-4-6-4-5-4
Bm7onE:     0-2-0-2-0-2
EonF#:      2-2-2-1-0-0    → 最低音が2弦のF#。独特の響き
```

> 瀬戸: Verse のポイントは F#m7 のバレーコード。人差し指で2フレットを全部押さえて（セーハ = 1本の指で複数弦をまとめて押さえる技術）、5弦4フレットに中指を置く。L5 で練習したから大丈夫だよね。DM7(9) → A7(♭13) → F#m7 の流れは、ローポジション → ローポジション → 2フレットのバレーだから、移動距離は少ない。

**やること:** Verse を8回繰り返す（BPM 50）

---

**Chorus の練習:**

```
BPM 50 / ストレート

Chorus:
| DM7(9)    | Fdim      | F#m7      | E7(♭9)    |
| A7(13)    | Gm6onA    | F#7(♭9)  |

DM7(9):     x-x-0-2-2-0
Fdim:       x-x-3-1-0-1    → 4弦・3弦・2弦で不穏な響き
F#m7:       2-4-2-2-2-2
E7(♭9):    0-2-0-1-0-1    → E7 にちょっとスパイスを足した音
A7(13):     x-0-2-0-2-2
Gm6onA:     x-0-5-3-3-0    → 5弦開放がベースA。3-5フレットの押さえ方
F#7(♭9):   2-4-2-3-2-3    → F#m7 から中指の位置を変える
```

> 瀬戸: Chorus は一番コードの種類が多い。7小節で7個のコードが出てくる。特に Fdim → F#m7 の流れと、Gm6onA → F#7(♭9) の流れが指の移動が大きい。最初は間に合わなくてもいいから、右手のストロークだけは止めない。ミュート（弦に軽く触れて音を消した状態）のまま弾いて、次のコードの頭から復帰する。

**やること:** Chorus を8回繰り返す（BPM 50）

---

**Bridge の練習:**

```
BPM 50 / ストレート

Bridge:
| C7(-5)    | F#dim     | DM7(9)    | A7(♭13)   |

C7(-5):     x-3-2-3-1-2    → 5弦3フレットがルートC。複雑な形だけど指4本で押さえる
F#dim:      x-x-4-2-1-2    → Fdim を1フレット上にスライドした形
DM7(9):     x-x-0-2-2-0    → ここで開放弦に戻ってホッとする
A7(♭13):   x-0-2-0-2-1
```

> 瀬戸: Bridge は4小節だけだから短い。C7(-5) は形がちょっと複雑だけど、L6 で練習した形そのまま。F#dim も Fdim の形を1フレットずらすだけ。後半は DM7(9) → A7(♭13) で Intro の雰囲気に戻る。「帰ってきた」感じがする進行だね。

**やること:** Bridge を8回繰り返す（BPM 50）

### STEP 3: ヴォイスリーディング（Voice Leading）

ここが今日の一番大事なパート。

**ヴォイスリーディングとは:** コードを切り替えるとき、指を全部いっぺんに動かすんじゃなくて、**動かさなくていい指はそのままにして、必要な指だけ最小限に動かす**こと。声部（各弦の音）が滑らかに繋がるから、コードチェンジが「ブツッ」と途切れずに流れるようになる。

---

**パターンA: A7(13) → A7(♭13) — 指1本、1フレット**

```
A7(13):     x-0-2-0-2-2        A7(♭13):   x-0-2-0-2-1
e  --2--  ← 中指                e  --1--  ← 人差し指（1フレット下がるだけ！）
B  --2--  ← 薬指                B  --2--  ← 薬指（動かない）
G  --0--  ← 開放                G  --0--  ← 開放（動かない）
D  --2--  ← 小指                D  --2--  ← 小指（動かない）
A  --0--  ← 開放                A  --0--  ← 開放（動かない）
E  --x--                        E  --x--

動くのは: 1弦だけ（2f → 1f）
残りの5弦は全部そのまま
```

> 瀬戸: これが一番わかりやすいヴォイスリーディングの例。A7(13) と A7(♭13) は1弦の音が半音違うだけ。小指で1弦2フレットを押さえていたのを、人差し指で1フレットに変える。たった1フレットの移動で、響きが「明るい」から「ちょっと翳りがある」に変わる。Verse の2小節目で使うよ。

**やること:** A7(13) → A7(♭13) を交互に10回。1弦の音色の変化を耳で感じる

---

**パターンB: Fdim → F#dim — 形ごとスライド**

```
Fdim:       x-x-3-1-0-1        F#dim:      x-x-4-2-1-2
e  --1--  ← 人差し指            e  --2--  ← 人差し指（+1f）
B  --0--  ← 開放                B  --1--  ← 人差し指（+1f）
G  --1--  ← 中指                G  --2--  ← 中指（+1f）
D  --3--  ← 薬指                D  --4--  ← 薬指（+1f）
A  --x--                        A  --x--
E  --x--                        E  --x--

動くのは: 全部の指が1フレットずつ上がる
形はまったく同じ。スライドするだけ。
```

> 瀬戸: dim コードの面白い特徴。形が同じまま平行移動（スライド）するだけで次のコードになる。Bridge の C7(-5) → F#dim のところで使う。左手の形をキープしたまま、指板の上を滑らせる感覚。

**やること:** Fdim → F#dim を交互に10回。形を崩さずスライドする

---

**パターンC: DM7(9) → Fdim — 開放弦からの切り替え**

```
DM7(9):     x-x-0-2-2-0        Fdim:       x-x-3-1-0-1
e  --0--  ← 開放                e  --1--  ← 人差し指
B  --2--  ← 中指                B  --0--  ← 開放
G  --2--  ← 薬指                G  --1--  ← 中指
D  --0--  ← 開放                D  --3--  ← 薬指
A  --x--                        A  --x--
E  --x--                        E  --x--

ポイント:
- 3弦: 2f(薬指) → 1f(中指) = 1フレット下がるだけ
- 4弦: 開放 → 3f(薬指) = 薬指が3弦から4弦に移動
- 1弦: 開放 → 1f(人差し指) = 人差し指を置くだけ
- 2弦: 2f(中指) → 開放 = 中指を離すだけ
```

> 瀬戸: DM7(9) は開放弦が多くて「手が楽」なコード。そこから Fdim に移るとき、全部の指を同時に動かそうとするとパニックになる。コツは**3弦と4弦を先に動かして、1弦は最後に置く**。Chorus の最初のコードチェンジだから、ここが滑らかに繋がると一気に気持ちよくなる。

**やること:** DM7(9) → Fdim を交互に10回。3弦→4弦→1弦の順で指を動かす

---

**パターンD: C#m7 → DM7(9) — 大きなジャンプ**

```
C#m7:       x-4-6-4-5-4        DM7(9):     x-x-0-2-2-0
e  --4--  ← 小指                e  --0--  ← 開放
B  --5--  ← 小指                B  --2--  ← 中指
G  --4--  ← 中指                G  --2--  ← 薬指
D  --6--  ← 薬指                D  --0--  ← 開放
A  --4--  ← 人差し指            A  --x--
E  --x--                        E  --x--

ポイント:
- 4-6フレット帯から一気に開放弦帯に戻る
- 「全部離す」動作。手全体をネック上方に移動。
- 最後の拍の裏で左手を浮かせ始めて、次の拍頭で DM7(9) に着地
```

> 瀬戸: これが一番の難所。C#m7 はハイポジション（4-6フレット）で、DM7(9) は開放弦。移動距離が大きい。ここだけは「ヴォイスリーディングで滑らかに」っていうより、**思い切ってパッと手を離す**方がいい。C#m7 の最後の拍の裏拍あたりで左手をフワッと浮かせて、開放弦が一瞬鳴る→すぐ DM7(9) を押さえる。その一瞬の開放弦の響きが、意外と綺麗な「つなぎ」になるんだ。

**やること:** C#m7 → DM7(9) を交互に10回。「浮かせる→着地」のタイミングを安定させる

---

**パターンE: F#m7 → E7(♭9) — バレーからローポジション**

```
F#m7:       2-4-2-2-2-2        E7(♭9):    0-2-0-1-0-1
e  --2--  ← セーハ              e  --1--  ← 人差し指
B  --2--  ← セーハ              B  --0--  ← 開放
G  --2--  ← セーハ              G  --1--  ← 中指
D  --2--  ← セーハ              D  --0--  ← 開放
A  --4--  ← 中指                A  --2--  ← 薬指
E  --2--  ← セーハ              E  --0--  ← 開放

ポイント:
- セーハを全部離す → 1弦1f, 3弦1f, 5弦2f だけ押さえる
- 2フレットのバレーから0-2フレットに落ちるので距離は近い
```

> 瀬戸: F#m7 のセーハを解除して、E7(♭9) の形に切り替える。セーハ（バレー）を「パッ」と離して、1弦・3弦・5弦だけポンポンポンと押さえる。Chorus の3小節目→4小節目がここ。

**やること:** F#m7 → E7(♭9) を交互に10回

---

### STEP 4: 通し演奏

全セクションを繋げて1曲通す。

```
BPM 50 / ストレート / 全拍ダウンストローク

--- Intro ---
| DM7(9)    | A7(13)    | Bm7onE    | C#m7      |

--- Verse ---
| DM7(9)    | A7(♭13)   | F#m7      | C#m7      |
| Bm7onE    | EonF#     |

--- Chorus ---
| DM7(9)    | Fdim      | F#m7      | E7(♭9)    |
| A7(13)    | Gm6onA    | F#7(♭9)  |

--- Bridge ---
| C7(-5)    | F#dim     | DM7(9)    | A7(♭13)   |
```

> 瀬戸: いよいよ通し。BPM 50 でゆーっくり弾く。止まっても大丈夫、右手だけは動かし続けて。コードチェンジが間に合わなかったらミュートのままストロークして、次の小節の頭から復帰する。**完璧に弾くことが目標じゃなくて、「21小節を途中で止まらずに最後まで行く」のが今日のゴール。**

**やること:**
1. BPM 50 で通しを3回弾く
2. 間に合わなかったコードチェンジをメモしておく
3. そのチェンジだけ STEP 3 の方法で10回ずつ反復
4. もう一度通しを3回弾く

> 瀬戸: 2回目の通しでは、さっきよりスムーズになってるはず。1回目で「ここがキツい」って感じた場所を潰してから再挑戦するのが一番効率いい。

## ② コード理論

> 黒田: ここからは「今弾いた曲のハーモニーの正体」を見ていくよ。

### この曲のキー（調）

> 黒田: 「満ちていく」のキーは**A メジャー（イ長調）**がベースになっている。A メジャーの基本コード（ダイアトニックコード = その調の音だけで作ったコード）はこう:

```
A メジャーのダイアトニックコード（7th入り）:

I    = AM7       （トニック / 家）
ii   = Bm7       （サブドミナント系）
iii  = C#m7      （トニック代理）
IV   = DM7       （サブドミナント / 散歩）
V    = E7        （ドミナント / 帰りたくなる力）
vi   = F#m7      （トニック代理 / 平行短調の主役）
vii° = G#m7(♭5)  （ドミナント系）
```

> 黒田: 曲の中で DM7(9) が何度も出てくるよね。これはダイアトニックの IV 度（サブドミナント）。DM7 に 9th のテンション（E の音）を足したのが DM7(9)。A メジャーのキーの中で一番「明るくて、ちょっと浮遊感がある」ポジションにいるコード。

---

### 各コードの役割（ファンクション）

> 黒田: 1コードずつ、「この曲の中でどんな役割を果たしているか」を見ていこう。

```
コード        | 機能            | 解説
--------------+-----------------+------------------------------------------
DM7(9)        | IV（サブドミ）   | 浮遊感のある出発点。曲のホームポジション
A7(13)        | I7（トニック系） | 本来 AM7 だが、7th をドミナント化。ブルージーな色
A7(♭13)      | I7（変化形）     | 13th を半音下げて翳りを加えた A7
C#m7          | iii（トニック代理）| F#m7 と並ぶ「もう一つの安定点」
F#m7          | vi（平行短調）   | A メジャーの平行短調 F#m の主役。切なさ担当
Bm7onE        | ii/V（ツーオンファイブ）| Bm7 のコードをベース E で弾く。V 度の響きを持つ
EonF#         | V/vi（経過音）   | E コードを F# ベースで弾く。F#m7 に向かう経過
E7(♭9)       | V7alt（ドミナント）| A に帰る力。♭9 のテンションが緊張を生む
F#7(♭9)      | VI7（セカンダリーD）| Bm7 に解決するためのセカンダリードミナント
Fdim          | ♭VIdim（経過dim）| DM7 と F#m7 を半音で繋ぐ「橋渡し」
F#dim         | VIdim（経過dim） | 同じく半音の橋渡し。Fdim の1フレット上
C7(-5)        | ♭III7（借用）    | ブルースから借りてきたコード。異世界感
Gm6onA        | ♭VIIm6/I（借用）| A のペダルポイント上にGm6 を載せた響き
```

---

### ジャズから借りてきた手法

> 黒田: この曲のすごいところは、ポップスのメロディの下に**ジャズの語彙**が隠れていること。特に3つの手法が目立つ。

**1. セカンダリードミナント（Secondary Dominant）**

> 黒田: F#7(♭9) は、本来 A メジャーの中にはないコード。でも「F#7 → Bm7」と動くと、一時的に Bm7 がトニックになったかのような「解決感」が生まれる。これが「セカンダリードミナント」。一瞬だけ別のキーに寄り道して、すぐ戻ってくる技術。ジャズでは日常的に使う。

**2. ディミニッシュ経過（Passing Diminished）**

> 黒田: Fdim と F#dim は「経過ディミニッシュ」と呼ばれる使い方。DM7 → Fdim → F#m7 を見てみると、ベース音が D → F → F# と半音ずつ上がっていく。dim コードが2つのコードの間を「半音の階段」で繋いでくれるんだ。これもジャズの定番テクニック。

**3. 借用コード（Modal Interchange / 借用和音）**

> 黒田: C7(-5) は A メジャーのダイアトニックにはないコード。これは「A マイナー」や「A ブルース」の世界から借りてきた響き。Gm6onA も同じで、A メジャーの中に突然マイナーの色が差し込まれる。この「明→暗→明」の切り替えが、「満ちていく」の独特の奥行きを作ってる。

> 瀬戸: 理論の話が長くなったけど、大事なのは「ふーん、そういう仕組みなんだ」くらいでOK。弾いてるうちに体が覚えるから。「この流れ気持ちいいな」って感じてたら、それが一番正しい理解だよ。

## ③ 応用ドリル

### ドリル 1: コードチェンジ・スピード練習（メトロノームで追い込む）

メトロノームに合わせて、**2拍ごとにコードを切り替える**。通常は1小節（4拍）に1コードだから、これは倍速チェンジの練習。

```
BPM 50 / ストレート / 2拍で切り替え

パターン1（Intro のチェンジ）:
```

### ドリル 2: 目を閉じて弾く（マッスルメモリー）

STEP 4 の通し演奏を、**目を閉じて**弾く。

```
手順:
1. 楽譜を見て Intro を弾く → 目を閉じて同じ Intro を弾く
2. 楽譜を見て Verse を弾く → 目を閉じて同じ Verse を弾く
3. Chorus, Bridge も同様
4. 最後に全セクション目を閉じて通す
```

> 瀬戸: 「目を閉じて弾けるか」はマッスルメモリー（筋肉の記憶）のテスト。指が覚えてたら目を閉じても弾ける。弾けなかったらまだ「目で確認しながら弾いてる」段階。目を閉じると最初は不安だけど、失敗してもいいから試してみて。**意外と弾けるもんだよ。**

---

### ドリル 3: メロディをハミングしながら弾く

コードを弾きながら、「満ちていく」のメロディを口ずさむ。

```
手順:
1. まずコードだけ弾く（BPM 50）
2. 次にメロディだけ歌う（コードなし）
3. 両方同時にやる
4. 最初は Intro だけ → 慣れたら Verse → Chorus → 通し
```

> 瀬戸: これができたら「弾き語り」の第一歩。最初はコードとメロディのタイミングがバラバラになるけど、コード進行が体に入ってれば自然に合ってくる。メロディがわからない部分は「ンーンー」ってハミングでいいよ。コードの響きに寄り添って声を出す感覚。

## ④ 自分で組み合わせる

### チャレンジ 1: オリジナル4コード進行を作る

「満ちていく」の13コードの中から好きな4つを選んで、自分だけの4小節を作ってみよう。

**使えるコード（全13個）:**

```
DM7(9), A7(13), A7(♭13), C#m7, F#m7,
Bm7onE, EonF#, E7(♭9), F#7(♭9),
Fdim, F#dim, C7(-5), Gm6onA
```

**ヒント — 響きの方向性で選ぶ:**

```
キラキラ系:    DM7(9) → A7(13) → F#m7 → Bm7onE
ダーク系:      C7(-5) → F#dim → E7(♭9) → C#m7
切ない系:      F#m7 → A7(♭13) → DM7(9) → EonF#
ミステリアス系: Gm6onA → Fdim → C7(-5) → DM7(9)
```

> 瀬戸: 正解はないよ。4つ選んで弾いてみて、「おっ、これいい」って感じたらそれが正解。コードネームの見た目が難しそうでも、L5-L7 で全部弾いてるから指は覚えてるはず。好きなコードを並べてループさせるだけで、オリジナルの雰囲気が出る。

---

### チャレンジ 2: セクションの順番を変えて弾く

「満ちていく」のセクションをシャッフル（並べ替え）してみよう。

```
例1: Chorus → Verse → Bridge → Intro（クライマックスから始める）
例2: Bridge → Intro → Chorus → Verse（不思議な雰囲気で始める）
例3: Intro → Chorus → Intro → Bridge（Verse を飛ばしてみる）
```

> 瀬戸: 曲の構成を変えるだけで全然違う印象になる。Chorus から始めると「いきなりサビ」感があるし、Bridge から始めると「どこに行くんだろう」っていう浮遊感が出る。作曲家がどうやってセクションの順番を決めてるか、体感でわかるようになるよ。
