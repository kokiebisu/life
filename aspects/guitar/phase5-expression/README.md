# Phase 5: Expression（表現）

自分の音を見つけるフェーズ。技術を超えて、音楽で何を伝えるか。

## Goals

このフェーズを終えると:
- 自分だけのサウンドがある
- 曲を自分のスタイルでアレンジできる
- オリジナル曲を作れる
- 人と一緒に演奏を楽しめる

## Topics

### 1. ダイナミクスとタッチ
- 強弱の幅を広げる
- ピッキングのニュアンス
- 指弾きとピック弾きの使い分け
- John Mayerのタッチを研究

### 2. フレージングとブレス
- 音を「歌う」とは
- 間の取り方
- フレーズの始め方と終わり方
- 歌詞のように弾く

### 3. アレンジ
- カバー曲を自分のスタイルに
- シンプルな曲を豊かにする
- 複雑な曲をシンプルに再解釈
- ソロギターアレンジ

### 4. 作曲・編曲
- コード進行を作る
- メロディを乗せる
- 曲の構成（Verse, Chorus, Bridge）
- アレンジの引き出し

### 5. アンサンブル
- 他の楽器との対話
- バンドでのギターの役割
- セッションのマナー
- 聴くことの大切さ

## Practice

### 毎日意識すること
- どんな音を出したいか考えてから弾く
- 録音して聴き返す
- 好きなギタリストのタッチを真似る

### 挑戦
- オープンマイクに参加
- セッションに行く
- 誰かとジャムする
- オリジナル曲を録音する

## Reflection Questions

> 「今日、何を表現したかった？」
> 「聴いてくれた人に何を感じてほしかった？」
> 「自分の演奏のどこが好き？」

## Notes

表現についての気づき、アレンジのアイデアを記録。
