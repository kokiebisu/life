# fashion

ワードローブ管理。所有アイテムを `wardrobe.md` に一元化し、シルエット・素材・着こなしメモを残しておく。

## Overview

ミニマリスト方針で「年中使える一枚」を厳選する。コーデ内で同一ブランドが偶発的に揃うのを避ける（ミックス必須）。新しいアイテムを買う前に `wardrobe.md` を確認し、被りや不足を判断する。買いたいものは [../shopping/stores/](../shopping/stores/) に記録する。

## What's here

```
wardrobe.md     所有アイテム（アウター / トップス / ボトムス など）
images/         アイテムの参考画像
```
