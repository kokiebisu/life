# Phase 3: スケール＆リック

## ゴール

- ソロで使えるスケールの語彙を体系的に構築する
- ペンタトニック、ブルース、モードスケールを実践で使えるようにする
- Speed Drill で早弾き技術（オルタネイト、レガート、スウィープ）を鍛える

## このフェーズを終えると

- ペンタ+ブルーノートで自然なソロが弾ける
- ブルース進行の上でアドリブソロが取れる
- ドリアン・ミクソリディアンを場面に応じて使い分けられる
- コードトーンとアルペジオでコード感のあるソロが弾ける
- オルタネイト・レガート・スウィープの基礎速度が上がっている

各レッスンに Speed Drill を組み込み、スケール知識とピッキング技術を同時に伸ばす。BPM を記録して成長を可視化する。
