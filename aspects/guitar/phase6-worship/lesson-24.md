## ① TABで弾く

### STEP 1: Key=D のダイアトニック5コードの「性格」を体感する

L21 で学んだ通り、Key of D のダイアトニックコードは:

```
度数:   I      ii     iii    IV     V      vi     vii
コード: D      Em     F#m    G      A      Bm     C#m7♭5
カポ2:  C      Dm     Em     F      G      Am     Bm7♭5
```

この中でワーシップで使う5つ（I, ii, IV, V, vi）には、それぞれ固有の「性格」がある。その性格の正体が**モード（mode = 旋法）**。

> 黒田: モードって何かというと、**同じ7つの音を使って「スタート地点」を変えたスケール**のこと。D メジャースケール（D E F# G A B C#）の音を全部使うけど、D から始めるか、E から始めるか、G から始めるかで、響きの「色」が全然変わる。

**体感エクササイズのやり方:**

```
① コードをジャラーンと鳴らす（カポ2の形）
② そのコードを鳴らしたまま、モードスケールを単音で弾く
③ 響きの「色」を感じる — どんな感情？ どんな風景？
```

#### I (D) = Ionian（アイオニアン = メジャースケール）

コード: C shape（カポ2 = D）

```
Cadd9:
e --0--
B --3--
G --0--
D --2--
A --3--
E -----
```

**C Ionian スケール（カポ2、0-3フレット）:**

```
上昇:
e ----------------0--1--3--
B -----------0--1--3-------
G --------0--2-------------
D -----0--2--3-------------
A --0--2--3----------------
E -------------------------

下降:
e --3--1--0----------------
B -----------3--1--0-------
G ----------------2--0-----
D --------------------3--2--0--
A ---------------------------3--2--0--
E -------------------------------------
```

> 瀬戸: Cadd9 をジャラーンと鳴らして、その余韻が残ってるうちにスケールを弾いてみて。

> 黒田: どう感じた？ 明るくて、安定していて、「家に帰ってきた」ような安心感。これが Ionian の色。**ワーシップの賛美の大半はこの色**。「神は良いお方」「感謝します」系の曲はほぼ Ionian。

```
特徴音: なし（= メジャースケールそのもの。基準になる「普通」の響き）
感情:  明るい・安定・ホーム。「ただいま」の感じ
Worship: 賛美の土台。安心感のある曲
```

#### ii (Em) = Dorian（ドリアン）

コード: Dm shape（カポ2 = Em）

```
Dm7:
e --1--
B --1--
G --2--
D --0--
A -----
E -----
```

**D Dorian スケール（カポ2、同じ C メジャーの7音を D から開始）:**

```
上昇:
e -------------------0--1--3--
B --------------1--3----------
G ---------0--2---------------
D -----0--2--3----------------
A --0--2--3-------------------
E ----------------------------

下降:
e --3--1--0-------------------
B -----------3--1-------------
G ----------------2--0--------
D --------------------3--2--0--
A --------------------------3--2--0--
E ---------------------------------------
```

> 瀬戸: Dm7 を鳴らして、そのまま D から始めるスケールを弾く。C Ionian と全く同じ音なのに、D から始めるだけで雰囲気がガラッと変わるのがわかる？

> 黒田: 切ないんだけど、どこか希望がある。普通のマイナー（Aeolian）よりも「暗すぎない」。これが Dorian の特徴。秘密は**ナチュラル6度（♮6 = B の音）**。

```
普通のマイナー（D Aeolian）: D E F G A B♭ C D  ← 6度が♭（B♭）
Dorian:                     D E F G A B  C D  ← 6度がナチュラル（B♮）
                                     ↑
                                ここが明るさの源！
```

```
特徴音: ♮6（ナチュラル6度 = B♮）→ 普通のマイナーより明るい
感情:  切ないけど希望がある。マイナーだけど暗すぎない
Worship: マイナー調の賛美、悔い改めの中にも光がある場面
```

#### IV (G) = Lydian（リディアン）

コード: F shape（カポ2 = G）

```
Fsus2:
e --1--
B --1--
G --0--
D --3--
A --3--
E --1--
```

**F Lydian スケール（カポ2、同じ C メジャーの7音を F から開始）:**

```
上昇:
e --------------------------0--1--3--
B --------------------0--1--3--------
G ---------------0--2----------------
D ----------0--2--3------------------
A -----0--2--3----------------------
E --1--3-----------------------------

下降:
e --3--1--0--------------------------
B -----------3--1--0-----------------
G ------------------2--0-------------
D -----------------------3--2--0-----
A ----------------------------3--2--0--
E ---------------------------------3--1--
```

> 瀬戸: F のコードを鳴らしてから、F の音（1弦1フレット）から上昇スケールを弾く。普通のメジャースケール（Ionian）とどこか違う感じ、わかる？

> 黒田: 浮遊感。神秘的。「天が開く」ような広がり。これが Lydian。秘密は**シャープ4度（#4 = B♮）**。普通の F メジャースケールなら4度は B♭ なんだけど、Lydian では B♮ になる。たった半音の違いだけど、響きは劇的に変わる。

```
普通のメジャー（F Ionian）: F G A B♭ C D E F  ← 4度が普通（B♭）
Lydian:                    F G A B  C D E F  ← 4度が#（B♮）
                                ↑
                           浮遊感の正体！
```

```
特徴音: #4（シャープ4度 = B♮）→ 浮遊感・神秘的
感情:  浮遊・神秘的・広がり。「天が開く」感じ
Worship: 神秘的な瞬間、ワーシップの高揚パート
```

#### V (A) = Mixolydian（ミクソリディアン）

コード: G shape（カポ2 = A）

```
G worship:
e --3--
B --3--
G --0--
D --0--
A --2--
E --3--
```

**G Mixolydian スケール（カポ2、同じ C メジャーの7音を G から開始）:**

```
上昇:
e -----------------------0--1--3--
B -----------------0--1--3--------
G ------------0--2----------------
D -------0--2--3------------------
A --2--3--------------------------
E --3-----------------------------

下降:
e --3--1--0-----------------------
B -----------3--1--0--------------
G ------------------2--0----------
D -----------------------3--2--0--
A ----------------------------3--2--
E --------------------------------3--
```

> 瀬戸: G worship をジャラーンと鳴らして、G（6弦3フレット）からスケールを弾く。Ionian（C から始めるやつ）とは違う「開放感」が聴こえない？

> 黒田: メジャーなんだけど、どこか「未解決」。完全に明るくなりきらない。旅の途中、まだ家に着いてない感じ。これが Mixolydian の色。秘密は**フラット7度（♭7 = F♮）**。

```
普通のメジャー（G Ionian）: G A B C D E F# G  ← 7度がシャープ（F#）
Mixolydian:                G A B C D E F  G  ← 7度がフラット（F♮）
                                      ↑
                                 「未解決」の正体！
```

```
特徴音: ♭7（フラット7度 = F♮）→ メジャーなのに完全には解決しない
感情:  開放感・未解決。「旅の途中」感
Worship: Hillsong / Bethel 系。♭7 のロック的な響き
```

#### vi (Bm) = Aeolian（エオリアン = ナチュラルマイナースケール）

コード: Am shape（カポ2 = Bm）

```
Am7:
e --0--
B --1--
G --0--
D --2--
A --0--
E -----
```

**A Aeolian スケール（カポ2、同じ C メジャーの7音を A から開始）:**

```
上昇:
e -------------------------0--1--3--
B -------------------1--3----------
G -------------0--2----------------
D --------0--2--3------------------
A --0--2--3------------------------
E ---------------------------------

下降:
e --3--1--0------------------------
B -----------3--1------------------
G ----------------2--0-------------
D ---------------------3--2--0-----
A --------------------------3--2--0--
E ------------------------------------
```

> 瀬戸: Am7 を鳴らして、A（5弦開放）からスケールを弾く。Dorian（D から始めるやつ）と比べてみて。同じマイナー系なのに、Aeolian の方がもっと暗い、もっと内側に向かう感じがしない？

> 黒田: Aeolian は**ナチュラルマイナースケール**そのもの。最も「悲しい」マイナーの響き。Dorian と比べると、6度がフラットしている（♭6 = F♮）のが違い。

```
Dorian:   A B C D E F# G A  ← 6度がナチュラル（F#）→ 明るめマイナー
Aeolian:  A B C D E F  G A  ← 6度がフラット（F♮）→ 暗いマイナー
                     ↑
                Dorian との違い！
```

```
特徴音: ♭3 + ♭6 → Dorian より暗い
感情:  悲しい・内省的。「祈り」の深さ
Worship: 悔い改め・嘆きの曲。詩篇の「深い淵から」的な場面
```

### STEP 2: 感情→コード変換の実践

5つのモードの「性格」を体感したら、次はその性格を使い分ける練習。

同じ Key of D（カポ2 = C シェイプ）で、3つのコード進行を弾き比べてみよう。**進行が違うだけで感情が全然変わる**のを体で感じて。

#### 進行 A:「安定・安心」— Ionian 中心

```
| Cadd9 (I)  | Fsus2 (IV) | G wor. (V) | Cadd9 (I)  |
  Ionian       Lydian       Mixolydian   Ionian

ストラム: ↓  ↓↑ ↑↓↑  （各コード4拍）

※ I で始まり I で終わる。一番「安心」する進行。
  「神さまは良いお方、感謝します」の空気感。
```

#### 進行 B:「切ないけど希望」— Dorian 強調

```
| Cadd9 (I)  | Dm7 (ii)   | Fsus2 (IV) | Cadd9 (I)  |
  Ionian       Dorian       Lydian       Ionian

ストラム: ↓  ↓↑ ↑↓↑  （各コード4拍）

※ ii（Dm7 = Dorian）を経由することで、切なさが入る。
  でも IV → I で明るく帰ってくる。
  「試練の中でも主が共にいてくださる」の空気感。
```

#### 進行 C:「神秘・浮遊」— Lydian ペダル

```
| Cadd9 (I)  | Fsus2 (IV) | Cadd9 (I)  | Fsus2 (IV) |
  Ionian       Lydian       Ionian       Lydian

ストラム: ↓  ↓↑ ↑↓↑  （各コード4拍）

※ I と IV だけを行ったり来たり。V に行かないから「解決しない」。
  Lydian の浮遊感がずっと漂う。
  祈りの時間、聖霊が動いている瞬間の空気感。
```

> 瀬戸: 3つ全部弾いてみて。使ってる音は全部 C メジャーの7音だけなのに、進行が変わるだけで全然違う感情が生まれるでしょ？ これがモードの力。「どのコードを選ぶか = どの感情を選ぶか」ってこと。

> 黒田: ワーシップのリーダーが「ここは静かに祈ろう」と言ったら → 進行 C（Lydian 浮遊）。「主の恵みを喜ぼう」なら → 進行 A（Ionian 安定）。「試練を経て希望へ」なら → 進行 B（Dorian 切なさ + 希望）。感情を音でコントロールできるようになる。

### STEP 3: As the Deer のコード進行をモード的に分析する

L22 で弾いた As the Deer の進行を、モードの「性格」でラベリングしてみよう。

**オリジナル進行（カポ2）:**

```
Verse:
| C (I)       | G (V)       | Am (vi)     | F (IV)      |
  Ionian        Mixolydian    Aeolian       Lydian
  安定           開放          切ない        神秘

| C (I)       | G (V)       | Am (vi)     | F (IV)      |
  安定           開放          切ない        神秘

Chorus:
| F (IV)      | C (I)       | G (V)       | Am (vi)     |
  Lydian        Ionian        Mixolydian    Aeolian
  神秘           安定          開放          切ない

| F (IV)      | C (I)       | G (V)       |             |
  神秘           安定          開放→着地

Bridge:
| Am (vi)     | G (V)       | F (IV)      | G (V)       | C (I)       |
  Aeolian       Mixolydian    Lydian        Mixolydian    Ionian
  切ない→        開放→         神秘→         開放→         帰還（安定）
```

> 黒田: ほら、見えてきた？ As the Deer の Verse は「安定 → 開放 → 切ない → 神秘」って感情が4小節で一周してる。Chorus は同じ4コードだけど順番が変わって「神秘 → 安定 → 開放 → 切ない」。**コードの並び順が変わると、感情の流れが変わる**。

> 瀬戸: Bridge は「切ない（Am）」から始まって、最後に I（C）に「帰ってくる」。これが曲のクライマックス — 暗いところから光へ向かう動き。モードの性格を知ってると、この「暗→明」の流れが見えるようになる。

#### リハーモナイズ実験

モードの性格がわかったら、**コード進行を意図的に組み替え**て曲の雰囲気を変えることができる。

**実験 1: V → I の終わり方を IV → I に変える**

```
オリジナル Verse 最終行:
| Dm (ii)     | G (V)       | C (I)       |
  Dorian        Mixolydian    Ionian
  「準備 → 緊張 → 解決」= 力強い着地

リハーモナイズ版:
| Dm (ii)     | F (IV)      | C (I)       |
  Dorian        Lydian        Ionian
  「準備 → 浮遊 → 解決」= ふわっと着地

弾き比べてみよう:
① | Dm7  | G wor. | Cadd9 |  ← V経由（力強い）
② | Dm7  | Fsus2  | Cadd9 |  ← IV経由（浮遊感）
```

> 瀬戸: ①は「ジャーン！帰ってきた！」って感じ。②は「ふわぁ……帰ってきた」って感じ。どっちも I に解決してるけど、途中のモードの色が違うだけで着地感が全然違う。

**実験 2: ii を V の前に入れる（ii → V → I）**

```
オリジナル Chorus 最終行:
| G (V)       | C (I)       |
  開放           安定

リハーモナイズ版:
| Dm (ii)     | G (V)       | C (I)       |
  Dorian        Mixolydian    Ionian
  切なさ→        開放→         帰還

弾き比べてみよう:
① | G wor. | Cadd9 |           ← V→I のみ（シンプルな解決）
② | Dm7    | G wor. | Cadd9 |  ← ii→V→I（「憧れ」を経由した解決）
```

> 黒田: ii → V → I は L21 でも出てきた「音楽で最も重要な進行」。ii（Dorian = 切なさ + 希望）を挟むことで、「ただ帰る」じゃなくて「憧れて、求めて、ようやく帰る」という感情の深みが出る。

**実験 3: vi で始める（暗い入り口）**

```
オリジナル Verse 冒頭:
| C (I)       | G (V)       | Am (vi)     | F (IV)      |
  明るいスタート

リハーモナイズ版:
| Am (vi)     | F (IV)      | C (I)       | G (V)       |
  Aeolian       Lydian        Ionian        Mixolydian
  暗い入り口→    神秘→         明るさへ→      開放

弾き比べてみよう:
① | Cadd9  | G wor. | Am7   | Fsus2  |  ← I スタート（明るい）
② | Am7    | Fsus2  | Cadd9 | G wor. |  ← vi スタート（暗い→明るい）
```

> 瀬戸: ②は「暗闇の中から光が差してくる」感じ。悔い改めの歌とか、嘆きから賛美へ向かう曲にめちゃくちゃ合う。同じ4つのコードを並べ替えただけなのに、ストーリーが全然違うでしょ？

### STEP 4: リハーモナイズ版 As the Deer を通し演奏

3つの実験を踏まえて、モードの性格を意識したリハーモナイズ版を弾いてみよう。

**コンセプト:** オリジナルより「内省的→解放」の流れを強調する

```
リハーモナイズ版 As the Deer（カポ2、全コード押さえ方付き）

Verse（静かに、内省的）:
| Cadd9 (I)  | Dm7 (ii)   | Am7 (vi)   | Fsus2 (IV) |
| Cadd9 (I)  | Dm7 (ii)   | Am7 (vi)   | Fsus2 (IV) |
| Dm7 (ii)   | Fsus2 (IV) | Cadd9 (I)  |            |

コード TAB:
Cadd9:       Dm7:         Am7:         Fsus2:
e --0--      e --1--      e --0--      e --1--
B --3--      B --1--      B --1--      B --1--
G --0--      G --2--      G --0--      G --0--
D --2--      D --0--      D --2--      D --3--
A --3--      A -----      A --0--      A --3--
E -----      E -----      E -----      E --1--

弾き方: 軽いストローク or アルペジオ
変更点: G(V) → Dm(ii) に差し替え。Mixolydian の開放感を
        Dorian の切なさに変えて、より内省的なヴァースに。
        最終行を ii→IV→I にして、ふわっと着地。

Chorus（力強く、解放へ）:
| Fsus2 (IV) | Cadd9 (I)  | Dm7 (ii)   | Gsus4→G (V)|
| Fsus2 (IV) | C/G (I)    | Am (vi)    | Gsus4→G (V)|
| Dm7 (ii)   | Gsus4→G (V)| Cadd9 (I)  |            |

追加コード TAB:
Gsus4:       G worship:   C/G:         Am:
e --3--      e --3--      e --0--      e --0--
B --1--      B --3--      B --1--      B --1--
G --0--      G --0--      G --0--      G --2--
D --0--      D --0--      D --2--      D --2--
A --3--      A --2--      A --3--      A --0--
E --3--      E --3--      E --3--      E -----

弾き方: フルストローク
変更点: Chorus 後半で C/G（L22 の力強いヴォイシング）を使用。
        ii→V→I の着地を活かして、解放感のあるクライマックスに。
        Gsus4→G の解決の動き（L23）を組み合わせる。

Bridge（祈り、静寂から光へ）:
| Am7 (vi)   | Dm7 (ii)   | Fsus2 (IV) | Gsus4→G (V)| Cmaj7 (I)  |

追加コード TAB:
Cmaj7:
e --0--
B --0--
G --0--
D --2--
A --3--
E -----

弾き方: アルペジオ、最小の音量から少しずつ上げる
変更点: vi→ii→IV→V→I で「暗→明」のグラデーション。
        Aeolian(暗)→Dorian(希望)→Lydian(神秘)→Mixolydian(開放)→Ionian(帰還)
        モードの5つの色が全部1行に詰まっている。
        最後は Cmaj7 で夢見るように終わる（L22 の色替え応用）。
```

> 瀬戸: 通して弾いてみて。オリジナルと比べると、Verse が静かで内省的になって、Chorus で一気に開ける感じが強まってない？ それが ii（Dorian）を入れた効果。Bridge は5つのモード全部を通過するミニ旅行みたいになってる。

> 黒田: ポイントは「何を何に差し替えたか」と「なぜそうしたか」がモードで説明できること。V(Mixolydian = 開放) を ii(Dorian = 切なさ+希望) に替えたら、当然響きは「開放→内省」に変わる。これがモードを知ってる人のコード選び。

## ② 事後理論

> 黒田（理論メモ）: 今弾いたものの「なぜ」を整理するよ。今日は俺がメインで。

### モード = スケールの「スタート地点」を変えたもの

```
C メジャースケール: C D E F G A B

C から始める → C D E F G A B C → Ionian（I）
D から始める → D E F G A B C D → Dorian（ii）
E から始める → E F G A B C D E → Phrygian（iii）※今回は省略
F から始める → F G A B C D E F → Lydian（IV）
G から始める → G A B C D E F G → Mixolydian（V）
A から始める → A B C D E F G A → Aeolian（vi）
B から始める → B C D E F G A B → Locrian（vii）※今回は省略

→ 全部同じ7つの音。スタート地点が違うだけで「色」が変わる。
```

> 黒田: 7つあるけど、ワーシップで使うのは5つ。Phrygian（iii = F#m）と Locrian（vii = C#m7♭5）はワーシップではほぼ出番がない。不安定すぎて賛美の文脈に合わないから。今日はこの5つに絞って覚えればOK。

### 5つのモードの特徴音まとめ

```
┌──────────────┬────────┬──────────────────────┬──────────────┐
│ モード        │ 度数   │ 特徴音               │ 響き          │
├──────────────┼────────┼──────────────────────┼──────────────┤
│ Ionian       │ I      │ ♮4, ♮7（= 普通）     │ 明るい・安定   │
│ Dorian       │ ii     │ ♮6（普通のminorは♭6） │ 希望あるマイナー│
│ Lydian       │ IV     │ #4                   │ 浮遊・神秘     │
│ Mixolydian   │ V      │ ♭7                   │ 開放・未解決   │
│ Aeolian      │ vi     │ ♭6（Dorianとの違い）  │ 悲しい・内省   │
└──────────────┴────────┴──────────────────────┴──────────────┘
```

> 黒田: この表の「特徴音」が各モードの「指紋」。たった1音の違いが、コード全体の性格を決めてる。Dorian の ♮6 が希望を生み、Lydian の #4 が浮遊感を生み、Mixolydian の ♭7 が未解決を生み、Aeolian の ♭6 が悲しみを深める。

### ワーシップ活用マップ: 感情 → モード → コード機能

```
「感情」から逆引きでコードを選ぶ:

┌──────────────────────────┬────────────────┬─────────────────┐
│ 作りたい感情              │ モード          │ コード（カポ2）  │
├──────────────────────────┼────────────────┼─────────────────┤
│ 安定・安心・感謝          │ Ionian (I)     │ Cadd9, C/G      │
│ 切なさの中の希望          │ Dorian (ii)    │ Dm7             │
│ 神秘的・天が開く・浮遊    │ Lydian (IV)    │ Fsus2, Fmaj7    │
│ 開放感・ロック的          │ Mixolydian (V) │ G worship, Gsus4│
│ 深い祈り・悔い改め・嘆き  │ Aeolian (vi)   │ Am7, Asus2      │
└──────────────────────────┴────────────────┴─────────────────┘

使い方:
① 「今、会衆にどんな感情を届けたい？」を考える
② 上の表で対応するモード → コードを選ぶ
③ L23 のテンション判断で add9/sus4 を入れるか決める
④ L22 の色替えでヴォイシングを選ぶ
```

> 黒田: L21（コードの形）→ L22（コードの色替え）→ L23（テンションの足し引き）→ L24（コードの性格選び）。4つのレイヤーが積み上がってきた。これで「どのコードを」「どの形で」「テンションを入れるか」「なぜそのコードなのか」まで全部説明できるようになった。

## ③ 応用ドリル

### ドリル 1: 「慕い求めます」の各セクションにモード感情ラベルをつける

L21 で弾いた「慕い求めます」の進行を、モードの色で読み解こう。
**※ カポ2 で弾きながらラベリングする。**

```
慕い求めます（カポ2）:

Verse:
| Cadd9 (I)   | G wor. (V)  | Am7 (vi)    | Fsus2 (IV)  |
  安定          開放          祈り          神秘
  ↓ Ionian     ↓ Mixolydian  ↓ Aeolian     ↓ Lydian

| Cadd9 (I)   | G wor. (V)  | Am7 (vi)    | Fsus2 (IV)  |
  安定          開放          祈り          神秘

| Dm7 (ii)    | G wor. (V)  | Cadd9 (I)   |             |
  切なさ+希望   開放          帰還
  ↓ Dorian     ↓ Mixolydian  ↓ Ionian

→ 最後の Dm7→G→C は ii-V-I。
  Dorian(希望あるマイナー)→Mixolydian(開放)→Ionian(帰還)
  = 「切なさから開放を経て安心へ」

Chorus:
| Fsus2 (IV)  | Cadd9 (I)   | Dm7 (ii)    | Gsus4→G (V) |
  神秘          安定          切なさ+希望    開放→解決

| Fsus2 (IV)  | C/G (I)     | Am7 (vi)    | Gsus4→G (V) |
  神秘          安定(力強い)   祈り          開放→解決

| Dm7 (ii)    | G wor. (V)  | Cadd9 (I)   |             |
  切なさ+希望   開放          帰還

→ Chorus は IV(Lydian=神秘) から始まる。
  ヴァースが I(安定) スタートだったのと対照的。
```

> 瀬戸: 弾きながらラベルを口で言ってみて。「安定、開放、祈り、神秘……」って。そうすると各コードを弾く瞬間に、その「色」を意識できるようになる。これをやると演奏が変わるよ。同じコードでも「今は祈りの色だ」って思って弾くと、タッチが自然に変わる。

### ドリル 2: As the Deer を「明るく」「切なく」「神秘的に」弾き分ける

**同じ As the Deer の Verse を、コードの差し替えだけで3つの感情バージョンに。**
**※ カポ2 で練習。**

#### 明るいバージョン（Ionian + Mixolydian 中心）

```
| Cadd9 (I)  | G wor. (V) | Cadd9 (I)  | G wor. (V) |
  Ionian       Mixolydian   Ionian       Mixolydian

→ I と V だけで構成。ずっと明るい・開放的。
  暗さが全くない「主を喜ぶ」バージョン。
```

#### 切ないバージョン（Dorian + Aeolian 中心）

```
| Am7 (vi)   | Dm7 (ii)   | Am7 (vi)   | Dm7 (ii)   |
  Aeolian      Dorian       Aeolian      Dorian

→ vi と ii だけ。マイナー系のみ。
  でも Dorian が入っているから「絶望」ではなく「希望ある切なさ」。
  悔い改めの曲、嘆きの詩篇に合う。
```

#### 神秘的バージョン（Lydian + Ionian 中心）

```
| Fsus2 (IV) | Cadd9 (I)  | Fsus2 (IV) | Cadd9 (I)  |
  Lydian       Ionian       Lydian       Ionian

→ IV と I を交互。Lydian の #4 が浮遊感を作る。
  聖霊の臨在、異言の祈り、静まりの時間に合う。
```

> 瀬戸: 3つ連続で弾いてみて。明るい → 切ない → 神秘。全部同じキー（D メジャー）、全部カポ2、使う音は同じ7音。でもコードの選び方だけでこれだけ変わる。モードってそういうこと。

> 黒田: ここまで来たら、もう「なんとなくコードを選ぶ」じゃなくて「この感情を出したいからこのコード」って選べるようになってるはず。それがモードの力。

## ④ 自分で組み合わせる

### チャレンジ: L22 + L23 + L24 を統合した As the Deer 完全版

**L22（ヴォイシング色替え）+ L23（テンション判断）+ L24（モード性格選び）を全部組み合わせて、自分だけの As the Deer を作ろう。**

**3つのレイヤーの組み合わせ方:**

```

  → I(安定)? ii(希望ある切なさ)? IV(神秘)? V(開放)? vi(祈り)?

```

```
優先順位（迷ったらこの順番で判断）:
```
