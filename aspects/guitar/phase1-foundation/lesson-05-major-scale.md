## 1. メジャースケールって何？（10分）

### ドレミファソラシド = C メジャースケール

実は、みんなが子供の頃から知っている「ドレミファソラシド」。
これがそのまま **C メジャースケール** なんだよね。

```
ド  レ  ミ  ファ ソ  ラ  シ  ド
C   D   E   F    G   A   B   C
```

**スケール（scale）** というのは「音の階段」のこと。
ラテン語の "scala"（はしご）が語源で、音が一段ずつ上がっていく並びのことを言う。

**メジャー（major）** は「明るい」という意味。
だから **メジャースケール = 明るい響きの音の階段**。

ちなみに、反対の **マイナースケール** は「暗い響きの音の階段」。
「ラシドレミファソラ」--- つまり A から始めてピアノの白鍵だけ弾くと、それが **A マイナースケール** になる。
今日はメジャーだけに集中するけど、「明るい/暗い = メジャー/マイナー」という対応は覚えておいて。

> **黒田メモ:** 「スケール」って聞くと難しそうに感じるかもしれないけど、要は「ドレミファソラシド」のこと。すでに耳で知ってるものを、ギターの指板上で弾けるようにするだけだよ。実はメジャースケールを知ってるだけで、ポップスやロックの曲の 90% 以上のメロディが理解できる。それくらい音楽の根幹にある考え方なんだ。

### 7つの音 + 1（オクターブ）

メジャースケールは **7つの異なる音** でできている。
8番目の音は1番目と同じ音の高い版（オクターブ）。

```
1番目  2番目  3番目  4番目  5番目  6番目  7番目  8番目（= 1番目の高い版）
 C      D      E      F      G      A      B      C
 ド     レ     ミ     ファ    ソ     ラ     シ     ド
```

12個ある音のうち、**7つを選んで並べたもの** がメジャースケール。
「12個の音すべてを使う」のではなく、「特定のルール（並び）で7つを選ぶ」のがポイント。

なぜ7つなのか？ 実は人間の脳が一度に処理しやすい数が大体7つ前後と言われていて（電話番号も7桁だよね）、音楽においても7つの音がちょうど「豊かだけど混乱しない」バランスなんだ。5つだと表現の幅が狭く、12全部だとどこに向かってるのか分からなくなる。7つが絶妙な「ちょうどいい」なんだよね。

では、そのルールとは何か？ --- 次のセクションで見ていこう。

## 2. メジャースケールの構造: 全全半全全全半（15分）

### 「全音」と「半音」の並びがすべてを決める

Lesson 1〜2 で、**全音（ぜんおん）= フレット2個分** と **半音（はんおん）= フレット1個分** を学んだよね。

メジャースケールの正体は、この全音と半音の並び順。

```
C → D → E → F → G → A → B → C
  全  全  半  全  全  全  半
  W   W   H   W   W   W   H
```

- **W = Whole step（全音）**: フレット2個分
- **H = Half step（半音）**: フレット1個分

この **「全全半全全全半」** という並びが、メジャースケールの設計図。

ここがすごいのは、**この並びさえ知っていれば、どの音からでもメジャースケールが作れる** ということ。
例えば G から始めて「全全半全全全半」を適用すると:

```
G → A → B → C → D → E → F# → G
  全  全  半  全  全  全   半
```

G メジャースケールの完成。F が F# になったのは「全全半全全全半」のルールに従った結果。
D からなら:

```
D → E → F# → G → A → B → C# → D
  全  全   半  全  全  全   半
```

D メジャースケール。1つの公式から12のメジャースケールすべてが導き出せる。

### なぜこの並びなのか？

ここがおもしろいところ。

12個の音を全部並べると、こうなる:

```
C  C#  D  D#  E  F  F#  G  G#  A  A#  B  C
 半  半  半  半 半  半  半  半  半  半  半  半
```

全部が半音ずつ均等に並んでいる。これを **クロマチックスケール（半音階）** という。
でも、12個の音を全部使うと「どこがゴールか分からない道」みたいになる。

そこで、**人間の耳が「明るくて安定している」と感じる7つの音を選んだ** のがメジャースケール。

例え話で言うと:

```
クロマチック（12音全部）= 地図上のすべての道
メジャースケール（7音）= その中から選んだ「メインストリート」
```

街を歩くとき、すべての路地裏を通る必要はないよね。
メインストリートを知っていれば、街全体の構造が分かる。
メジャースケールは、音楽という街の **メインストリート** なんだ。

### 確認: 半音の位置に注目

もう一度、全全半全全全半を見てみよう:

```
C → D → E → F → G → A → B → C
  全  全  半  全  全  全  半
        ↑                   ↑
      3→4番目             7→8番目
      （ミ→ファ）         （シ→ド）
```

半音が来るのは **3番目→4番目** と **7番目→8番目（オクターブ）** の2箇所だけ。
残りは全部「全音」。

これは覚えておくと便利。ピアノの鍵盤で言うと、E→F と B→C の間に黒鍵がないのと同じ理由。

この半音の位置が、メジャースケールに独特の「明るいけどどこか切ない」響きを与えている。特に7番目→8番目（B→C）の半音は **導音（リーディングトーン）** と呼ばれて、「もうすぐ家に帰る」感覚を作る。John Mayer の曲を聴いていると、メロディが B の音に来たとき「次は C に行きたい！」という引力を感じるはず。それがこの半音の力なんだ。

> **黒田メモ:** 「全全半全全全半」--- これは呪文みたいに口で唱えて覚えちゃうのが一番早い。「ゼンゼンハン、ゼンゼンゼンハン」ってリズムよく言うと覚えやすいよ。もう一つ覚え方がある。半音の位置は「3-4」と「7-8」。3+4=7 だし、7+1=8。数字の関係で覚えるのも手だね。

## 3. C メジャースケールを弾いてみよう --- オープンポジション（15分）

### 6弦〜1弦を使った C メジャースケール（ローポジション）

まずは一番基本的な形。開放弦（0フレット）を含む、最もローポジション（低い位置）の弾き方。

```
【TAB譜】C メジャースケール（オープンポジション）上昇

e |----------------------------------0--1--3--|
B |-------------------------0--1--3-----------|
G |------------------0--2----------------------|
D |-----------0--2--3--------------------------|
A |----0--2--3---------------------------------|
E |--3-----------------------------------------|
     ↑
   ここがスタート（低いほうの C）

音名:
E |----------------------------------E--F--G--|
B |-------------------------C--D--E-----------|
G |------------------G--A----------------------|
D |-----------D--E--F--------------------------|
A |----A--B--C---------------------------------|
E |--G-----------------------------------------|

    G  A  B  C  D  E  F  G  A  B  C  D  E  F  G
```

### 弾くときの指使い（フィンガリング）

左手の指の番号:
- **人差し指** = 1
- **中指** = 2
- **薬指** = 3
- **小指** = 4

```
【フィンガリング】

e |----------------------------------1--2--4--|
B |-------------------------1--2--4-----------|
G |------------------1--3----------------------|
D |-----------1--3--4--------------------------|
A |----1--3--4---------------------------------|
E |--3-----------------------------------------|

※ 開放弦（0フレット）は左手を使わない
```

**弾き方のコツ:**

1. 6弦3フレットの G からスタート（ここは薬指で押さえる）
2. 1つの弦を弾き終えてから次の弦に移る
3. 最初はゆっくり、1音ずつはっきり鳴らす
4. 急がない。メトロノームなしでOK
5. 左手の親指はネックの裏の中央あたりに置く。親指の位置が安定すると、他の指が自然に動く
6. 弦を押さえるとき、フレットの真上ではなく **フレットのすぐ左側**（ナット寄り）を押さえると、きれいな音が出やすい
7. 右手のピッキングは「ダウン（下方向）」だけでOK。1音1音しっかり弾く

### 5弦から始める C メジャースケール

6弦だと G から始まってしまうので、**C の音（5弦3フレット）** から始めるバージョンも弾いてみよう。

```
【TAB譜】C メジャースケール（5弦スタート）上昇 → 下降

上昇:
e |-------------------------------|
B |-------------------------------|
G |--------------------0--2-------|
D |-----------0--2--3-------------|
A |--3--0--2--3-------------------|
E |-------------------------------|

   ×  ×
   ↑  ↑ 5弦の開放A と 2フレットB はスケール外の音も含むので注意

修正版（C から始める）:
e |-------------------------------|
B |-------------------------------|
G |--------------------0--2-------|
D |-----------0--2--3-------------|
A |--3-----------------------------|
E |-------------------------------|

音名:
A |--C-----------------------------|
D |-----------D--E--F-------------|
G |--------------------G--A-------|

     C  D  E  F  G  A  ... と上がっていく
```

実はオープンポジションだと、Cから始めてきれいに1オクターブ弾くのは少し工夫がいる。
だから次のセクションで、もっとすっきりした「ポジション」を紹介するよ。

## 4. 指板上のポジション --- 最も基本的な形（10分）

### ポジションとは？

**ポジション** というのは、指板上の「手の位置」のこと。
ギターはピアノと違って、同じ音を複数の場所で弾ける。
だから、「この辺りに手を置いて弾く形」がいくつもある。

今日は一番基本的なポジションを1つだけ覚えよう。

### C メジャースケール: ポジション1（5弦ルート）

5弦3フレット（C）を起点にしたポジション。
手をほとんど動かさずに、1オクターブ分を弾ける形。

```
【TAB譜】C メジャースケール ポジション1（上昇）

e |-------------------------------|
B |-------------------------------|
G |-------------------------------|
D |--------2--3--5----------------|
A |--3--5--------------------------|
E |-------------------------------|

音名:
A |--C--D--------------------------|
D |--------E--F--G----------------|

     C → D → E → F → G

     全   全   半   全
```

ここまでで5音（C D E F G）。さらに続けると:

```
【TAB譜】C メジャースケール ポジション1（1オクターブ全体）

e |-------------------------------|
B |-------------------------------|
G |--2--4--5----------------------|
D |--------2--3--5----------------|
A |--3--5--------------------------|
E |-------------------------------|

音名:
A |--C--D--------------------------|
D |--------E--F--G----------------|
G |--A--B--C（高い）----------------|

     C → D → E → F → G → A → B → C
       全  全  半  全  全  全  半
```

### この形を覚えよう

指板上で見ると、こんな形になる:

```
【指板図】（フレット 2〜5）

フレット:  2     3     4     5
     e |------|------|------|------|
     B |------|------|------|------|
     G |--A---|---B--|--C---|------|
     D |------|--E---|--F---|--G---|
     A |------|--C---|---D--|------|
     E |------|------|------|------|

● = スケールの音

フレット:  2     3     4     5
     e |------|------|------|------|
     B |------|------|------|------|
     G |--●---|---●--|--●---|------|
     D |------|--●---|--●---|--●---|
     A |------|--●---|---●--|------|
     E |------|------|------|------|
```

**ポイント:**

- 5弦と4弦は「2フレット幅」（3→5 / 2→3→5）
- 3弦は「3フレット幅」（2→4→5）
- この「形」はどのフレットに移動しても使える（後のレッスンでやる）

**なぜ「形」で覚えるのか？** ギターの最大の特徴は、同じ「形」をずらすだけで別のキー（調）のスケールが弾けること。ピアノだと C メジャーと D メジャーでは指使いが全然違うけど、ギターでは今覚えたポジション1の「形」を2フレット右にずらすだけで D メジャースケールになる。この「形の移動（トランスポーズ）」がギターの大きな武器なんだ。

```
【参考】同じ形を2フレット右にずらすと → D メジャースケール

フレット:  4     5     6     7
     G |--B---|---C#-|--D---|------|
     D |------|--F#--|--G---|--A---|
     A |------|--D---|---E--|------|

D → E → F# → G → A → B → C# → D
  全  全   半  全  全  全   半
```

### フィンガリング

```
フレット:  2     3     4     5
     G |--1---|---3--|--4---|------|
     D |------|--1---|--2---|--4---|
     A |------|--1---|---3--|------|
```

- 2フレット = 人差し指（1）
- 3フレット = 人差し指（1）または 中指（3）
- 4フレット = 中指（2）または 薬指（3）
- 5フレット = 小指（4）

> **黒田メモ:** 最初は形がぎこちなくても大丈夫。大事なのは「こういう形がある」ということを知ること。体が覚えるまでに少し時間がかかるけど、毎日5分弾いてれば自然と手に馴染んでくるよ。ちなみにプロのギタリストでも、ウォームアップでまずスケールを弾くことが多い。John Mayer もリハーサル前にスケールを弾いてる映像がある。基本は一生使う「型」なんだ。

## 5. なぜメジャースケールが重要なのか（5分）

### コードもメロディも、ここから生まれる

メジャースケールが「音楽の基本地図」だと言ったのには理由がある。

#### コードはスケールから生まれる

C メジャースケールの音を1つ飛ばしで積み重ねると、コードができる:

```
C メジャースケール:  C  D  E  F  G  A  B

C コード = C + E + G（1番目 + 3番目 + 5番目）
Dm コード = D + F + A（2番目 + 4番目 + 6番目）
Em コード = E + G + B（3番目 + 5番目 + 7番目）
F コード = F + A + C（4番目 + 6番目 + 1番目）
G コード = G + B + D（5番目 + 7番目 + 2番目）
Am コード = A + C + E（6番目 + 1番目 + 3番目）
```

知ってるコードが全部、C メジャースケールの音だけで作られている。

さらに面白いのは、メジャースケールの音を積み重ねると、**メジャーコード（明るい）とマイナーコード（暗い）が自動的に決まる** ということ:

```
C メジャースケールから生まれるコード:

I    = C   (メジャー) → 明るい
ii   = Dm  (マイナー) → 暗い
iii  = Em  (マイナー) → 暗い
IV   = F   (メジャー) → 明るい
V    = G   (メジャー) → 明るい
vi   = Am  (マイナー) → 暗い
vii  = Bdim(ディミニッシュ) → 不安定
```

John Mayer の曲で使われるコードのほとんどは、この7つのコードとそのバリエーション。
Gravity は C → Am7 → Dm7 → G7、つまり I → vi7 → ii7 → V7 という進行。全部ここから来ている。

#### メロディもスケールから生まれる

歌のメロディも、ほとんどの場合スケールの音を行ったり来たりしているだけ。

```
「きらきら星」のメロディ:
C C G G A A G - F F E E D D C -

→ すべて C メジャースケールの音
```

John Mayer の他の曲も見てみよう:

```
「Waiting on the World to Change」サビ:
G  G  A  G  E  D  C  D  E ...

→ すべて C メジャースケールの音

「No Such Thing」イントロリフ:
E  D  C  D  E  G  A  G ...

→ すべて C メジャースケールの音
```

ジャンルを問わず、ほとんどのポップス・ロックのメロディはスケールの音を行き来しているだけなんだ。

#### つまり

```
メジャースケール ──→ コードが生まれる
      │
      └──→ メロディが生まれる

1つのスケールから、曲のすべてが作られている
```

これが、メジャースケールを「音楽の基本地図」と呼ぶ理由。
地図を持っていれば、知らない街でも迷わない。
スケールを知っていれば、知らない曲でも「何が起きているか」が分かるようになる。

もう少し具体的に言うと、メジャースケールを理解すると以下のことができるようになる:

- **耳コピ** --- 聴こえたメロディが「スケールの何番目の音か」分かるので、指板上で見つけやすい
- **アドリブ** --- 「この曲は C メジャーだ」と分かれば、C メジャースケールの音を使って自由にフレーズが作れる
- **作曲** --- コードとメロディの両方をスケールから導き出せる
- **コミュニケーション** --- 他のミュージシャンと「Cメジャーで行こう」と言えば、全員が同じ地図を共有できる

## 6. Gravity で確認しよう --- すべてメジャースケールの音（5分）

### Gravity のコード

John Mayer の Gravity で使われている主なコードを見てみよう:

```
Gravity のコード進行（バース）:

C → Am7 → Dm7 → G（→ G7）
```

それぞれのコードに含まれる音を書き出してみる:

```
C   = C, E, G
Am7 = A, C, E, G
Dm7 = D, F, A, C
G   = G, B, D
G7  = G, B, D, F
```

使われている音をすべて集めると:

```
C, D, E, F, G, A, B
```

--- これは **C メジャースケールそのもの**。

スケール外の音は1つも使われていない。

### Gravity のメロディ（サビ）

サビで John Mayer が歌うメロディも確認してみよう:

```
"Gravity is working against me"

歌のメロディの音:
G  G  G  G  A  G  E  D  C ...

→ すべて C メジャースケールの音
```

コードもメロディも、C メジャースケールの7音だけで成り立っている。

### なぜ Gravity は「シンプルなのに深い」のか

ここでもう一歩踏み込んでみよう。Gravity のメロディで使われている音の「役割」を見てみると:

```
G = スケールの5番目（ソ）→ 安定感のある音。「ここにいても大丈夫」
A = スケールの6番目（ラ）→ 少し緊張する音。「ちょっと外に出た」
E = スケールの3番目（ミ）→ 甘い音。コードの「核」にある音
```

John Mayer は5番目の音（G）を「ベースキャンプ」にして、6番目（A）にちょっと登って戻り、3番目（E）に降りて感情を表現している。まるで「安全圏から少し冒険して、感情的な場所に着地する」ようなメロディの作り方。

これは Gravity だけじゃなく、John Mayer の多くの曲に共通するパターン。たとえば「Slow Dancing in a Burning Room」のサビも、スケールの中心的な音（1度、3度、5度）を軸にメロディが展開される。

> **黒田メモ:** Gravity はシンプルな曲に聴こえるけど、だからこそメジャースケールの力がよく分かる。たった7つの音の組み合わせで、あれだけ豊かな音楽が生まれてるんだ。John Mayer はスケールの音の「選び方」と「タイミング」がとにかくうまい。スケールを知ることで、彼の音選びの理由が見えるようになるよ。ちなみに、Gravity のスタジオ版とライブ版（Where the Light Is）を聴き比べると、同じメロディなのにライブ版の方がずっと「深い」のが分かる。音列は変わってないのに、弾き方と歌い方だけで別の曲みたいに聴こえる。これが表現力というもの --- Lesson 7 で橋本が詳しくやってくれるよ。

## 7. 練習: 声に出しながら弾こう（残りの時間）

### なぜ声に出すのか？

ギターを弾くだけだと「指の運動」になりがち。
声に出すことで、**耳と手と脳** が同時につながる。

プロのミュージシャンの多くは、弾いてる音を口ずさみながら演奏する。
John Mayer も、ライブで弾きながらよく声を出してるよね。
これは「指」ではなく「耳」で音楽をやっている証拠。

声を出すことには科学的な裏付けもある。音を「指で弾く」「耳で聴く」「声で出す」の3つを同時にやると、脳の複数の領域が同時に活性化する。結果として、指だけで弾くよりも **2〜3倍早く** 音の配置が身につくと言われている。最初は恥ずかしいかもしれないけど、効果は抜群だよ。

### 練習 7-1: C メジャースケール上昇（声 + ギター）

ポジション1を使って、**音名を声に出しながら** 弾く。

```
【TAB譜】

e |-------------------------------|
B |-------------------------------|
G |--2--4--5----------------------|
D |--------2--3--5----------------|
A |--3--5--------------------------|
E |-------------------------------|

声: 「シー、ディー、イー、エフ、ジー、エー、ビー、シー」
     C     D     E    F     G     A     B     C
```

日本語の音名で言ってもOK:

```
声: 「ド、レ、ミ、ファ、ソ、ラ、シ、ド」
```

**どちらでも好きな方でOK。** 最初はドレミの方が馴染みがあって楽かもしれない。
慣れてきたら英語の音名（C, D, E...）にチャレンジしてみて。

### 練習 7-2: 下降も弾こう

上がるだけじゃなく、降りる練習もセットでやろう。

```
【TAB譜】下降

e |-------------------------------|
B |-------------------------------|
G |--5--4--2----------------------|
D |-----------5--3--2-------------|
A |--------------------5--3-------|
E |-------------------------------|

声: 「ド、シ、ラ、ソ、ファ、ミ、レ、ド」
     C    B    A    G    F     E    D    C
       半  全   全   全   半   全   全
```

下降のときも「全全半全全全半」が逆順になっているのを確認してみて:

```
上昇: 全 全 半 全 全 全 半
下降: 半 全 全 全 半 全 全  ← 逆になってる
```

### 練習 7-3: 上昇 → 下降をつなげる

```
【TAB譜】上昇 → 下降（1往復）

上昇:
e |-------------------------------|
B |-------------------------------|
G |--2--4--5----------------------|
D |--------2--3--5----------------|
A |--3--5--------------------------|
E |-------------------------------|

（頂上の C に着いたら折り返し）

下降:
e |-------------------------------|
B |-------------------------------|
G |--5--4--2----------------------|
D |-----------5--3--2-------------|
A |--------------------5--3-------|
E |-------------------------------|
```

**テンポの目安:**

- 最初は 1音につき2秒くらい（とてもゆっくり）
- 声と指が合うようになったら、少しずつ速くする
- メトロノームを使うなら BPM 60（1秒に1音）から

**よくある問題と対処法:**

| 問題 | 原因 | 対処 |
|:---|:---|:---|
| 音がビリビリする | 指がフレットから遠い、または押さえが弱い | フレットのすぐ左側を、指先でしっかり押さえる |
| 弦移動で音が途切れる | 次の弦に移るとき指を早く離しすぎ | 次の音を鳴らす直前まで前の音を押さえ続ける |
| 小指が力んで動かない | 小指だけに意識が集中してる | 腕全体をリラックス。小指は「添える」感覚で |
| 下降がスムーズにいかない | 下降は上昇と違う筋肉の動きが必要 | 上昇の半分の速度で練習。焦らないこと |

> **弾き方のコツ:** 速さよりも「すべての音がはっきり鳴っているか」を優先すること。音がビリビリしたり、途切れたりしていたら、テンポを落とそう。きれいに鳴る速さが、今の自分にとっての正しいテンポだよ。もう一つ大事なのは、**1回弾いたら必ず耳で確認すること**。「鳴ったかな？」と思ったら、もう一度同じ音を弾いて聴き比べる。この「弾く→聴く→確認する」のサイクルが、練習の質を大きく変えるよ。

## まとめ

今日学んだことを整理しよう。

1. **メジャースケール = ドレミファソラシド**。7つの音でできた「明るい音の階段」。
2. **構造は「全全半全全全半」**。この並びがメジャースケールの設計図。どの音から始めてもこのルールでメジャースケールが作れる。
3. **半音が来るのは「3→4番目（ミ→ファ）」と「7→8番目（シ→ド）」** の2箇所だけ。特に7→8番目の半音は「家に帰りたくなる」引力を生む。
4. **コードもメロディも、メジャースケールから生まれる。** だから「音楽の基本地図」。コードの明暗（メジャー/マイナー）もスケールの構造から自動的に決まる。
5. **Gravity は C メジャースケールの7音だけ** でコードもメロディもできている。たった3音（G, A, E）でサビのメロディが成立している。
6. **声に出しながら弾く** ことで、耳と手と脳がつながる。定着の速さが格段に変わる。
7. **ポジション1の「形」はどこにでも移動できる**。今日覚えた形をずらすだけで、12のキーすべてに対応できる。
