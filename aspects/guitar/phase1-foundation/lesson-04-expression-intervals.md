## はじめに

ここまで黒田さんと瀬戸さんから、インターバルの「仕組み」と「弾き方」を学んできたよね。
今日は少し違う角度から入るよ。

**「何を感じるか」**

これが今日のテーマ。理論を知っていることと、それを「伝える」ことは全然違う。
たとえば、「短3度は半音3個分の距離」って知識として分かってる。でも、それを弾いたときに胸の奥がキュッとなる感覚 --- それを「感じられるか」が、音楽をやる上で一番大事なことだと思ってる。

テクニックは後からいくらでもつく。でも「音を聴く耳」と「感じる心」は、今この瞬間から育てていけるもの。

John Mayer がインタビューでこんなことを言ってた --- 「ギターは僕の声なんだ。歌では言えないことを、ギターが代わりに言ってくれる」。彼のプレイを聴くとわかるけど、テクニックがすごいだけじゃなくて、1音1音に「何かを伝えたい」という意志がある。それが聴く人の心を動かしてる。

今日は一緒に、インターバルの中にある「感情」を探しに行こう。

## 1. 聴く練習 --- 短3度と長3度の「感情」を言葉にする（10分）

### まずは、弾いて聴く

ギターを手に取って。5弦3フレット（C）を出発点にする。

```
【TAB譜】短3度: C → Eb

e |------|
B |------|
G |------|
D |------|
A |--3---6---|
E |------|
    C   Eb
```

C を弾いて、少し間を置いて、Eb を弾く。
急がないで。C の音が空気に溶けていくのを待ってから、Eb を弾いてみて。

**何を感じた？**

言葉にしてみて。「悲しい」でも「切ない」でも「雨の日みたい」でも「胸が重い」でも、なんでもいい。
正解はない。自分の中に浮かんだ言葉が、それが答え。

### 次に、長3度

```
【TAB譜】長3度: C → E

e |------|
B |------|
G |------|
D |------|
A |--3---7---|
E |------|
    C    E
```

同じように。C を弾いて、間を置いて、E を弾く。

**何を感じた？ さっきと何が違う？**

多くの人が「明るい」「開ける感じ」「空が見えた」みたいに言う。でも、もしかしたら「懐かしい」とか「朝っぽい」とか、自分だけの言葉が出てくるかもしれない。それが一番いい。

面白いことに、同じ長3度でも**弾く場所（ポジション）によって少しニュアンスが変わる**。5弦の低い位置で弾くと「温かい」感じ、1弦の高い位置で弾くと「キラキラした」感じがする。同じインターバル、同じ距離なのに。これは音の高さ（音域）が持つ固有の性格で、ギターの面白いところ。余裕があったら、1弦の5フレット（A）→ 9フレット（C#）で長3度を弾いてみて。5弦で弾いたときとの違いを感じてみよう。

### 交互に弾いてみる

```
C → Eb（短3度）→ C → E（長3度）→ C → Eb → C → E ...
```

これを5回くらい繰り返す。
だんだん、弾く前から「次はこういう気持ちになるな」って分かるようになる。
それが**インターバルを「体で覚えた」**ということ。

ここでもう一歩踏み込んでみよう。**同じ短3度でも、弾く順番を逆にすると印象が変わる。**

```
C → Eb（上行の短3度）：上に向かうけど暗い。「立ち上がろうとしてるけど重い」感じ
Eb → C（下行の短3度）：下に落ちて暗い。「あきらめて沈む」感じ
```

同じ2つの音なのに、順番だけで印象が変わる。メロディを作るとき、「上に向かう」か「下に向かう」かで感情の方向性が変わるんだ。John Mayer の "Slow Dancing in a Burning Room" のサビメロディは、短3度の下行が繰り返し出てくる。あの「沈んでいく」感覚は、まさにこの下行の短3度が生み出しているもの。

> **海のメモ:** 音楽理論の教科書には「短3度 = 暗い、長3度 = 明るい」って書いてある。でも、実際に弾いて感じることはそれよりずっと豊かなんだよね。同じ短3度でも、弾く人によって「悲しい」だったり「美しい」だったり「雨上がり」だったりする。その違いが個性になるし、表現になる。もっと言うと、同じ人でも昨日と今日で感じ方が変わったりする。それは自分の内面が変わったということ。ギターを弾くことは、自分の心を映す鏡を見ることでもあるんだよね。

## 2. ダイナミクス --- 同じ音でも弾き方で世界が変わる（10分）

### ダイナミクスって何？

ダイナミクスというのは、**音の強弱**（きょうじゃく）のこと。
ピアノなら鍵盤を強く押すか弱く押すか。ギターなら弦を強く弾くか、そっと弾くか。

同じ C の音でも、弾き方を変えるだけで全然違うものになる。
試してみよう。

### 練習 2-1: 強弱の4段階

5弦3フレット（C）を1つだけ使って、4段階で弾いてみて。

| 段階 | 弾き方 | イメージ |
|:---:|:---|:---|
| **pp（ピアニッシモ）** | 弦にそっと触れるように。指先で撫でるくらい | 夜中の独り言。誰にも聞かれたくない秘密 |
| **p（ピアノ）** | 軽く弾く。音はちゃんと出るけど控えめ | 朝の挨拶。穏やかに「おはよう」 |
| **f（フォルテ）** | しっかり弾く。芯のある音 | 大事なことを伝えるとき。「聞いてほしい」 |
| **ff（フォルティッシモ）** | 力強く弾く。弦が大きく振動する | 叫び。感情が溢れ出す瞬間 |

> **用語メモ:** pp, p, f, ff はクラシック音楽で使われる音量の記号。イタリア語が語源で、p = piano（弱い）、f = forte（強い）という意味。ギターの世界では記号よりも感覚が大事なので、「そっと」「しっかり」くらいの理解で十分。

### 練習 2-2: 同じ音を4回弾く --- 物語をつくる

C の音を4回弾くだけ。でも強弱を変える。

```
パターン A: pp → p → f → ff
（だんだん大きく → 決意が固まっていく感じ）

パターン B: ff → f → p → pp
（だんだん小さく → 余韻が消えていく。別れの場面）

パターン C: pp → ff → pp → pp
（急に叫んで、すぐ静かに → 驚き、または心の中の爆発）
```

たった1つの音を4回弾いただけなのに、それぞれ全然違う「物語」に聞こえない？

**これがダイナミクスの力。**

テクニックがなくても、コードをたくさん知らなくても、音の強弱だけで感情は伝わる。

もう1つ試してほしいパターンがある。

```
パターン D: p → p → p → ff
（ずっと静かに、最後だけ爆発 → 溜め込んでいた感情が弾ける瞬間）

パターン E: ff → pp → ff → pp
（激しさと静けさの交互 → 葛藤、揺れる心）
```

パターン D は、John Mayer が Gravity のライブで実際によく使う構成。サビの前は抑えめに弾いて、サビで一気にダイナミクスを上げる。「溜めて、解放する」。この繰り返しが聴く人の心を掴む。

ちなみにダイナミクスには「音の大きさ」だけじゃなく、**音色（おんしょく）の変化**も含まれる。弱く弾くとまろやかで柔らかい音、強く弾くとギラッとした鋭い音になる。同じギター、同じ弦、同じフレットなのに。これはピック（弦を弾く道具）の当たり方や、指の肉の当たる面積が変わるから。指弾き（ピックを使わない奏法）だと、この変化がもっと顕著になるよ。

> **海のメモ:** John Mayer のすごいところの一つは、このダイナミクスの幅がとてつもなく広いこと。ささやくような音から、ステージを震わせるような音まで、同じギターから全く違う声が出てくる。それは指の力じゃなくて、「ここはこう弾きたい」という意志の問題。意志があれば、指はそれについてくる。星野源もそう。彼のアコギ弾き語りを聴くと、サビに向かってダイナミクスが自然に上がっていくのがわかる。コード進行は複雑じゃないのに惹きつけられるのは、このダイナミクスのコントロールが絶妙だから。

## 3. Gravity サビの表現分析 --- 音で「重力」を描く（15分）

### まず聴いてみて

Gravity のサビ部分を、歌詞を読みながら聴いてみよう。
YouTube でも Spotify でもいいから、目を閉じて聴いてみて。

歌詞の核心:
```
Gravity is working against me
And gravity wants to bring me down
（重力が僕に逆らっている）
（重力が僕を引きずり下ろそうとしている）
```

**何を感じた？ 音はどう動いてた？**

### 音の「重力」--- メロディの下降

Lesson 2 で黒田さんが教えてくれた通り、Gravity のベースラインは C → B → A と下がっていく。
でも今日注目したいのは、それが**どう弾かれているか**ということ。

サビに入る直前、John Mayer はグッと音量を上げる。
そして "Gravity" と歌い始めるところで、ベースラインが下がり始める。

```
【サビの流れ】

[サビ前: 静かめ、ためる]
         ↓
[「Gravity」: 一気に音量アップ、エモーショナルなピーク]
         ↓
[ベースが C → B → A と下降: 引力に引かれるように落ちていく]
         ↓
[フレーズの終わり: 音量がすっと落ちる → また次の波へ]
```

ここで起きていることを分解すると:

1. **ダイナミクスの波**: 静 → 動 → 静 のサイクル。これが繰り返されることで「波」が生まれる
2. **インターバルの方向**: 下降（ベースが段階的に下がる）→ 重力で引っ張られる感覚
3. **音色の変化**: クリーンから少しドライブがかかる → 感情の高まり
4. **テンポの揺れ**: よく聴くと、サビに入る直前にほんの少しテンポが溜まる（遅くなる）。これは意図的なもので、「次に何かが来るぞ」という期待感を作る

この4つの要素が組み合わさって、Gravity のサビのあの圧倒的な「うわっ」という感覚が生まれてる。そして実は、どれ一つとして「高度なテクニック」じゃないんだ。強弱、音の方向、音色、テンポ --- 全部「意識の問題」。だからこそ、今の段階でも真似できる。

### 練習 3-1: Gravity のベースラインを「表現」で弾く

Lesson 2 で弾いた C → B → A のベースラインを、今度は「表現」を意識して弾いてみよう。

```
【TAB譜】ベースラインのみ

A弦: --3----2----0----

      C     B     A
```

ただし、3つの弾き方で弾いてみて。

**弾き方 1: 平坦に（比較用）**
全部同じ強さで、同じタイミングで。「ド、シ、ラ」と機械的に。

**弾き方 2: 重力を感じて**
C を一番強く弾いて、B は少し弱く、A はさらに弱く。
「高いところから落ちていく」感じ。音量が重力に引かれるように小さくなっていく。

**弾き方 3: 最後に抗って**
C を強く、B を弱く、でも A をもう一度しっかり弾く。
「落ちそうになるけど、最後に踏ん張る」感じ。John Mayer がこの曲で歌っているのは、まさにそういう気持ち。

**3つのうち、どれが一番「Gravity」っぽかった？**

正解はないけど、自分の中で「これだ」と思えるものがあったら、それが今の自分の解釈。
それを大事にしてほしい。

ちなみに余裕があったら、**4つ目の弾き方**も試してみて。

**弾き方 4: C だけ弾いて、B と A は心の中で鳴らす**

C を弾いたら、B を弾かずに「頭の中で B の音を想像しながら」間を置く。そして A を弾く。
つまり、C ... （沈黙）... A。

間に音を飛ばすことで、聴く人の脳が勝手に「ない音」を補完しようとする。そこに生まれる独特の緊張感 --- これも「表現」の技法の一つ。音を弾かないことで何かを伝える、というのは音楽の中でもかなり深いレベルの話だけど、感覚として体験しておくと、後ですごく効いてくる。

> **海のメモ:** John Mayer は Gravity を弾くたびに、微妙に違う弾き方をする。YouTube で "John Mayer Gravity live" と検索すると、何十本もライブ映像が出てくる。同じ曲なのに全部違う。2005年の LA ライブ、2008年の Where the Light Is、2019年のツアー --- 全部聴き比べると、その時期の彼の感情や人生が音に出ているのが分かる。音楽ってそういうもの。同じ曲でも、今日の自分と明日の自分では違う弾き方になる。それでいい。

## 4. 1音チャレンジ --- 1つの音だけで感情を表現する（10分）

### ルール

使う音は **5弦3フレットの C** だけ。コードも使わない。メロディも弾かない。
たった1つの音で、3つの感情を表現してみよう。

### チャレンジ 1: 悲しい

C の音1つで「悲しさ」を表現して。

ヒント:
- 弱く弾く（pp〜p）
- ゆっくり弾く --- 音と音の間に沈黙を入れる
- 音が消えていくのをじっと聴く
- もう一度弾くときに、少しためらうように

### チャレンジ 2: 嬉しい

同じ C の音で「嬉しさ」を。

ヒント:
- はっきり弾く（f）
- リズミカルに、テンポよく
- 短い間隔で2〜3回弾くのもあり
- 音を弾いた瞬間に笑顔になってみて（本当に表情が音に出る）

### チャレンジ 3: 切ない

「悲しい」と「切ない」は違う。切なさは、悲しみの中にどこか美しさがある感じ。

ヒント:
- 中くらいの強さ（mp〜mf）で、丁寧に弾く
- 1回弾いて、長い沈黙。もう1回弾いて、また沈黙
- 音が空気の中で震えている間、その振動を感じる
- 弾いた後に、弦に指を軽く触れてミュート（音を止める）するのも効果的

> **用語メモ:** mp（メゾピアノ）= やや弱く、mf（メゾフォルテ）= やや強く。pとfの中間くらい。

### 振り返り

3つとも同じ音なのに、全然違う印象になったでしょう？

違いを作ったのは:
- **音量**（強弱）
- **タイミング**（音と音の間隔、ためらい方）
- **音の切り方**（余韻を残すか、すっと止めるか）
- **弾く人の気持ち**（これが一番大きい）
- **体の状態**（呼吸、姿勢、力の入り具合）

**音には、弾く人の心がそのまま乗る。** これは嘘じゃなくて、物理的にも本当のこと。力の入れ方、指の角度、タイミング --- 全部が気持ちに連動してる。

ここで一つ面白い実験をしてみてほしい。**笑顔で C を弾く** のと、**しかめっ面で C を弾く** のを比べてみて。

嘘みたいな話だけど、**表情が音に影響する**。笑顔のときは自然と体がリラックスして、指先のタッチが柔らかくなる。しかめっ面のときは体に力が入って、タッチが硬くなる。結果、同じ C でも音のニュアンスが変わる。これはプロの世界では常識で、John Mayer がステージで目を閉じて気持ちよさそうに弾いてるのは、パフォーマンスだけじゃなく、あの状態が一番いい音を出せるから。

だから、テクニックを磨く前に「何を伝えたいか」を持っていることが大事なんだ。

## 5. C → Am の切り替えを「感情の変化」として弾く（10分）

### コードチェンジ = 場面転換

Lesson 2 で学んだ通り、C はメジャーコード（長3度が入ってる → 明るい）、Am はマイナーコード（短3度が入ってる → 切ない）。

でも今日は理論じゃなくて、この切り替えを**物語の場面転換**として弾いてみよう。

### 練習 5-1: 「晴れから曇り」

```
C（4回ストローク）→ Am（4回ストローク）→ C → Am → ...
```

C を弾いているときは「晴れた日の午後」をイメージして。
Am に変わった瞬間、「急に雲が出てきた」感じ。

**コードを切り替える瞬間に、気持ちも一緒に切り替える。**

やってみて。最初は意識的に。だんだん自然にできるようになる。

### 練習 5-2: ダイナミクスを加える

同じ C → Am だけど、今度は強弱もつけてみる。

```
パターン A:
C（f: しっかり）→ Am（p: そっと）
「楽しい気持ちが、ふと静かな寂しさに変わる」

パターン B:
C（p: 控えめ）→ Am（f: 強く）
「穏やかだった心が、急に切なさで胸がいっぱいになる」
```

**同じ C → Am なのに、強弱を変えるだけで全然違う物語になる。**

どっちが好きだった？ どっちが今の自分の気持ちに近かった？
そういう「選ぶ」ことが、表現の第一歩。

もう一つ試してみよう。

```
パターン C:
C（f: しっかり）→ Am（f: 同じくらいしっかり）
「場面は変わったのに、意志の強さは変わらない。覚悟を持って暗い現実に向き合う感じ」
```

ほら、3つのパターンで全部印象が違うでしょう。**表現の正解は「自分がどう感じるか」**で決まる。

この C → Am の切り替えは、実はポピュラー音楽で最も使われるコードチェンジの一つ。John Mayer の "Say" もこの切り替えが曲の核にある。藤井風の「何なんw」のイントロも C 系と Am 系の行き来が印象的。星野源の「恋」にも C → Am → F → G の進行が使われてる。**基本コードの表現力を最大限に引き出す**のが、優れたミュージシャンの共通点なんだ。

> **海のメモ:** John Mayer の曲って、コード進行自体はそんなに難しくないものが多い。C, G, Am, F みたいな基本コードが中心。でも聴くと全然「基本」に聴こえない。それは彼が一つひとつのコードチェンジに「意味」を込めて弾いているから。音を変えるんじゃなくて、気持ちを変える。その気持ちの変化が音に出る。これが「表現」。彼の "Neon" を聴くと「こんな複雑なことを…」と思うけど、"Gravity" や "Waiting on the World to Change" は使ってるコード自体はシンプル。テクニックの難しさと、音楽の深さは、全くの別物なんだ。

## 6. 好きな曲を「感じながら弾く」練習（5分）

### 最後の練習

今日の仕上げ。自分の好きな曲を1つ選んで、サビの部分だけでいいから弾いてみよう。

ただし、今日学んだことを1つでもいいから意識して:

- **コードチェンジの瞬間に、感情が変わることを感じる**
- **強く弾くところと弱く弾くところを自分で決める**
- **「この部分で何を伝えたいか」を1秒だけ考えてから弾く**

完璧に弾けなくてもいい。間違えてもいい。
大事なのは、**「何かを伝えようとして弾いている」**ということ。

それだけで、昨日までの演奏とは全然違うものになるから。

もし曲が思い浮かばなかったら、Gravity でもいい。あるいは John Mayer の "Free Fallin'" のカバー（これは Tom Petty の原曲で、C, F, Am あたりの基本コードで弾ける）もおすすめ。大事なのは曲の難しさじゃなくて、「この部分、こう弾きたい」という気持ちがあるかどうか。

> **海のメモ:** 実は「好きな曲を感じながら弾く」練習は、プロでもウォームアップとしてやっている人が多い。本番前にただ好きな曲をゆるく弾いて、「音楽を楽しむ」感覚を呼び起こす。指の準備じゃなくて、心の準備。ギターを弾く前の数秒、「今日は何を伝えたいかな」と考える習慣をつけると、毎回の練習が少しだけ特別なものになるよ。

## まとめ

今日やったことを振り返ろう。

1. **短3度と長3度を「感情」として聴いた** --- 暗い/明るいを体で感じた
2. **ダイナミクス（音の強弱）を体験した** --- 同じ音でも弾き方で全然違う
3. **Gravity を「表現」の視点で分析した** --- ダイナミクスの波、下降するインターバル、音色の変化
4. **1音だけで3つの感情を表現した** --- 悲しい、嬉しい、切ない
5. **C → Am を「場面転換」として弾いた** --- コードチェンジに意味を込める
6. **好きな曲を「感じながら」弾いた** --- 伝えたいことを持って弾く

### 一番伝えたいこと

音楽は「正しく弾くこと」じゃない。**「何かを伝えること」**。

コードを3つしか知らなくても、ペンタトニックが弾けなくても、表現はできる。
むしろ、少ない音で表現しようとするときのほうが、1音1音に込める気持ちが強くなる。

これからどんどん新しいコードやスケールを覚えていくけど、今日の「感じながら弾く」感覚を忘れないでほしい。テクニックはあくまで道具。使う人の心が、音楽を音楽にする。

### 3人の講師から学んだことの繋がり

ここまでの4レッスンで学んだことを、**「表現」の視点**からまとめてみるよ。

| レッスン | 学んだこと | 表現との関係 |
|:---|:---|:---|
| Lesson 1（黒田）| 12音の名前 | 表現の「素材」を知った |
| Lesson 2（黒田）| インターバル = 音の距離 | 「なぜ明るいか/暗いか」が分かった |
| Lesson 3（瀬戸）| 指板上でインターバルを使う | 「伝えたいことを弾ける手段」を手に入れた |
| Lesson 4（橋本）| ダイナミクスと感情表現 | 「どう伝えるか」を体験した |

黒田さんが「なぜ」を教えてくれて、瀬戸さんが「どうやって」を教えてくれて、僕が「何のために」を伝えた。この3つが揃ったとき、初めて「音楽をする」ことが始まる。

そしてこれは一方通行じゃない。表現したいことが増えると「もっと理論を知りたい」と思うようになるし、指が動くようになると「もっと深い感情を込めたい」と思うようになる。理論 → 実践 → 表現のサイクルは、ここから先ずっと回り続けるもの。楽しみにしていてね。

> **海のメモ:** 今日のレッスンで一番大事なのは、実は「上手くなること」じゃなくて、「自分がギターで何かを感じた」という体験そのもの。1音の C を弾いて心が動いた瞬間 --- あれが音楽の原点。どんなにテクニックが上達しても、あの瞬間の新鮮さを忘れないでいてほしい。いつか壁にぶつかったとき、今日のこの感覚が帰る場所になるから。
