# 3ヶ月ロードマップ

2026年2月〜5月（最終出社日: 2月13日）

## Month 1（2月後半〜3月前半）: セットアップ期

生活リズムの確立。各aspectの基盤を整える。

| aspect | 目標 |
|--------|------|
| sumitsugi | ロードマップ整理、開発スプリント開始 |
| ジム | 入会、生活リズムに運動を組み込む、食事管理開始 |
| ギター | 1時間練習の習慣化、Phase 2開始 |
| 投資 | ポートフォリオ棚卸し、NISA戦略検討 |
| study | 起業に必要な法律の基礎学習開始 |
| 福岡 | 情報収集フェーズ |

**このフェーズのキーワード:** 「習慣をつくる」

## Month 2（3月中旬〜4月前半）: 加速期

習慣が回り始めたら、成果に向けてアクセルを踏む。

| aspect | 目標 |
|--------|------|
| sumitsugi | MVP or 主要機能の完成に向けて |
| ジム | 運動習慣が定着、食事管理の微調整 |
| ギター | Phase 2のコア完了 |
| 投資 | 実際のポジション構築 |
| study | 法人設立の具体検討 |
| 福岡 | 一時滞在の検討 |

**このフェーズのキーワード:** 「加速する」

## Month 3（4月中旬〜5月前半）: 成果期

3ヶ月の集中期間の仕上げ。成果を出し切る。

| aspect | 目標 |
|--------|------|
| sumitsugi | リリース or ベータ |
| ジム | -5kg 達成目標 |
| ギター | Phase 2完了 → Phase 3開始 |
| 投資 | ポートフォリオ運用・振り返り |
| study | 起業準備の具体化 |
| 福岡 | 判断 |

**このフェーズのキーワード:** 「成果を出す」

## 全体の優先順位

1. sumitsugi開発（本業。2-3ヶ月集中。開発+営業）
2. 運動/ジム（3ヶ月で5kg減量）
3. ギター（1日1時間）
4. 投資
5. study（起業知識・技術）
6. 読書
7. 福岡移住検討

迷ったらこの順番で判断する。上位を犠牲にして下位をやらない。
