# God is a True Friend
> *わたしはもう、あなたがたをしもべとは呼びません。しもべは主人が何をしているか知らないからです。わたしはあなたがたを友と呼びました。父から聞いたことをすべて、あなたがたに知らせたからです。*
> 

ヨハネの福音書15:15

ウェイン牧師

私たちには避雷針の友達が必要

- 私たちのそばに寄り添い時間をかけて私たちの怒りの言葉を聞いてくれる
    - これをすることによって私たちの心を守ってくれる

ただし、夫や妻を避雷針にしてはいけない

- 結婚するということは精神的にも感情的にも相手を深く結びつくことだから
- 相手は自分自身のものとして受け取ってしまう

より良い方法は自分が信頼できる友人に自分の不満をぶちまけてしまっていることを配偶者に伝える

避雷針がいない場合は？

神様にお願いしてみよう！

神様はいつも友達として、歩み寄ろうとしている！理由があって、神様がまだ与えてくれていないのかもしれない。もしかしたら、あなたが先に変わる必要があるのかも

私たちが周囲の人々の避雷針として選んでくれているかもしれない

- 物理的な地面に繋がっていて、始めて、避雷針の役割を果たしているように、私たちも、まずはイエス様と繋がっていないといけない

> *人が私に留まり、私もその人に留まっているなら、その人は多くの身を結びます。私を離れては、あなたがたは何もすることができないのです。*
> 

ヨハネの福音書15:5
