## ① TABで弾く

### STEP 1: ダイナミクスの基本

**ダイナミクス** = 音の強弱のこと。ピッキング（弦を弾く）の力加減でコントロールする。

**強弱の記号:**

```
pp（ピアニッシモ）= とても弱く。ささやくように
p （ピアノ）    = 弱く。控えめに
mp（メゾピアノ） = やや弱く
mf（メゾフォルテ）= やや強く。普通の強さ
f （フォルテ）   = 強く。しっかり
ff（フォルティッシモ）= とても強く。全力で
```

> 橋本: 多くの初心者は全部 mf（普通の強さ）で弾いてる。それだと「平坦」に聞こえる。pp から ff まで使い分けるだけで、同じフレーズが別物になるよ。

**ダイナミクスのコントロール方法:**

```
弱く弾く（pp-p）:
  - ピックを浅く当てる（弦に触れるくらい）
  - 右手の力を抜く
  - ピックを柔らかく持つ（握り込まない）
  - 指弾きの場合は指の腹で優しく撫でる

強く弾く（f-ff）:
  - ピックを深く当てる（弦をしっかり弾く）
  - 右手に力を込める（でも力みすぎない）
  - ピックをしっかり持つ
  - 指弾きの場合は指を引っ掛けるように弾く

※ 左手は変えない。右手だけで強弱をつける
```

まず同じ音（2弦0フレット B）を強弱を変えて弾いてみよう。

```
BPM なし（フリーテンポ）

e |----------------------------------------------|
B |--0------0------0------0------0------0--------|
G |----------------------------------------------|
D |----------------------------------------------|
A |----------------------------------------------|
E |----------------------------------------------|
     pp     p      mp     mf     f      ff

  ※ 同じ音を6回。pp（超弱い）から ff（超強い）まで段階的に大きくする
  ※ 6回目が1回目の何倍も大きく聞こえるように
```

> 橋本: pp と ff の差が大きいほど、表現の幅が広がる。最初は pp が弱すぎて音が出なくてもいい。ff が力みすぎて汚い音でもいい。両極端を知ることが大事。そこから「ちょうどいい」を見つけていく。

**やること:** 2弦0フレットで pp → ff のグラデーション。6回 × 3セット。

**mf バージョン（普通）:**

```

強弱: mf  mf  mf  mf  mf  mf  mf

  ※ いつも通りの力で弾く。これが基準
```

**ff バージョン（叫び）:**

```
【ff: とても強く】
e |----------------------------------------------|
B |----------------------------------------------|
G |----------------------------------------------|
D |----------------------------------------------|
A |----------------------------------------------|

Beat: 1       2   &   3   &   4
強弱: ff  ff  ff  ff  ff  ff  ff

  ※ 力強くガッと弾く。怒ってるくらいの勢いで
  ※ でも雑にならないように。音はクリアに
```

> 橋本: 同じフレーズなのに、全然違う印象でしょ？ pp は悲しみ、mf は語り、ff は叫び。音符は同じ。変わったのは「力加減」だけ。これがダイナミクスの力。

> 瀬戸: pp で弾くの、意外と難しいでしょ？ 小さい音をきれいに出すのって、大きい音を出すより難しいんだよ。でもこれができるようになると、ソロの表現力が爆上がりする。

**やること:** pp → mf → ff の順に3回弾く。次に ff → mf → pp の順（逆パターン）。計6回。

### STEP 3: 「間（ま）」の練習

> 橋本: 間は「何もしてない」時間じゃない。前のフレーズの余韻が残ってる時間。次のフレーズへの期待が溜まっていく時間。間があるからこそ、次の音が「来た！」って感じる。

### STEP 4: BB King風フレーズ1 & 2 — 少ない音で泣かせる

BB King のスタイル: 音が少ない。1-2音だけ弾いて、あとはビブラートと間。それだけで人の心を動かす。

**BB King風フレーズ 1: 1音のビブラート**

```
Key of E  BPM 65  スローブルース

【BB King風フレーズ 1: たった1音】
e |----------------------------------------------|----------------------------------------------|
B |--3b(4)~~~~~~~~~~~~~~~~~~~--------------------|----------------------------------------------|
G |----------------------------------------------|----------------------------------------------|
D |----------------------------------------------|----------------------------------------------|
A |----------------------------------------------|----------------------------------------------|
E |----------------------------------------------|----------------------------------------------|
Beat: 1~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ 2  3  4  1   2   3   4（←2小節目は完全に休み）

  ※ 2弦3フレットをベンドして、そのままビブラートで伸ばし続ける
  ※ 2小節分（8拍）のうち、音を出してるのは最初の4拍だけ
  ※ 残り4拍は「余韻」。指を離して、音が消えていくのを聴く
```

> 橋本: この1音に全てを込める。ベンドのスピード（ゆっくり上げるか、素早く上げるか）、ビブラートの幅（大きく揺らすか、小さく揺らすか）、ビブラートの速さ（ゆっくり揺らすか、速く揺らすか）。1音の中にこれだけのパラメータがある。1音で3種類の表現ができる。

```
【ビブラートのバリエーション】

バリエーションA: ゆっくり大きく揺らす → 悲しみ・嘆き
  B |--3b(4)~~~~~~~~~~~~~~~~~~~~~~~|
  ビブラート: ゆ〜〜〜ら〜〜〜ゆ〜〜〜ら〜〜〜

バリエーションB: 速く小さく揺らす → 切迫感・焦り
  B |--3b(4)~~~~~~~~~~~~~~~~~~~~~~~|
  ビブラート: ゆらゆらゆらゆらゆらゆらゆらゆら

バリエーションC: 最初ゆっくり → 徐々に速く → フェードアウト
  B |--3b(4)~~~~~~~~~~~~~~~~~~~~~~~|
  ビブラート: ゆ〜〜ら〜〜ゆ〜らゆらゆらゆら...
```

**やること:** フレーズ1を3回。A, B, C のビブラートバリエーションを各2回。

**やること:** フレーズ2を4回。毎回違う感情を込めて弾く。1回目は悲しく、2回目は怒りを込めて、3回目は切なく、4回目は希望を持って。

> 橋本: バージョンAとBを弾き比べてみて。どっちが「心に残る」？ 技術的にはAの方が難しい。でも音楽的にはBの方が伝わることが多い。なぜかっていうと、Bには「呼吸」があるから。人間は息を止め続けられない。音楽も同じ。「吸って、吐いて」のリズムが自然だから心地いい。

> 瀬戸: これは好みの問題じゃなくて、「使い分け」の問題。テンポが速い曲なら音を詰めた方がカッコいいし、スローブルースなら間を活かした方が泣ける。両方できるのが理想。でもまずは「間を恐れない」ことから始めよう。

**やること:**
1. バージョンA（詰め込み）を1回弾く
2. バージョンB（スカスカ）を1回弾く
3. どちらが好きか、体で感じる
4. バージョンBを3回繰り返す。「間」を味わう

## ② 事後理論

> 黒田（理論メモ）: ダイナミクスと「間」について。

### なぜ「弾かない」がカッコいいのか

> 黒田: 映画で例えると、ずっと爆発シーンが続いたら疲れるよね。静かなシーンがあるから、爆発シーンが際立つ。音楽も同じで、「静」があるから「動」が生きる。
>
> 心理学的にも「コントラスト効果」と言って、人間は「差」に反応する。pp の後の ff は、最初から ff で弾いたときの何倍も強く感じる。間（沈黙）の後の1音は、音が連続しているときの何倍もインパクトがある。
>
> BB King、Miles Davis、John Mayer。ジャンルを超えて「間」の達人はたくさんいる。彼らに共通するのは「弾かない勇気」。音を出すことより、出さないことの方が難しい。でもそれができると、表現者として1段上に行ける。

## ③ 応用ドリル

### ドリル 1: 「弾いていい音は N個だけ」ルール

12小節を通すが、弾いていい音の数に制限をかける。

```
【レベル1: 24音まで（1小節あたり平均2音）】
  → 12小節で24音。半分以上の時間は「間」

【レベル2: 12音まで（1小節あたり平均1音）】
  → 極限の「間」。どこに音を置くかを厳選する

【レベル3: 6音まで】
  → 12小節で6音。2小節に1音。BB King レベル
```

> 瀬戸: レベル3はマジでキツい。12小節で6音って、「どの6音を選ぶか」で全てが決まる。でもこれができると「1音の価値」がわかるようになる。全部の音に意味を持たせる感覚。

**やること:** レベル1を3回 → レベル2を3回 → レベル3を2回。

---

### ドリル 2: クレッシェンド＆デクレッシェンド12小節

12小節全体を1つの大きなクレッシェンド（だんだん大きく）にする。

```
Key of E  BPM 70  シャッフル

小節1-4（E7）:   pp で弾く（ささやき。ほぼ聞こえない）
小節5-6（A7）:   p → mp（少しずつ大きく）
小節7-8（E7）:   mf（普通の強さ）
小節9（B7）:     f → ff（クライマックス。全力）
小節10-12:       ff → p（一気に落とす。デクレッシェンド）
```

> 橋本: 12小節で1つの「波」を作る。海の波が小さくなって...大きくなって...ドカーンと来て...引いていく。その感覚。リックは何でもいい。大事なのは音量の「設計」。

**やること:** 3回通す。1回目は大げさにやる（pp と ff の差をめちゃくちゃ大きく）。2回目で自然な差に調整。3回目で「自分の好きなバランス」を見つける。

### ドリル 3: 好きなアーティストのソロを「間」で分析

YouTube で BB King のライブを1曲聴いて、「弾いてる時間」と「弾いてない時間」を観察する。

**おすすめ動画:**
- "BB King - The Thrill Is Gone (Live)"
- "BB King - Live at the Royal Albert Hall" の任意のソロ

**観察ポイント:**
```
1. 1フレーズ弾いた後、何拍休んでいるか？
2. ベンドの後、どのくらい音を伸ばしているか？
3. クライマックスの瞬間は12小節のどこか？
4. 一番小さい音と一番大きい音の差はどのくらいか？
5. 「弾いてる時間」と「弾いてない時間」の比率は？
```

> 橋本: 聴いたことを自分の演奏に取り入れてみて。「BB King はここで3拍休んでた」→ 自分も3拍休んでみる。模倣から始まるのは全然OK。プロも全員、誰かの模倣から始まってる。

## ④ 自分で組み合わせる

### ダイナミクス＆間を使ったマイ・ソロ

L14-L15 で作ったソロに、今日学んだダイナミクスと間を加えてみよう。

**ダイナミクス設計シート:**

```
【マイ・ソロのダイナミクス設計】

起（1-4小節）:
  音量: pp → p
  音数: 少なめ（1小節3-4音）
  間: 2小節目と4小節目は休む or 1音だけ

承（5-8小節）:
  音量: p → mf → f
  音数: 増やしていく
  間: 8小節目は休む（転への溜め）

転（9-12小節）:
  音量: ff（9小節）→ p（11-12小節）
  音数: 9小節目は多め → 10小節目は休む → 11-12小節は少なめ
  間: 10小節目は完全に休む

全体のアーチ: pp → ff → p
```

**やること:**
1. L14 or L15 で作ったソロのTABを用意する
2. 上の設計シートに沿って、各小節の強弱を決める
3. 通しで弾く。強弱を意識しながら
4. 休む小節を増やしたバージョンも試す
5. 一番気持ちいいバランスを見つける

> 瀬戸: Phase 4 の全技術がここに集まった。コール＆レスポンス（L13）、ソロ構築（L14）、コードチェンジ（L15）、ダイナミクス＆間（L16）。全部組み合わせたら、もう「自分のブルースソロ」が弾ける。12小節を自分の物語として語れるギタリストになったってこと。マジでおめでとう。

> 橋本: ここまで来たら、あとは「弾き込む」だけ。テクニックはもう十分ある。大事なのは「何を表現したいか」。嬉しいとき、悲しいとき、怒ってるとき、ギターで語ってみて。それがブルースの本質だから。Phase 5 ではもっと色んなスタイルに挑戦するよ。楽しみにしてて。
