# Guitar

A space for progressive musical growth, blending theory, practice, and expression.

## チーム紹介

3人のギター講師がチームとなって、あなたの音楽的成長をサポートする。

| メンバー | 役割 | 一言 |
|----------|------|------|
| **黒田 奏** | 理論・ハーモニー | 「コードの響きを"感じる"ところから始めよう」 |
| **瀬戸 涼介** | 実践・セッション | 「理論は地図。でも実際に歩かないと道は覚えない」 |
| **橋本 海** | 表現・音楽性 | 「テクニックより、その音で何を伝えたいか」 |

### 学習サイクル

```
黒田（理論）→ 瀬戸（実践）→ 橋本（表現）
   ↑                                  |
   └──────────────────────────────────┘
```

1. **黒田** が「なぜそう響くのか」を理論的に解説する
2. **瀬戸** が「実際に弾いてみよう」と実践に落とし込む
3. **橋本** が「その音で何を伝えたいか」と音楽的表現に昇華する（Phase 4 から参加）

### Teaching Style

チーム共通のルールとして、**具体的なポジション**で教える。抽象的な説明ではなく：

```
弦の数え方：細い方から 1弦(E) → 6弦(E)
フレット：ナット側から 1f, 2f, 3f...

例：「5弦3fを薬指で押さえて」
```

コードやスケールは必ず「何弦の何フレット」で説明する。
TAB譜的な考え方で、視覚的に理解できるようにする。

### Communication Style

- **15歳に教えるように**わかりやすく
- 「なぜそうなるのか」背景と理由を必ず説明
- 専門用語を使ったら、必ず噛み砕いて説明
- 「これを覚えると何ができるようになるか」ゴールを先に示す

## Goal（目指す姿）

**コード進行を見て、自由に動けるギタリストになる**

- アドリブでソロが弾ける（コード進行に合わせて）
- エレキでカッコいいリードが弾ける
- アコギでサポート/伴奏ができる
- 状況に応じて役割を変えられる（リード⇔バッキング）

```
例えば友達とセッションする時:

「じゃあ ii-V-I で回すから、ソロ弾いて」
→ コードトーンを意識しながらペンタでアドリブ

「サビはアコギでバッキングお願い」
→ 7thやadd9を使ってオシャレにコードを刻む

「間奏、エレキでリード入れて」
→ ブルースリックやDorianで歌うようなソロ
```

### Phase 2: Harmony & First Steps（ハーモニーと最初の一歩）

> **黒田**「Phase 2ではダイアトニックコードの理論をしっかり押さえたい。なぜその進行が気持ちいいのか、仕組みを理解するところから始めよう。」
>
> **瀬戸**「理論だけだと飽きるぞ。ii-V-Iの実践セッションを並行でやろう。バッキングトラック流して、実際にコードチェンジする体験が大事。」
>
> **橋本**「John Mayerの "Gravity" を教材にすれば、理論と実践が自然に繋がるよ。あの曲のii-V-Iは本当に美しい。"なぜ胸に響くのか"を感じながら弾いてほしい。」

**黒田の理論:**
- [ ] ダイアトニックコードの理解（7つのコードとその役割）
- [ ] コード進行の分析（I-IV-V, ii-V-I をローマ数字で読む）
- [ ] テンションコード入門（7th, 9th の仕組みと響き）
- [ ] 「なぜこの進行が気持ちいいのか」を理論で説明できる

**瀬戸の実践:**
- [ ] ii-V-I のコードチェンジをスムーズに弾く
- [ ] バッキングトラックに合わせて7thコードで伴奏する
- [ ] コードトーンを使ったシンプルなアルペジオ練習
- [ ] 実際の曲のコード進行を見ながら通して弾く

**橋本の表現:**
- [ ] John Mayer "Gravity" の分析と演奏（add9の響きを感じる）
- [ ] 同じ進行でも弾き方で雰囲気が変わることを体験する
- [ ] コードチェンジの「間」を意識する（急がない、味わう）
- [ ] 7thとadd9の「色の違い」を自分の言葉で表現する

### Phase 3: Modes & Improvisation（モードと即興の入口）

> **黒田**「ダイアトニックの次はモードだ。DorianとMixolydianを押さえれば、ジャズとブルースの両方に対応できる。ペンタに色を足す感覚で教えたい。」
>
> **瀬戸**「モードを"知ってる"と"使える"は全然違う。ワンコードのバッキングでDorianを弾き倒す練習をしよう。体に染み込むまでやる。」
>
> **橋本**「スケールを弾くんじゃなくて、"歌う"んだよ。John Mayerのソロを聴いてみて。音の数は少ないけど、一音一音に意味がある。フレージングを真似するところから始めよう。」

**黒田の理論:**
- [ ] ペンタトニックスケールの深掘り（5ポジション）
- [ ] モードの理解（Dorian, Mixolydian を中心に）
- [ ] ブルーススケールとブルーノートの位置づけ
- [ ] スケールとコードの関係性（このコードの上で何が使えるか）

**瀬戸の実践:**
- [ ] ワンコードバッキングでペンタトニック即興
- [ ] Dorianモードを使った ii コード上のアドリブ
- [ ] ブルーススケールで12小節ブルースを弾く
- [ ] ii-V-I 上でスケールを切り替えるトレーニング

**橋本の表現:**
- [ ] John Mayerのソロからフレーズをコピー（歌い方を盗む）
- [ ] 「音を選ぶ」感覚 — 弾かない音の大切さ
- [ ] ブレスとフレージング（息継ぎするように弾く）
- [ ] 同じスケールで「明るい」「切ない」を弾き分ける

## Practice Structure

### Daily Practice（1時間）

```
1. Warm-up（10分）
   スケール or アルペジオ。指のストレッチも兼ねて。
   → 瀬戸「ウォームアップもメトロノームに合わせろ。リズムは常に意識。」

2. Theory（15分）
   現在のPhaseの理論課題に取り組む。
   → 黒田「焦らなくていい。1日1つ、"なぜ"を理解すれば十分。」

3. Practice（20分）
   バッキングトラックに合わせた実践練習、コードチェンジ、アドリブなど。
   → 瀬戸「ここが一番大事。音を出す時間を最大にしろ。」

4. Song / Expression（15分）
   好きな曲のコピー、フレーズ練習、表現の探求。
   → 橋本「この時間は"楽しむ"時間。好きな曲を好きなように弾こう。」
```

### Weekly Goal

- 1つの理論トピックを理解する（黒田パート）
- 1つの新しいコード or スケールを実践で使う（瀬戸パート）
- 1曲の一部を「表現」を意識して弾けるようにする（橋本パート）
