テクニックは後からいくらでもつく。でも「音を聴く耳」と「感じる心」は、今この瞬間から育てていけるもの。

## 1. 聴く練習 --- 短3度と長3度の「感情」を言葉にする（10分）

### まずは、弾いて聴く

ギターを手に取って。5弦3フレット（C）を出発点にする。

```
【TAB譜】短3度: C → Eb

e |------|
B |------|
G |------|
D |------|
A |--3---6---|
E |------|
    C   Eb
```

C を弾いて、少し間を置いて、Eb を弾く。
急がないで。C の音が空気に溶けていくのを待ってから、Eb を弾いてみて。

**何を感じた？**

言葉にしてみて。「悲しい」でも「切ない」でも「雨の日みたい」でも「胸が重い」でも、なんでもいい。
正解はない。自分の中に浮かんだ言葉が、それが答え。

### 次に、長3度

```
【TAB譜】長3度: C → E

e |------|
B |------|
G |------|
D |------|
A |--3---7---|
E |------|
    C    E
```

同じように。C を弾いて、間を置いて、E を弾く。

**何を感じた？ さっきと何が違う？**

多くの人が「明るい」「開ける感じ」「空が見えた」みたいに言う。でも、もしかしたら「懐かしい」とか「朝っぽい」とか、自分だけの言葉が出てくるかもしれない。それが一番いい。

### 交互に弾いてみる

```
C → Eb（短3度）→ C → E（長3度）→ C → Eb → C → E ...
```

これを5回くらい繰り返す。
だんだん、弾く前から「次はこういう気持ちになるな」って分かるようになる。
それが**インターバルを「体で覚えた」**ということ。

> **海のメモ:** 音楽理論の教科書には「短3度 = 暗い、長3度 = 明るい」って書いてある。でも、実際に弾いて感じることはそれよりずっと豊かなんだよね。同じ短3度でも、弾く人によって「悲しい」だったり「美しい」だったり「雨上がり」だったりする。その違いが個性になるし、表現になる。

## 2. ダイナミクス --- 同じ音でも弾き方で世界が変わる（10分）

### ダイナミクスって何？

ダイナミクスというのは、**音の強弱**（きょうじゃく）のこと。
ピアノなら鍵盤を強く押すか弱く押すか。ギターなら弦を強く弾くか、そっと弾くか。

同じ C の音でも、弾き方を変えるだけで全然違うものになる。
試してみよう。

### 練習 2-1: 強弱の4段階

5弦3フレット（C）を1つだけ使って、4段階で弾いてみて。

| 段階 | 弾き方 | イメージ |
|:---:|:---|:---|
| **pp（ピアニッシモ）** | 弦にそっと触れるように。指先で撫でるくらい | 夜中の独り言。誰にも聞かれたくない秘密 |
| **p（ピアノ）** | 軽く弾く。音はちゃんと出るけど控えめ | 朝の挨拶。穏やかに「おはよう」 |
| **f（フォルテ）** | しっかり弾く。芯のある音 | 大事なことを伝えるとき。「聞いてほしい」 |
| **ff（フォルティッシモ）** | 力強く弾く。弦が大きく振動する | 叫び。感情が溢れ出す瞬間 |

> **用語メモ:** pp, p, f, ff はクラシック音楽で使われる音量の記号。イタリア語が語源で、p = piano（弱い）、f = forte（強い）という意味。ギターの世界では記号よりも感覚が大事なので、「そっと」「しっかり」くらいの理解で十分。

### 練習 2-2: 同じ音を4回弾く --- 物語をつくる

C の音を4回弾くだけ。でも強弱を変える。

```
パターン A: pp → p → f → ff
（だんだん大きく → 決意が固まっていく感じ）

パターン B: ff → f → p → pp
（だんだん小さく → 余韻が消えていく。別れの場面）

パターン C: pp → ff → pp → pp
（急に叫んで、すぐ静かに → 驚き、または心の中の爆発）
```

たった1つの音を4回弾いただけなのに、それぞれ全然違う「物語」に聞こえない？

**これがダイナミクスの力。**

テクニックがなくても、コードをたくさん知らなくても、音の強弱だけで感情は伝わる。

> **海のメモ:** John Mayer のすごいところの一つは、このダイナミクスの幅がとてつもなく広いこと。ささやくような音から、ステージを震わせるような音まで、同じギターから全く違う声が出てくる。それは指の力じゃなくて、「ここはこう弾きたい」という意志の問題。意志があれば、指はそれについてくる。

## 3. Gravity サビの表現分析 --- 音で「重力」を描く（15分）

### まず聴いてみて

Gravity のサビ部分を、歌詞を読みながら聴いてみよう。
YouTube でも Spotify でもいいから、目を閉じて聴いてみて。

歌詞の核心:
```
Gravity is working against me
And gravity wants to bring me down
（重力が僕に逆らっている）
（重力が僕を引きずり下ろそうとしている）
```

**何を感じた？ 音はどう動いてた？**

### 音の「重力」--- メロディの下降

Lesson 2 で黒田さんが教えてくれた通り、Gravity のベースラインは C → B → A と下がっていく。
でも今日注目したいのは、それが**どう弾かれているか**ということ。

サビに入る直前、John Mayer はグッと音量を上げる。
そして "Gravity" と歌い始めるところで、ベースラインが下がり始める。

```
【サビの流れ】

[サビ前: 静かめ、ためる]
         ↓
[「Gravity」: 一気に音量アップ、エモーショナルなピーク]
         ↓
[ベースが C → B → A と下降: 引力に引かれるように落ちていく]
         ↓
[フレーズの終わり: 音量がすっと落ちる → また次の波へ]
```

ここで起きていることを分解すると:

1. **ダイナミクスの波**: 静 → 動 → 静 のサイクル
2. **インターバルの方向**: 下降（長2度ずつ下がる）→ 重力で引っ張られる感覚
3. **音色の変化**: クリーンから少しドライブがかかる → 感情の高まり

### 練習 3-1: Gravity のベースラインを「表現」で弾く

Lesson 2 で弾いた C → B → A のベースラインを、今度は「表現」を意識して弾いてみよう。

```
【TAB譜】ベースラインのみ

A弦: --3----2----0----

      C     B     A
```

ただし、3つの弾き方で弾いてみて。

**弾き方 1: 平坦に（比較用）**
全部同じ強さで、同じタイミングで。「ド、シ、ラ」と機械的に。

**弾き方 2: 重力を感じて**
C を一番強く弾いて、B は少し弱く、A はさらに弱く。
「高いところから落ちていく」感じ。音量が重力に引かれるように小さくなっていく。

**弾き方 3: 最後に抗って**
C を強く、B を弱く、でも A をもう一度しっかり弾く。
「落ちそうになるけど、最後に踏ん張る」感じ。John Mayer がこの曲で歌っているのは、まさにそういう気持ち。

**3つのうち、どれが一番「Gravity」っぽかった？**

正解はないけど、自分の中で「これだ」と思えるものがあったら、それが今の自分の解釈。
それを大事にしてほしい。

> **海のメモ:** John Mayer は Gravity を弾くたびに、微妙に違う弾き方をする。それはこの曲が「完成品」じゃなくて、弾くたびにその時の気持ちを込めているから。音楽ってそういうもの。同じ曲でも、今日の自分と明日の自分では違う弾き方になる。それでいい。

## 4. 1音チャレンジ --- 1つの音だけで感情を表現する（10分）

### ルール

使う音は **5弦3フレットの C** だけ。コードも使わない。メロディも弾かない。
たった1つの音で、3つの感情を表現してみよう。

### チャレンジ 1: 悲しい

C の音1つで「悲しさ」を表現して。

ヒント:
- 弱く弾く（pp〜p）
- ゆっくり弾く --- 音と音の間に沈黙を入れる
- 音が消えていくのをじっと聴く
- もう一度弾くときに、少しためらうように

### チャレンジ 2: 嬉しい

同じ C の音で「嬉しさ」を。

ヒント:
- はっきり弾く（f）
- リズミカルに、テンポよく
- 短い間隔で2〜3回弾くのもあり
- 音を弾いた瞬間に笑顔になってみて（本当に表情が音に出る）

### チャレンジ 3: 切ない

「悲しい」と「切ない」は違う。切なさは、悲しみの中にどこか美しさがある感じ。

ヒント:
- 中くらいの強さ（mp〜mf）で、丁寧に弾く
- 1回弾いて、長い沈黙。もう1回弾いて、また沈黙
- 音が空気の中で震えている間、その振動を感じる
- 弾いた後に、弦に指を軽く触れてミュート（音を止める）するのも効果的

> **用語メモ:** mp（メゾピアノ）= やや弱く、mf（メゾフォルテ）= やや強く。pとfの中間くらい。

### 振り返り

3つとも同じ音なのに、全然違う印象になったでしょう？

違いを作ったのは:
- **音量**（強弱）
- **タイミング**（音と音の間隔、ためらい方）
- **音の切り方**（余韻を残すか、すっと止めるか）
- **弾く人の気持ち**（これが一番大きい）

**音には、弾く人の心がそのまま乗る。** これは嘘じゃなくて、物理的にも本当のこと。力の入れ方、指の角度、タイミング --- 全部が気持ちに連動してる。

だから、テクニックを磨く前に「何を伝えたいか」を持っていることが大事なんだ。

## 5. C → Am の切り替えを「感情の変化」として弾く（10分）

### コードチェンジ = 場面転換

Lesson 2 で学んだ通り、C はメジャーコード（長3度が入ってる → 明るい）、Am はマイナーコード（短3度が入ってる → 切ない）。

でも今日は理論じゃなくて、この切り替えを**物語の場面転換**として弾いてみよう。

### 練習 5-1: 「晴れから曇り」

```
C（4回ストローク）→ Am（4回ストローク）→ C → Am → ...
```

C を弾いているときは「晴れた日の午後」をイメージして。
Am に変わった瞬間、「急に雲が出てきた」感じ。

**コードを切り替える瞬間に、気持ちも一緒に切り替える。**

やってみて。最初は意識的に。だんだん自然にできるようになる。

### 練習 5-2: ダイナミクスを加える

同じ C → Am だけど、今度は強弱もつけてみる。

```
パターン A:
C（f: しっかり）→ Am（p: そっと）
「楽しい気持ちが、ふと静かな寂しさに変わる」

パターン B:
C（p: 控えめ）→ Am（f: 強く）
「穏やかだった心が、急に切なさで胸がいっぱいになる」
```

**同じ C → Am なのに、強弱を変えるだけで全然違う物語になる。**

どっちが好きだった？ どっちが今の自分の気持ちに近かった？
そういう「選ぶ」ことが、表現の第一歩。

> **海のメモ:** John Mayer の曲って、コード進行自体はそんなに難しくないものが多い。C, G, Am, F みたいな基本コードが中心。でも聴くと全然「基本」に聴こえない。それは彼が一つひとつのコードチェンジに「意味」を込めて弾いているから。音を変えるんじゃなくて、気持ちを変える。その気持ちの変化が音に出る。これが「表現」。

## 6. 好きな曲を「感じながら弾く」練習（5分）

### 最後の練習

今日の仕上げ。自分の好きな曲を1つ選んで、サビの部分だけでいいから弾いてみよう。

ただし、今日学んだことを1つでもいいから意識して:

- **コードチェンジの瞬間に、感情が変わることを感じる**
- **強く弾くところと弱く弾くところを自分で決める**
- **「この部分で何を伝えたいか」を1秒だけ考えてから弾く**

完璧に弾けなくてもいい。間違えてもいい。
大事なのは、**「何かを伝えようとして弾いている」**ということ。

それだけで、昨日までの演奏とは全然違うものになるから。

## まとめ

今日やったことを振り返ろう。

1. **短3度と長3度を「感情」として聴いた** --- 暗い/明るいを体で感じた
2. **ダイナミクス（音の強弱）を体験した** --- 同じ音でも弾き方で全然違う
3. **Gravity を「表現」の視点で分析した** --- ダイナミクスの波、下降するインターバル、音色の変化
4. **1音だけで3つの感情を表現した** --- 悲しい、嬉しい、切ない
5. **C → Am を「場面転換」として弾いた** --- コードチェンジに意味を込める
6. **好きな曲を「感じながら」弾いた** --- 伝えたいことを持って弾く

### 一番伝えたいこと

音楽は「正しく弾くこと」じゃない。**「何かを伝えること」**。

コードを3つしか知らなくても、ペンタトニックが弾けなくても、表現はできる。
むしろ、少ない音で表現しようとするときのほうが、1音1音に込める気持ちが強くなる。

これからどんどん新しいコードやスケールを覚えていくけど、今日の「感じながら弾く」感覚を忘れないでほしい。テクニックはあくまで道具。使う人の心が、音楽を音楽にする。

### 次回予告

次は黒田さんに戻って、新しい理論のトピックに入っていくよ。でも今日やった「表現」の視点は、これから先ずっと使い続けるもの。理論を学ぶときも「これを知ったら、どんな感情が表現できるようになるんだろう」って考えてみてね。

### 練習のポイント

- **上手く弾こうとしない。** 今週は「感じること」が目標。ミスしても気にしない。
- **音を出したら、その音を最後まで聴く。** 弾いた瞬間だけじゃなく、音が消えていくところまで聴いてみて。そこに物語がある。
- **言葉にする習慣をつける。** 「今の音、どんな感じだった？」と自分に問いかけてみて。言葉にすることで、感覚が鮮明になる。
- **できれば、目を閉じて弾く時間を作る。** 視覚を遮断すると、耳が敏感になる。音がもっと近くに聴こえてくるよ。
