# Phase 1: Foundation（基礎）

音楽の基本言語を身につけるフェーズ。

## Goals

このフェーズを終えると:
- 音名とインターバルが瞬時に分かる
- メジャースケールを複数のポジションで弾ける
- 基本的なコードを綺麗に鳴らせる
- リズムに乗って演奏できる

## Topics

### 1. 音名とインターバル
- 12音の名前（C, C#, D...）
- インターバル（度数）: 2度、3度、4度、5度...
- 全音と半音の関係

### 2. メジャースケール
- メジャースケールの構造（全全半全全全半）
- Cメジャースケールから始める
- 5つのポジション（CAGED system入門）

### 3. 基本コード
- オープンコード（C, G, D, A, E, Am, Em, Dm）
- バレーコード入門（F, Bm）
- 7thコード入門（G7, C7, Am7）

### 4. リズム
- 4/4拍子の理解
- 8分音符のストローク
- 16分音符のストローク
- シンコペーション入門

## Practice Songs

基礎を固めるのに最適な曲:
- 「Gravity」- John Mayer（シンプルなコード進行）
- 「Waiting on the World to Change」- John Mayer
