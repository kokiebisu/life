# Phase 2: Harmony（ハーモニー）

コードの世界を広げるフェーズ。「なぜこの進行が気持ちいいのか」を体で理解する。

**このフェーズで得られるもの:**
- 曲を聴いて「あ、この進行ね」と分かるようになる
- 7thコードを使ってオシャレな響きが作れる
- John MayerやKing Gnuの曲を分析できるようになる

---

## Lesson 2-1: ダイアトニックコードを知る

### このレッスンのゴール
「この曲、Key=Cだから使えるコードはこれだな」と分かるようになる。

### そもそも「Key（キー）」って何？

曲には「ホーム」になる音がある。それがKey。

例えば、Key=C の曲は「ド」が家。
どんなに遠くに行っても、最後は「ド」に帰ってくると落ち着く。

カラオケで「キーを下げる」ってあるでしょ？
あれは家の位置を下げてるってこと。

### ダイアトニックコードって何？

**Key=C（ドレミファソラシド）の音だけで作れる7つのコード。**

なぜこれが大事かというと...
ほとんどのポップスは、この7つのコードの組み合わせでできてるから！

```
Key=C のダイアトニックコード：

番号:   I     ii    iii   IV    V     vi    vii°
コード: C     Dm    Em    F     G     Am    Bdim
読み方: いち  に    さん  よん  ご    ろく  なな

※ 大文字 = メジャー（明るい）
※ 小文字 = マイナー（暗い）
```

**なぜローマ数字で書くの？**
→ どのKeyでも同じ番号で呼べるから便利。
「I→V→vi→IV」って言えば、どのKeyでも通じる。

### 実際に弾いてみよう：I → V → vi → IV

**これ、世界で一番使われてるコード進行。**

- 「Let It Be」（The Beatles）
- 「No Woman No Cry」（Bob Marley）
- 「With or Without You」（U2）
- 「マリーゴールド」（あいみょん）

全部これ。

Key=C なら **C → G → Am → F**

```
C (I)           G (V)           Am (vi)         F (IV)
e|---0---       e|---3---       e|---0---       e|---1---
B|---1---       B|---0---       B|---1---       B|---1---
G|---0---       G|---0---       G|---2---       G|---2---
D|---2---       D|---0---       D|---2---       D|---3---
A|---3---       A|---2---       A|---0---       A|---3---
E|---x---       E|---3---       E|---x---       E|---1---

x = 弾かない（ミュートする）
```

**なぜこの進行が気持ちいいの？**

1. **I (C)** = 家。安心感。
2. **V (G)** = 「そろそろ家に帰りたい」って気持ち（ドミナント）
3. **vi (Am)** = ちょっと切ない感じ。でもCと仲良し（平行調）
4. **IV (F)** = 「次に進もう」って気持ち（サブドミナント）

この「家→外出→ちょい切ない→前進→家...」のループが心地いい。

> 「コード進行は物語なんだ。起承転結がある。」— Kai-sensei

## Lesson 2-2: 7thコードでオシャレに

### このレッスンのゴール
普通のコードを「7th」にして、ジャズ/Neo Soulっぽい響きを作れるようになる。

### なぜ7thを足すと「オシャレ」になるの？

普通のコードは3つの音でできてる。
7thコードは4つ目の音を足したもの。

**3音 → シンプル、はっきり**
**4音 → 複雑、おしゃれ、余韻がある**

料理で言うと：
- 3音 = 塩だけの味付け（シンプルで美味しい）
- 4音 = スパイスを足した味付け（深みが出る）

どっちが良いじゃなくて、場面で使い分ける。

### 7thって何番目の音？

```
ドレミファソラシド
1 2 3 4 5 6 7 8(=1)

7th = 「シ」の音をコードに足す

C + シ = Cmaj7（シーメジャーセブン）
```

### Cmaj7（シーメジャーセブン）

**普通のCとの違いは1箇所だけ！**

```
普通のC:                    Cmaj7:
e|---0--- (ミ)              e|---0--- (ミ)
B|---1--- (ド)  ←ここが      B|---0--- (シ) ←7thの音！
G|---0--- (ソ)              G|---0--- (ソ)
D|---2--- (ミ)              D|---2--- (ミ)
A|---3--- (ド=ルート)        A|---3--- (ド=ルート)
E|---x---                   E|---x---

やることは簡単：2弦を1fから開放に変えるだけ！

押さえ方:
- 5弦3f: 薬指
- 4弦2f: 中指
- 他は全部開放
```

**弾いてみて。** CとCmaj7を交互に。
Cmaj7の方が「ふわっ」とした、余韻のある響きでしょ？

### Am7（エーマイナーセブン）

```
普通のAm:                   Am7:
e|---0---                   e|---0---
B|---1---                   B|---1---
G|---2--- ←ここが           G|---0--- ←7thの音！
D|---2---                   D|---2---
A|---0--- (ラ=ルート)        A|---0--- (ラ=ルート)
E|---x---                   E|---x---

やることは簡単：3弦を2fから開放に変えるだけ！

押さえ方:
- 4弦2f: 中指
- 2弦1f: 人差し指
- 他は開放
```

### Dm7（ディーマイナーセブン）

```
Dm7:
e|---1---  ← 人差し指
B|---1---  ← 人差し指（1fと2fをセーハ）
G|---2---  ← 中指
D|---0---  (レ=ルート)
A|---x---
E|---x---

※セーハ = 1本の指で複数の弦を押さえること
人差し指を寝かせて1弦と2弦の1fを同時に押さえる
```

### G7（ジーセブン）

```
G7:
e|---1---  ← 人差し指（これが7thの「ファ」）
B|---0---
G|---0---
D|---0---
A|---2---  ← 中指
E|---3---  (ソ=ルート) ← 薬指

普通のGから、1弦を1fに変えるだけ！
```

### Fmaj7（エフメジャーセブン）

**Fのバレーコードが苦手な人に朗報！Fmaj7は簡単。**

```
Fmaj7:
e|---0---  ← 開放（これが7thの「ミ」）
B|---1---  ← 人差し指
G|---2---  ← 中指
D|---3---  ← 薬指
A|---x---  ← 弾かない（親指でミュート）
E|---x---  ← 弾かない

普通のFより押さえやすいのにオシャレ！
```

### 練習：さっきの進行を7thで弾き直す

**C → G → Am → F** を **Cmaj7 → G7 → Am7 → Fmaj7** で弾いてみて。

同じ進行なのに、雰囲気が全然違う。
これがハーモニーの面白さ。

> 「7thは『足す』んじゃなくて『変える』感覚。コードの色を変えてるんだ。」— Kai-sensei

## Lesson 2-3: ii-V-I を覚える（ジャズの基本）

### このレッスンのゴール
ジャズで最も重要な進行「ii-V-I」を弾けて、聴いて分かるようになる。

### ii-V-I って何？

**ジャズの80%はこの進行でできてる。**

```
Key=C の場合：

  ii      V       I
  Dm7  →  G7  →  Cmaj7
（レ）  （ソ）   （ド）

番号の意味:
ii = 2番目のコード（マイナー）
V  = 5番目のコード（セブンス）
I  = 1番目のコード（家）
```

### なぜ ii-V-I が気持ちいいの？

**これ、「家に帰る」感覚の最強版。**

1. **ii (Dm7)** = 「そろそろ帰ろうかな」（準備）
2. **V (G7)** = 「早く家に帰りたい！」（緊張がMAX）
3. **I (Cmaj7)** = 「ただいま〜」（解決、安心）

V → I だけでも気持ちいいんだけど、
その前に ii を置くと、「帰りたい感」が増幅される。

助走をつけてからジャンプする感じ。

### 弾いてみよう

```
Dm7             G7              Cmaj7
e|---1---       e|---1---       e|---0---
B|---1---       B|---0---       B|---0---
G|---2---       G|---0---       G|---0---
D|---0---       D|---0---       D|---2---
A|---x---       A|---2---       A|---3---
E|---x---       E|---3---       E|---x---

リズム:
- Dm7 を2拍
- G7 を2拍
- Cmaj7 を4拍（ここで解決するから長めに）

ゆっくり、コードチェンジを丁寧に。
```

**体で「緊張→解決」を感じて。**
G7からCmaj7に行った瞬間、「あ〜」って落ち着く感じ、分かる？

> 「ii-V-Iが分かると、ジャズの曲が『読める』ようになる。John Mayerの曲にもこっそり入ってるよ。」— Kai-sensei

## Lesson 2-4: 9thコードに挑戦（Neo Soulへの入口）

### このレッスンのゴール
9thコードを使って、Neo Soul/John Mayer的なキラキラした響きを作れるようになる。

### 9thって何？

7thの上にもう1つ音を足したもの。

```
ドレミファソラシド レ
1 2 3 4 5 6 7 8  9

9th = 「レ」の音

※ 2ndと同じ音だけど、オクターブ上にあるから「9th」と呼ぶ
```

### add9（アドナイン）って何？

「7thは入れないで、9thだけ足す」コード。
これがめちゃくちゃ使いやすくてオシャレ。

**add = 「足す」って意味**

Cadd9 = 「Cに9thを足したもの」

### Cadd9（シーアドナイン）

```
Cadd9:
e|---0---  ← 開放
B|---3---  ← 小指 ★これが9thの音（レ）！
G|---0---  ← 開放
D|---2---  ← 中指
A|---3---  ← 薬指（ド=ルート）
E|---x---

普通のCから、2弦を1fから3fに変えるだけ！

押さえ方:
- 5弦3f: 薬指
- 4弦2f: 中指
- 2弦3f: 小指
```

**弾いてみて。** キラキラしてない？
John Mayerがめちゃくちゃ使うコード。

### Gadd9（ジーアドナイン）

```
Gadd9:
e|---0---  ← 開放
B|---0---  ← 開放
G|---0---  ← 開放 ★これが9thの音（ラ）！
D|---0---  ← 開放
A|---2---  ← 中指
E|---3---  ← 薬指（ソ=ルート）

普通のGから、1弦と2弦を開放にするだけ！
むしろ普通のGより簡単。
```

### 練習：add9を使ったJohn Mayer風進行

**Cadd9 → Gadd9 → Am7 → Fmaj7**

この進行、「Waiting on the World to Change」っぽくない？

```
Cadd9           Gadd9           Am7             Fmaj7
e|---0---       e|---0---       e|---0---       e|---0---
B|---3---       B|---0---       B|---1---       B|---1---
G|---0---       G|---0---       G|---0---       G|---2---
D|---2---       D|---0---       D|---2---       D|---3---
A|---3---       A|---2---       A|---0---       A|---x---
E|---x---       E|---3---       E|---x---       E|---x---
```

> 「add9は『隠し味』。入れすぎると変だけど、ここぞで入れると光る。」— Kai-sensei

## John Mayer 分析コーナー

### 「Gravity」 Key=G

```
イントロ/ヴァース:
G → C

※ add9で弾くとよりJohn Mayer感が出る
  Gadd9 → Cadd9

サビ:
Am7 → D7 → Gmaj7 → Cmaj7

これ、ii-V-I の変形だって気づいた？
Am7 → D7 → Gmaj7 が「ii-V-I in Key of G」

Cmaj7 は IV で、ちょっと余韻を持たせてる。
```

### 「Waiting on the World to Change」 Key=G

```
ヴァース:
G → Gadd9 → C → Cadd9

同じコードをadd9と行ったり来たり。
これがあの「キラキラ感」の正体。
```

## 覚えておきたい用語

| 用語 | 意味 | 例 |
|------|------|-----|
| ダイアトニック | そのKeyの音だけで作ったコード | Key=CならC,Dm,Em,F,G,Am,Bdim |
| ルート | コードの「根っこ」になる音 | CmのルートはC（ド） |
| テンション | 7th以上の音（9th, 11th, 13th） | Cmaj9の「9」の部分 |
| 解決 | 緊張から安定に戻ること | G7→C は「解決」 |
| ドミナント | 「家に帰りたい」感を出すコード | Key=CならG7 |
