## はじめに

黒田さんがメジャースケールの「地図」を見せてくれて、瀬戸さんがその地図を使って「歩く練習」をしてくれたね。

今日は、その道を歩きながら **「何を見ているか」「何を感じているか」** を音にする日。

スケールは音の階段。でも、階段をただ上り下りするのと、階段の途中で立ち止まって景色を見るのとでは、全然違う体験になるよね。

今日やりたいのは、立ち止まること。急がないこと。1つ1つの音に耳を傾けること。

## 1. 同じスケールを「3つの気持ち」で弾く（10分）

### Lesson 4 の応用

Lesson 4 で、1つの音（C）だけで3つの感情を表現したのを覚えてる？
今日はそれをスケール全体に広げるよ。

### 練習 1-1: 「朝の散歩」

C メジャースケール（ポジション1）を上昇で弾く。
ただし、**朝の穏やかな散歩**をイメージして。

```
【TAB譜】ポジション1 上昇

e |-------------------------------|
B |-------------------------------|
G |--2--4--5----------------------|
D |--------2--3--5----------------|
A |--3--5--------------------------|
E |-------------------------------|
```

弾き方:
- **p（ピアノ = 弱め）** で弾く
- 各音の間に少し余裕を持たせる
- 急がない。朝の空気を吸いながら歩いてる感じ
- 最後の C（3弦5F）に着いたら、少し長めに伸ばす。「あ、気持ちいいな」

### 練習 1-2: 「坂道を駆け上がる」

同じスケールを、**坂道を駆け上がるような勢い**で弾く。

弾き方:
- **f（フォルテ = 強め）** で弾く
- テンポを速めに。加速する感じ
- 最後の音に向かってだんだん強くなる
- 頂上（高い C）に着いた瞬間、パッと止める

### 練習 1-3: 「探し物」

同じスケールを、**何かを探しているように**弾く。

弾き方:
- 音を1つ弾いて、少し止まる。「ここかな？」
- 次の音を弾いて、また止まる。「違うな…」
- 時々戻ったりする。C → D → E → D → E → F…
- 探してるものが見つかった音で、少し長めに伸ばす

**3つとも同じ C メジャースケールなのに、全然違う音楽に聴こえたでしょう？**

違いを作ったのは:
- テンポ（速さ）
- ダイナミクス（強弱）
- 間（音と音の間の時間）
- 意図（何をイメージして弾いているか）

> **海のメモ:** 「上手く弾こう」としなくていい。「何かを伝えよう」としてほしい。その「何か」が音に乗ったとき、聴いてる人の心に届く。技術はそのための道具でしかないんだよね。

## 2. 「間」の力 --- 沈黙は音楽の一部（15分）

### 音を弾かない勇気

音楽は「音」と「沈黙」の両方でできてる。
でも、ギターを始めたばかりの頃って、沈黙が怖くなることがある。
「何か弾いてなきゃ」って焦ってしまう。

実は、**沈黙の瞬間こそ、聴いてる人の想像力が動く時間**。

John Mayer のプレイを聴いてみて。彼はものすごく「弾かない」。
フレーズとフレーズの間に、たっぷり間を取る。
その間に聴いてる人は「次にどんな音が来るんだろう」って期待する。
そして次の音が来た瞬間、その音がグッと心に刺さる。

### 練習 2-1: 3音 + 沈黙

C メジャースケールから好きな音を3つだけ選んで、弾いてみて。
でも、**音と音の間に3秒以上の沈黙を入れる。**

```
例:

A |--3--------|（3秒待つ）--------5---------|（3秒待つ）---------|
D |-----------------------------------------2--------------------|

   C          ～～～沈黙～～～    D         ～～～沈黙～～～    E

   1音目を弾く → 音が消えるのを聴く → 次の音を弾く → ...
```

ルール:
- 3音だけ。それ以上弾かない
- 沈黙の間、弾いた音の余韻を最後まで聴く
- 次の音を弾く前に、「次はどんな気持ちで弾こう」と一瞬考える

### 練習 2-2: 同じ3音を、間を変えて弾く

同じ C, D, E の3音を使って:

**パターン A: 間を短く**
```
C → (1秒) → D → (1秒) → E
テンポよく、語りかけるように
```

**パターン B: 間を長く**
```
C → (5秒) → D → (5秒) → E
1音ずつ、大切に。手紙を書くように
```

**パターン C: 間をバラバラに**
```
C → (1秒) → D → (6秒) → E
2音目までは普通に。3音目の前に長い沈黙。
「言いたいことがあるのに、なかなか言えない」感じ
```

間の長さを変えるだけで、同じ3つの音がまったく違う物語になる。

> **海のメモ:** Miles Davis（ジャズの伝説的なトランペット奏者）の有名な言葉がある。「大事なのは弾いた音じゃない。弾かなかった音だ。」これは本当にそう。音楽の「余白」にこそ、深い表現がある。

## 3. 「重要な音」と「経過音」（10分）

### すべての音が同じ重さじゃない

C メジャースケールには7つの音がある: C, D, E, F, G, A, B。
でも、この7つの音が全部同じ重要度ってわけじゃない。

**重要な音（着地する音）:**
- **C（1番目）**: ホーム。一番安定する。ここに戻ると「解決した」感じ。
- **E（3番目）**: メジャーの「明るさ」を決める音。ここに着地すると明るい。
- **G（5番目）**: 安定感がある。パワーコードの響き。

**経過音（通り過ぎる音）:**
- **D, F, A, B**: もちろん使うけど、ここに長く止まると「まだ旅の途中」感が出る。

### 練習 3-1: 重要な音に「着地」する

```
【TAB譜】

フレーズ A: C に着地
e |-------------------------------|
B |-------------------------------|
G |-------------------------------|
D |--------2--3--2----------------|
A |--3--5-----------3-------------|
E |-------------------------------|
   C  D  E  F  E   C（ここに着地）
   ↑ 経過音たち → ★ ホームに帰ってきた

フレーズ B: E に着地
e |-------------------------------|
B |-------------------------------|
G |-------------------------------|
D |--5--3--2----------------------|
A |-------------------------------|
E |-------------------------------|
   G  F  E（ここに着地）
         ★ 明るい感じで終わる

フレーズ C: G に着地
e |-------------------------------|
B |-------------------------------|
G |-------------------------------|
D |--2--3--5----------------------|
A |--3-----------------------------|
E |-------------------------------|
   C  E  F  G（ここに着地）
            ★ 力強く安定した終わり方
```

### 感じてみて

3つのフレーズ、着地する音が違うだけで印象が変わるでしょう？

- **C で終わる**: 「ただいま」感。完結する。
- **E で終わる**: 「明るい気分」。前向き。
- **G で終わる**: 「どっしり」。自信がある感じ。

フレーズを弾くとき、**最後の音を意識する**だけで、表現がグッと変わる。

> **海のメモ:** 経過音は「旅の景色」、重要な音は「目的地」。目的地がはっきりしてる旅は、途中の景色もより楽しめる。フレーズも同じで、「どこに着地するか」を決めてから弾くと、途中の音も意味を持ち始めるんだよね。

## 4. Gravity のメロディを「自分の表現」で弾く（15分）

### Lesson 6 の発展

瀬戸さんと Gravity のサビメロディを弾いたよね。
今日は同じメロディを「自分ならどう弾くか」を考えてみよう。

### サビメロディ（簡略版）

```
【TAB譜】

"Gravity is working against me"

e |-------------------------------|
B |-------------------------------|
G |--4--4--5--4--2----------------|
D |------------------5-----------|
A |-------------------------------|
E |-------------------------------|

音名: G  G  A  G  (E)  G
```

### 練習 4-1: 3つの解釈で弾く

**解釈 A: 「諦め」**
- pp〜p で、力なく弾く
- テンポをゆっくりに
- 音が消えていくのをじっと聴く
- 重力に逆らう気力がない感じ

**解釈 B: 「怒り」**
- f〜ff で、力強く弾く
- 特に "against me" の部分（後半の音）を強く
- 重力に対して「なんでだよ」と言ってる感じ

**解釈 C: 「受け入れ」**
- mf で、穏やかだけど芯のある弾き方
- 急がず、1音1音を大切に
- 重力があることを受け入れている。「それでも生きていく」

### 練習 4-2: 自分の解釈を見つける

A, B, C の中で一番しっくりきたのはどれ？
もしかしたら、どれでもない「自分だけの解釈」があるかもしれない。

それを見つけて、5回繰り返し弾いてみよう。

**同じメロディを同じ解釈で弾くのに、5回とも少しずつ違う弾き方になるはず。**
それが自然なこと。音楽は生き物だから。

> **海のメモ:** John Mayer がライブで Gravity を弾くときは、毎回表情が違う。あるときは切なく、あるときは力強く、あるときは達観したように。曲は同じでも、その日の彼が違うから。自分の演奏も「今日の自分」を反映していい。正解は毎日変わるんだよ。

## 5. 3音の即興 --- 少ない音で語る（10分）

### 最後のチャレンジ

C メジャースケールから **3つの音だけ** 選んで、1分間の即興をしてみよう。

### おすすめの3音の組み合わせ

**組み合わせ 1: C, E, G（明るい）**
```
ポジション:
A |--3---|  ← C
D |--2---|  ← E
D |--5---|  ← G
```
Cメジャーコードの構成音。明るく安定した響き。

**組み合わせ 2: E, G, A（切ない）**
```
ポジション:
D |--2---|  ← E
D |--5---|  ← G
G |--2---|  ← A
```
Am コードの構成音に近い。ちょっと物悲しい。

**組み合わせ 3: C, D, G（開放的）**
```
ポジション:
A |--3---|  ← C
A |--5---|  ← D
D |--5---|  ← G
```
5度の響きが入って、広がりがある。

### やり方

1. 3つのうち1つの組み合わせを選ぶ
2. タイマーを1分にセット
3. その3音だけで自由に弾く

**意識すること:**
- 間を大事に。3音しかないから、沈黙が大きな役割を果たす
- 同じ音を繰り返してもいい
- 強弱をつける
- 「何かを伝えようとしている」気持ちで弾く

**1分は短いようで長い。** でも、3つの音と沈黙があれば、驚くほど豊かな音楽が生まれる。

> **海のメモ:** 制限があると、人は創造的になる。100色のクレヨンより、3色のクレヨンの方が、かえって印象的な絵が描けることがある。音楽も同じ。少ない音で「何を伝えられるか」を考えることが、表現力を育てる一番の近道。

## まとめ

今日やったことを振り返ろう。

1. **同じスケールを3つの気持ちで弾いた** --- 「朝の散歩」「駆け上がる」「探し物」。同じ音でも意図で変わる。
2. **「間」の力を体感した** --- 沈黙は音楽の一部。弾かないことも表現。
3. **「重要な音」と「経過音」を感じた** --- 着地する音で印象が決まる。C, E, G は重要な着地点。
4. **Gravity を自分の解釈で弾いた** --- 「諦め」「怒り」「受け入れ」。同じメロディでも表現で全く変わる。
5. **3音の即興** --- 少ない音で語る。制限が創造性を引き出す。

### 一番伝えたいこと

スケールは「音の階段」だけど、それを上り下りするだけなら「運動」で終わってしまう。

**階段の途中で立ち止まって、何を見て、何を感じて、何を伝えたいか。**

それを考えた瞬間、スケールは「音楽」になる。

これで Phase 1 Topic 2（メジャースケール）の3レッスンが完了。
理論（黒田）→ 実践（瀬戸）→ 表現（海）のサイクルを一周したよ。

次は新しいトピック、**ペンタトニックスケール**に入っていく。
5つの音だけでできたスケールなのに、ブルースもロックもジャズもこのスケールなしには語れない。
John Mayer のソロの核心部分。楽しみにしていてね。

### 練習のポイント

- **上手く弾こうとしない。** 今週は「感じること」「伝えること」が目標。
- **沈黙を怖がらない。** 弾かない時間は「失敗」じゃなく「表現」。
- **自分の感覚を信じる。** 「この弾き方が好き」と思えたら、それが正解。
- **できれば録音してみて。** スマホでOK。後で聴き返すと、自分の表現を客観的に聴ける。意外と良いところが見つかるよ。
- **Gravity を聴く時間を作る。** 歌詞の意味を考えながら聴いてみて。John Mayer が何を伝えたかったのか、感じてみて。
