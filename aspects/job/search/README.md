# 就職活動サポート

あなた専属の就職活動サポートチームです。各メンバーが専門分野を持ち、次のキャリアを掴むお手伝いをします。

## 使い方

`/job-search-ask` コマンドでチームに相談できます。

## フォルダ構成

```
search/
├── team/                 # サポートチーム
│   ├── CLAUDE.md         # チーム設定
│   └── roles/            # 各メンバーの詳細
├── resume/               # 履歴書・職務経歴書
├── cover-letters/        # 志望動機・自己PR
├── applications/         # 応募状況の管理
├── interviews/           # 面接準備メモ・振り返り
├── networking/           # 人脈・コンタクト管理
└── offers/               # 内定詳細・交渉メモ
```

## チームメンバー

| 役割 | 専門分野 | こんな時に相談 |
|------|----------|----------------|
| **履歴書スペシャリスト** | 履歴書・職務経歴書の最適化、実績の書き方 | 書類を更新・改善したい時 |
| **志望動機ライター** | 志望動機、自己PR、企業研究の反映 | 特定の企業に応募する時 |
| **面接コーチ** | 模擬面接、よくある質問対策、技術面接対策 | 面接前の準備 |
| **求人リサーチャー** | 求人市場調査、企業マッチング、企業情報 | 求人を探している時 |
| **ネットワーキングアドバイザー** | LinkedIn最適化、人脈作り、紹介依頼 | コネクションを広げたい時 |
| **年収交渉アドバイザー** | 年収相場調査、オファー評価、交渉戦略 | 内定・条件交渉の時 |

詳細は `team/roles/` 内の各ファイルを参照してください。
