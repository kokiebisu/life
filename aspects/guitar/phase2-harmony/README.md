# Phase 2: Harmony（ハーモニー）

コードの世界を広げるフェーズ。なぜこのコード進行が気持ちいいのか、理論で理解する。

## Goals

このフェーズを終えると:
- ダイアトニックコードを理解し、キーを分析できる
- よくあるコード進行のパターンが分かる
- テンションコードを使える
- John Mayerの曲を分析できる

## Topics

### 1. ダイアトニックコード
- Keyとは何か
- メジャーキーの7つのダイアトニックコード
- コードの機能（Tonic, Subdominant, Dominant）
- ローマ数字表記（I, ii, iii, IV, V, vi, vii°）

### 2. コード進行分析
- I-IV-V-I（最も基本的な進行）
- I-V-vi-IV（ポップスの王道）
- ii-V-I（ジャズの基本）
- vi-IV-I-V（エモーショナルな進行）

### 3. テンションコード入門
- なぜテンションを加えるのか
- Major 7th (Cmaj7)
- Minor 7th (Am7)
- Dominant 9th (G9)
- Add9 コード

### 4. コードトーンとアルペジオ
- コードトーンとは（1, 3, 5, 7）
- アルペジオの練習
- コードトーンを意識したフレージング

## Study Songs

### John Mayer
- 「Neon」- 複雑なコードヴォイシング
- 「Slow Dancing in a Burning Room」- テンションの使い方
- 「Who Says」- ダイアトニックの美しさ

### King Gnu
- 「白日」- 転調とテンションの融合
- 「Prayer X」- ドラマチックなコード展開
