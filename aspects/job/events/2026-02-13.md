# 最終出社日（2026-02-13）- freee退職チェックリスト

## 2/11（水）- 事務手続き・確認を片付ける

万が一トラブルがあっても翌日リカバリーできるよう、書類・アカウント系を先にやる。

### 書類・フォーム

- [ ] 退職手続きフォームの回答（Labor Managementからの自動送信メール内のフォーム）
  - 住民税の徴収方法、離職票の発行要否などを記入
- [ ] 退職届・誓約書の署名（freeeサインで届く署名依頼メール）
  - 「フリー株式会社」ではなく自分の名前で署名する
- [ ] 経費精算 - 未申請の経費があれば全て申請完了させる

### freee人事労務のログイン確認（超重要）

- [ ] **個人端末（スマホ or 私物PC）** からfreee人事労務にログインできるか確認
  - ID: 在籍中のメールアドレス
  - PW: 自分で設定したもの
- [ ] 二要素認証を設定している場合 → **退職前に解除する**（退職後ログイン不可になる）
- 退職後も給与明細・源泉徴収票の取得に必要

### 会社への確認・依頼

- [ ] 離職票の発行を依頼する（失業保険申請に必要）
- [ ] 健康保険資格喪失証明書の発行を依頼する（国保加入に必要）
- [ ] 源泉徴収票の発行を依頼する（確定申告に必要）
- [ ] 退職証明書の発行を依頼する（離職票到着前の国保加入等に使える）
- [ ] 雇用保険被保険者証を受け取る（会社保管の場合）
- [ ] 源泉徴収票は新住所に送付してほしいと人事・総務に伝える
- [ ] 退職後の連絡先・住所を人事に伝える（書類郵送先）
- [ ] 年金手帳が会社預かりか確認（預けていれば返却してもらう）
- [ ] 退職金の有無・支給時期を確認
- [ ] 最終給与の支払日・未消化有給の扱いを確認
- [ ] 通勤定期券の精算方法を確認

### その他確認

- [ ] 持株会・SO・RS保有者 → 「株式関連のご案内」資料を確認し手続き
- [ ] 住民税: 1月〜5月退職のため、5月分までの住民税が最終給与から一括徴収される → 手取り額を認識しておく
- [ ] Googleカレンダーで退職時面談（Member Developmentチーム）の予定を確認

## 2/12（木）- データ退避・返却準備

アカウント停止後はアクセス不可。この日中にデータと物の準備を終わらせる。

### 私物データの退避（重要）

- [ ] Googleドライブから持ち出し可能なデータを退避
- [ ] Slackから必要な連絡先・情報を控える
- [ ] その他業務ツールから必要なデータを退避
- 最終出社日の夜（アカウント停止処理後）から一切アクセス不可

### PC返却前の作業（Mac）

- [ ] Apple IDからサインアウトする
- [ ] 「Macを探す」をオフにする（必須。放置すると返却後に問題になる）
- [ ] PCに貼った私物シールを綺麗に剥がす
- [ ] 会社PC内の個人データを削除する
- [ ] 必要なローカルデータをチームに共有（引き継ぎ）
- 初期化は不要

### スマホ関連

- [ ] 社用スマホがある場合 → パスコードを解除、または「000000」に設定
- [ ] Workhubアプリを自分のスマホから削除（オフィス入退室用）

### 貸与品をまとめる

- [ ] PC本体
- [ ] 充電器
- [ ] 社用スマホ（ある場合）
- [ ] モニター
- [ ] 変換コネクタ等の周辺機器
- [ ] 名刺（自分の名刺の残り）
- [ ] その他貸与物
