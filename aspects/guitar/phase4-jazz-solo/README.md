# Phase 4: ジャズソロ＆アドリブ

## ゴール

- コード進行を追いかけながらソロを弾く力をつける
- ジャズの定番リック・語彙を実戦で使えるようにする
- コードバッキングとソロをシームレスに切り替えられるようにする

## このフェーズを終えると

- コードトーンを軸にしたソロが即興で弾ける
- ジャズの定番リックを場面に応じて引き出せる
- コンピング（伴奏）からソロへ自然に移行できる
- 1コーラス分のソロを起承転結をつけて構築できる
