# Phase 3: ブルースの土台

## ゴール

- シャッフルリズムと12小節ブルース進行を弾けるようにする
- ブルースで使うコードフォームとターンアラウンドを習得する
- ペンタトニック・ボックス1で最初のブルースリックを弾く
- ベンディング＆ビブラートで感情表現を加える
- 各コードのヴォイシングを Phase 2 で学んだフォーミュラで理解する

## このフェーズを終えると

- シャッフルのリズムでブルースをストロークできる
- E7-A7-B7 の12小節ブルース進行を通して弾ける
- ペンタトニック・ボックス1でリックを6本弾ける
- ベンド＋ビブラートの「泣きフレーズ」を弾ける
- 「なぜブルースは7thコードを使うのか」を説明できる
