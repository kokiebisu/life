# Phase 5: ブルース・スタイル攻略

## ゴール

- John Mayer のモダンブルーススタイルを弾けるようにする
- デルタブルースのフィンガーピッキングスタイルを体験する
- ファンキーブルース＆ジャズブルースで応用の幅を広げる
- 3つのスタイルでフルソロを完成させる

## このフェーズを終えると

- Mayer 風のメジャーペンタ混合フレーズが弾ける
- アコギでデルタブルースのフィンガーピッキングパターンが弾ける
- ファンキーブルース・マイナーブルースの引き出しがある
- 3曲分のフルソロを通して弾ける＋自分のオリジナルソロを組み立てられる
