## Lesson 3-1: ペンタトニックの5ポジションをつなげる

### 今弾けてるポジション（おそらくこれ）

**Aマイナーペンタトニック - ポジション1（5f付近）**

```
e|--------5---8---
B|--------5---8---
G|--------5---7---
D|--------5---7---
A|--------5---7---
E|--------5---8---
     ↑ 5f = ルート(A)

指使い: 人差し指=5f、薬指=7f、小指=8f
```

これ、5fに人差し指を置いたまま弾くから「ボックス」って呼ばれてる。
でもここだけで弾いてると、フレーズが単調になりがち。

### ポジション2（7f付近）

```
e|--------8--10---
B|--------8--10---
G|--------7---9---
D|--------7---9---
A|--------7--10---
E|--------8--10---
```

### ポジション1と2をつなげてみる

```
e|--------5---8--------8--10---
B|--------5---8--------8--10---
G|--------5---7--------7---9---
D|--------5---7--------7---9---
A|--------5---7--------7--10---
E|--------5---8--------8--10---
    [Position 1]    [Position 2]

接続ポイント: 1弦8f、2弦8f が両方のポジションに存在
ここを「橋」として使う
```

### 練習：上昇して降りてくる

```
TAB:
e|-----------------------------------------------10---8---
B|----------------------------------8--10---10-8----------
G|---------------------5---7---7-9------------------------
D|-----------5---7---7------------------------------------
A|---5---7------------------------------------------------
E|---------------------------------------------------------

5fから始めて、徐々に高い方へ。そして降りてくる。
これだけでソロっぽくなる！
```

> 「同じ音でも、違うポジションで弾くと響きが変わる。太い弦は太い音、細い弦は細い音。これを意識して。」— Kai-sensei

## Lesson 3-2: ブルーノートを加える

### ブルーノートって何？

ペンタトニックに「1音」加えるだけで、一気にブルージーになる。
その1音が **♭5（フラットファイブ）**。

### Aマイナーペンタトニック + ブルーノート

```
通常のペンタ: A - C - D - E - G - A
ブルース:     A - C - D - D#- E - G - A
                      ↑ ここがブルーノート

ポジション1にブルーノートを追加:

e|--------5---8------
B|--------5---8------
G|--------5---6---7--  ← 6fがブルーノート!
D|--------5---7------
A|--------5---6---7--  ← 6fがブルーノート!
E|--------5---8------
```

### ブルースリック（定番フレーズ）

```
John Mayerがよく使うやつ:

e|---------------------------
B|---------------------------
G|---5---6---7---5-----------
D|------------------7---5----
A|------------------------7--
E|---------------------------

3弦 5f→6f→7f のところがブルーノート経過
このチョーキングみたいな「うねり」がブルースの味
```

> 「ブルーノートは『経過音』として使う。そこに留まらない。通り過ぎる感じ。」— Kai-sensei

## Lesson 3-3: Dorian モードを知る

### Dorianって何がおいしいの？

マイナーペンタにない音を足すと、Neo Soulっぽくなる。
その代表が **Dorian（ドリアン）モード**。

```
Aマイナーペンタ:  A - C - D - E - G
A Dorian:         A - B - C - D - E - F# - G
                      ↑         ↑
                  2度追加    6度が#（ここが特徴）
```

### A Dorian - ポジション1

```
e|--------5---7------
B|--------5---7------
G|--------4---5---7--  ← 4fがF#（Dorianの特徴音）
D|--------5---7------
A|--------5---7------
E|--------5---7------

ペンタ+4f(3弦)で弾ける！
```

### ペンタ → Dorian への変化

```
まずペンタで弾く:
e|--------5------------------
B|-------------5-------------
G|-----------------5---7-----
D|-------------------------7-
A|---------------------------
E|---------------------------

Dorianの色を足す:
e|--------5------------------
B|-------------5-------------
G|-----------------4---5---7-  ← 4fを経過
D|-------------------------7-
A|---------------------------
E|---------------------------

この4f(F#)がNeo Soul感を出す！
```

> 「Dorianはマイナーなのに明るい。それが不思議で、クセになる響き。」— Kai-sensei

## Lesson 3-4: コードに合わせてスケールを選ぶ

### 基本ルール

```
コード        →  スケール
Am / Am7      →  Aマイナーペンタ or A Dorian
Dm / Dm7      →  Dマイナーペンタ or D Dorian
G / G7        →  Gメジャーペンタ or G Mixolydian
C / Cmaj7     →  Cメジャーペンタ
```

### 実践：ii-V-I でスケールを切り替える

```
進行: Dm7 → G7 → Cmaj7

Dm7 の上:
5弦5f(A)をルートにAマイナーペンタ...じゃなくて
5弦5f(D)をルートにDマイナーペンタ！

e|--------10--13---
B|--------10--13---
G|--------10--12---
D|--------10--12---  ← 10f = D(ルート)
A|--------10--12---
E|------------------

G7 の上:
Gミクソリディアン（今は難しければGメジャーペンタでOK）

Cmaj7 の上:
Cメジャーペンタ
e|--------8--10---
B|--------8--10---
G|--------7---9---
D|--------7--10---
A|--------8--10---  ← 8f = C(ルート)
E|------------------
```
