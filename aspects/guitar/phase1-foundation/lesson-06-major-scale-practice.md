## 1. ウォームアップ: ポジション1の上昇・下降を滑らかに（10分）

### まずは復習

Lesson 5 で覚えたポジション1（5弦ルート）を思い出そう。

```
【TAB譜】C メジャースケール ポジション1（上昇 → 下降）

上昇:
e |-------------------------------|
B |-------------------------------|
G |--2--4--5----------------------|
D |--------2--3--5----------------|
A |--3--5--------------------------|
E |-------------------------------|

下降:
e |-------------------------------|
B |-------------------------------|
G |--5--4--2----------------------|
D |-----------5--3--2-------------|
A |--------------------5--3-------|
E |-------------------------------|

音名:
上昇: C → D → E → F → G → A → B → C
下降: C → B → A → G → F → E → D → C
```

### ドリル 1-1: まず5往復（5分）

上昇→下降を1往復として、5回繰り返す。

**ルール:**
- 最初は超ゆっくりでOK（1音に2秒くらい）
- 音名を声に出しながら弾く（Lesson 5 でやったよね）
- 全部の音がビリビリせずに鳴っていることを確認
- 弦を移るところ（5弦→4弦、4弦→3弦）で止まらないように意識

### ドリル 1-2: メトロノームに合わせる（5分）

5往復できたら、メトロノームを使ってみよう。

```
BPM 60 から始める（1秒に1音）

カチ → C
カチ → D
カチ → E
カチ → F
...

楽に弾けたら BPM 70 → 80 と上げていく
今日は BPM 80 まで行けたら十分
```

> **瀬戸メモ:** テンポは無理に上げなくていい。「全部の音がきれいに鳴るテンポ」が今の自分の限界。それを少しずつ広げていくのが練習ってもんだ。

## 2. 3度飛ばし --- スケールをメロディっぽく弾く（15分）

### 「隣の音」ばかりだとつまらない

スケールを上から下に弾くだけだと、練習っぽくてメロディには聴こえない。
実際の曲のメロディは、隣の音だけじゃなくて「飛ぶ」動きが入ってる。

その中で一番よく使われるのが **3度飛ばし**。
スケールの音を1つ飛ばしで弾く方法だ。

### 3度飛ばしって何？

```
普通のスケール（順番に）:
C → D → E → F → G → A → B → C

3度飛ばし（1つ飛ばし）:
C → E, D → F, E → G, F → A, G → B, A → C

つまり:
1→3, 2→4, 3→5, 4→6, 5→7, 6→8
```

「3度」というのは Lesson 2 で学んだインターバル。スケールの中で1つ飛ばしにすると、自然と3度の間隔になる。

### ドリル 2-1: 3度飛ばし上昇（7分）

```
【TAB譜】C メジャースケール 3度飛ばし（上昇）

e |-------------------------------|
B |-------------------------------|
G |------2--4--4--5--5------------|
D |--2--3--3--5--5----------------|
A |--3--5--------------------------|
E |-------------------------------|

音名のペア:
C-E, D-F, E-G, F-A, G-B, A-C
```

**弾き方のコツ:**
- 2音をペアにして弾く。「C、E」「D、F」「E、G」...
- ペアの中は続けて弾いて、ペアの間で少し間を取ると分かりやすい
- これだけで「あ、メロディっぽい！」って感じるはず

### ドリル 2-2: 3度飛ばし下降（5分）

```
【TAB譜】C メジャースケール 3度飛ばし（下降）

e |-------------------------------|
B |-------------------------------|
G |--5--4--4--2--2----------------|
D |------5--3--3--2----------------|
A |----------------5--3------------|
E |-------------------------------|

音名のペア:
C-A, B-G, A-F, G-E, F-D, E-C
```

### なぜ3度飛ばしが大事なのか

実はコードの中身は「3度の積み重ね」（Lesson 2 で黒田が教えてくれた通り）。
3度飛ばしでスケールを弾くことは、**コードの音を意識しながらメロディを弾く練習**になる。

John Mayer のフレーズにも、3度飛ばしの動きがたくさん隠れてる。

> **瀬戸メモ:** 3度飛ばし、最初は指がもつれるかもしれない。でもこれができると「スケールを上下するだけの人」から「メロディを弾ける人」に一歩進む。地味だけど超重要な練習。

## 3. スケールを使ったシンプルなフレーズ3つ（15分）

### スケールの音でフレーズを作る

ここからが楽しいところ。スケールの音を使って、**実際に音楽として使えるフレーズ**を3つ覚えよう。

### フレーズ 1: 「上がって戻る」

メロディの基本パターン。上がったら少し戻る。話し方でいうと「ちょっと言いかけて、やっぱりやめる」みたいな動き。

```
【TAB譜】フレーズ1

e |-------------------------------|
B |-------------------------------|
G |-------------------------------|
D |--------2--3--5--3--2----------|
A |--3--5--------------------------|
E |-------------------------------|

音名: C → D → E → F → G → F → E

リズム: ♩ ♩ ♩ ♩ ♩ ♩ 𝅗𝅥
（最後の E を長めに伸ばす）
```

### フレーズ 2: 「飛んで降りる」

高い音にジャンプしてから、階段で降りてくる。インパクトがあるフレーズ。

```
【TAB譜】フレーズ2

e |-------------------------------|
B |-------------------------------|
G |--5--4--2----------------------|
D |-----------3--2----------------|
A |--3-----------------------------|
E |-------------------------------|

音名: C → (ジャンプ) → A → G → F → E

低い C からいきなり高い A に飛ぶ。
そこから F → E と降りてくる。
```

### フレーズ 3: 「Gravity 風 下降フレーズ」

Gravity のメロディに似た、ゆったり下降するフレーズ。

```
【TAB譜】フレーズ3

e |-------------------------------|
B |-------------------------------|
G |--5-----4-----2---------------|
D |------------------5-----3-----|
A |-------------------------------|
E |-------------------------------|

音名: A → G → ... → E（ゆっくり降りていく）

各音の間に少し間を置く。急がない。
Gravity のサビメロディ（G G A G E D C）の雰囲気。
```

### 3つのフレーズを続けて弾く

1 → 2 → 3 と続けて弾いてみよう。
それだけで「あ、なんかソロっぽい」って感じるはず。

> **瀬戸メモ:** プロのソロも、分解すると「上がって戻る」「飛んで降りる」「ゆっくり降りる」みたいなシンプルなパターンの組み合わせ。難しいことをやってるように聴こえるけど、基本動作は今日覚える3つと同じ。

## 4. バッキングトラック --- スケールで弾いてみる（15分）

### Lesson 3 の進化版

Lesson 3 ではルートと5度だけでバッキングトラックに合わせたよね。
今日はそこから大幅にレベルアップ。**スケールの音を全部使っていい。**

### 準備

YouTube で「C major backing track slow」を検索して流そう。BPM 70〜80 くらいがベスト。

### ドリル 4-1: スケール上下だけ（5分）

まずはシンプルに。バッキングトラックに合わせて、スケールを上がって降りるだけ。

```
バッキングトラックが流れてる中で:
上昇: C → D → E → F → G → A → B → C
下降: C → B → A → G → F → E → D → C
繰り返し...
```

「練習」に聴こえるかもしれないけど、バッキングトラックに合ってるだけで気持ちいいはず。

### ドリル 4-2: フレーズを混ぜる（5分）

さっき覚えた3つのフレーズをバッキングトラックの中で弾いてみよう。

```
バッキングトラック流しながら:
→ フレーズ1（上がって戻る）
→ 少し休む（2拍くらい）
→ フレーズ2（飛んで降りる）
→ 少し休む
→ フレーズ3（Gravity 風下降）
→ 休む
→ 繰り返し
```

**「休む」が超大事。** 弾きっぱなしにしない。音と沈黙の両方があって音楽になる。

### ドリル 4-3: 自由に弾く（5分）

最後は自由。スケールの音なら何を弾いてもOK。

**コツ:**
- 迷ったらフレーズ1〜3のどれかを弾く
- 詰まったらルートの C（5弦3F）に戻ればOK
- たくさん弾くより、少ない音で「間」を大事にする方がカッコいい
- コードが変わったら、そのコードのルート音を弾くと馴染む

> **瀬戸メモ:** 最初はぎこちなくて当然。でも「バッキングトラックに合わせてスケールの音を自由に弾いてる」--- これ、実はもうアドリブ（即興演奏）の入り口なんだよ。すごいことやってるんだぜ。

## 5. Gravity のメロディをスケールの視点で弾く（10分）

### サビのメロディ

Lesson 5 で黒田が分析してくれた通り、Gravity のメロディは C メジャースケールの音だけでできてる。

サビのメロディを実際に弾いてみよう。

```
【TAB譜】Gravity サビメロディ（簡略版）

"Gravity is working against me"

e |-------------------------------|
B |-------------------------------|
G |--4--4--5--4--2----------------|
D |------------------5-----------|
A |-------------------------------|
E |-------------------------------|

音名: G  G  A  G  (E を3弦で)  G(4弦)
```

※ これは完全な耳コピではなく、雰囲気をつかむための簡略版。実際のメロディはもう少し複雑だけど、核になる動きはこんな感じ。

### スケールとの関係を確認

```
Gravity サビメロディで使われてる音:
G, A, E ... すべて C メジャースケールの音

ポジション1の中で見ると:
     G = 4弦5F or 3弦開放
     A = 3弦2F
     E = 4弦2F or 3弦開放近辺
```

### 練習: メロディを弾いてからスケールを弾く

```
1. Gravity のメロディフレーズを弾く
2. そのまま C メジャースケールを上昇
3. また Gravity のメロディに戻る

これを繰り返すことで「メロディはスケールの一部」という感覚が体に入る
```

> **瀬戸メモ:** 好きな曲のメロディがスケールの音でできてると分かった瞬間、スケール練習のモチベーションが全然変わる。「Gravity を弾くためにスケールを練習してる」って思えたら最高だろ？

## まとめ

今日やったことを振り返ろう。

1. **ポジション1の上昇・下降を滑らかに** --- メトロノームに合わせてスムーズに弾く練習をした。
2. **3度飛ばし** --- 1つ飛ばしで弾くとメロディっぽくなる。コードの音を意識した弾き方。
3. **シンプルなフレーズ3つ** --- 「上がって戻る」「飛んで降りる」「Gravity 風下降」。
4. **バッキングトラック** --- スケールの音を全部使って、即興の入り口を体験した。
5. **Gravity のメロディ** --- サビのメロディをスケールの視点で弾いて、「メロディ = スケールの一部」を実感した。

### Lesson 5（黒田/理論）→ Lesson 6（瀬戸/実践）

| Lesson 5（黒田/理論） | Lesson 6（瀬戸/実践） |
|:---|:---|
| メジャースケール = ドレミファソラシド | ポジション1を滑らかに弾く |
| 全全半全全全半 の構造 | メトロノームに合わせる |
| コードはスケールから生まれる | 3度飛ばしでコードの音を体感 |
| Gravity は C メジャースケールの音 | Gravity のメロディを実際に弾く |

### 次回予告

次は橋本と一緒に、スケールに「表現」を乗せていく。「正しく弾く」じゃなく「伝わるように弾く」--- Lesson 4 でやったダイナミクスの考え方を、スケール演奏にも応用するよ。

## 練習メニュー（1週間）

毎日15〜20分でOK。バッキングトラック練習は後半に多めに入れてある。

| 日 | メニュー | 時間 | 内容 |
|:---:|:---|:---:|:---|
| Day 1 | ポジション1の上昇・下降 | 15分 | 5往復。声を出しながら（Lesson 5 の復習）。メトロノーム BPM 60 で。 |
| Day 2 | メトロノーム速度アップ | 15分 | BPM 60 → 70 → 80 と上げていく。全音がきれいに鳴るテンポを見つける。 |
| Day 3 | 3度飛ばし上昇 | 15分 | ドリル 2-1 をゆっくり繰り返す。ペアで弾く感覚をつかむ。下降もチャレンジ。 |
| Day 4 | フレーズ1 & 2 | 15分 | 「上がって戻る」と「飛んで降りる」を覚える。それぞれ10回ずつ弾く。 |
| Day 5 | フレーズ3 + 全フレーズ通し | 15分 | 「Gravity 風下降」を覚える。3つのフレーズを続けて弾く。 |
| Day 6 | バッキングトラック | 20分 | ドリル 4-1 → 4-2 → 4-3 の順に。自由に弾く時間を多めにとる。 |
| Day 7 | Gravity メロディ + 復習 | 20分 | Gravity のサビメロディを弾く → スケールを弾く → メロディに戻る。残り時間でバッキングトラック自由練習。 |

### 練習のポイント

- **メトロノームは友達。** 退屈に感じるかもしれないけど、リズムが安定すると全てが変わる。
- **3度飛ばしは最初は難しい。** 指がもつれても焦らない。1週間やればかなりスムーズになる。
- **フレーズは「覚える」→「バッキングトラックで使う」** が流れ。覚えただけで終わらせない。
- **バッキングトラック練習では「休む」を意識。** 弾きすぎない。間が音楽を作る。
- **Gravity を聴く時間も練習のうち。** メロディの動きを耳で追いかけてみて。
