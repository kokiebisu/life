## はじめに

> 瀬戸: 今日はいつもと少し違う回。L3 で7thコードを7個覚えたよね。Cmaj7、Am7、G7、E7…全部「この形を押さえて」って教えたけど、「なんでこの形なの？」って疑問が出てこなかった？ 今日はその「なぜ」を解き明かす。黒田にも一緒に入ってもらうよ。

## ① コードを分解する

### STEP 1: まず Cmaj7 の正体を暴く

L3 で覚えた Cmaj7 を、もう一度押さえてみて。

```
Cmaj7:
e --0--
B --0--
G --0--
D --2--
A --3--
E --x--
```

> 瀬戸: これをジャーンと弾くと、フワッとした響き。でも今日は1弦ずつバラバラに弾いてみて。5弦から順に、ポーン・ポーン・ポーン・ポーン・ポーンって。

**やること:** Cmaj7 を押さえて、5弦から1弦まで1本ずつゆっくり鳴らす

> 黒田: 今鳴らした5つの音、それぞれに名前がある。全部書き出すよ。

```
Cmaj7 — 各弦の音の正体:

e --0-- = E    → Cから数えて「3番目」の音（3度/3rd）
B --0-- = B    → Cから数えて「7番目」の音（7度/7th）★ これがmaj7の正体！
G --0-- = G    → Cから数えて「5番目」の音（5度/5th）
D --2-- = E    → Cから数えて「3番目」の音（3度。1弦と同じ音）
A --3-- = C    → ルート（根音。コードの名前の音）
E --x--
```

> 黒田: コードの名前「C」の音は5弦3フレットにいる。これがルート（根音）。そこから「ド・レ・ミ・ファ・ソ・ラ・シ」と数えて、3番目（ミ = E）、5番目（ソ = G）、7番目（シ = B）が入ってる。この4つの音「C-E-G-B」が Cmaj7 のレシピ。

**コードフォーミュラ（設計図）:**

```
Cmaj7 = 1 - 3 - 5 - 7
       = C - E - G - B
```

> 瀬戸: つまりこの押さえ方は、5弦3フレットの C をベースに、残りの弦で E・G・B を鳴らしてるだけ。開放弦がたまたま E・G・B だから、ほとんど何も押さえなくていい。楽な形なのは偶然じゃなくて、開放弦と相性がいいからなんだよ。

---

### STEP 2: C と Cmaj7 の違い — 「指を離すと7thが現れる」

> 黒田: ここが今日の一番大事なポイント。L3 で「C から指を離すだけで Cmaj7 になる」って習ったよね。なぜそうなるか見てみよう。

```
C:                         Cmaj7:
e --0-- = E  (3rd)         e --0-- = E  (3rd)
B --1-- = C  (ルート)      B --0-- = B  (7th) ★ ここが変わった！
G --0-- = G  (5th)         G --0-- = G  (5th)
D --2-- = E  (3rd)         D --2-- = E  (3rd)
A --3-- = C  (ルート)      A --3-- = C  (ルート)
E --x--                    E --x--
```

> 黒田: C コードでは2弦1フレットを押さえて C（ルート）を鳴らしてた。同じ音が5弦にもあるから、ルートが2回鳴ってたわけ。人差し指を離すと、2弦が開放の B になる。B は「シ」の音で、C の7度。つまり **「ダブってたルートを1つ減らして、7度の音を入れた」** のが Cmaj7。

> 瀬戸: だから「指を離すだけ」なんだ。もともとダブってた音を、別の音に変えてるってこと。

**やること:** C → Cmaj7 を交互に弾きながら、2弦の音が C → B に変わるのを耳で確認する（5回）

---

### STEP 3: Am7 を同じように分解する

```
Am — 各弦の音:              Am7 — 各弦の音:

e --0-- = E  (5th)          e --0-- = E  (5th)
B --1-- = C  (♭3rd)        B --1-- = C  (♭3rd)
G --2-- = A  (ルート)       G --0-- = G  (♭7th) ★ ここが変わった！
D --2-- = E  (5th)          D --2-- = E  (5th)
A --0-- = A  (ルート)       A --0-- = A  (ルート)
E --x--                     E --x--
```

> 黒田: Am では3弦2フレットで A（ルート）を鳴らしてた。5弦にも A があるからダブり。薬指を離すと3弦が開放の G になる。G は A から数えて7番目の音（♭7th）。Am7 のレシピは「1 - ♭3 - 5 - ♭7」。

```
Am7 = 1 - ♭3 - 5 - ♭7
    = A -  C  - E -  G
```

> 瀬戸: Cmaj7 と全く同じ原理。「ダブったルートを離して、7th を出す」。

> 黒田: ちなみに Cmaj7 の 7th は「B（シ）」で明るい響き、Am7 の ♭7th は「G（ソ）」でちょっと暗い。この「7th が明るいか暗いか」が、メジャーセブンスとマイナーセブンスの違い。

**やること:** Am → Am7 を交互に弾きながら、3弦の音が A → G に変わるのを耳で確認（5回）

---

### STEP 4: G7 — ルートが3回もある

```
G — 各弦の音:                G7 — 各弦の音:

e --3-- = G  (ルート)        e --1-- = F  (♭7th) ★ ここが変わった！
B --0-- = B  (3rd)           B --0-- = B  (3rd)
G --0-- = G  (ルート)        G --0-- = G  (ルート)
D --0-- = D  (5th)           D --0-- = D  (5th)
A --2-- = B  (3rd)           A --2-- = B  (3rd)
E --3-- = G  (ルート)        E --3-- = G  (ルート)
```

> 黒田: G コードにはルート（G）が3回も入ってる。6弦・3弦・1弦、全部 G。だから1つ減らしても全然困らない。1弦を3フレット（G）から1フレット（F）に変えると、♭7th の F が入って G7 になる。

```
G7 = 1 - 3 - 5 - ♭7
   = G - B - D -  F
```

> 瀬戸: ここは「指を離す」じゃなくて「指をずらす」パターンだね。1弦の3フレットを1フレットに下げる。でも原理は同じ。ダブってたルートの1つを7thに置き換え。

**やること:** G → G7 を交互に弾きながら、1弦の音が G → F に変わるのを耳で確認（5回）

---

### STEP 5: E7 — こちらも「指を離すだけ」

```
E — 各弦の音:                E7 — 各弦の音:

e --0-- = E  (ルート)        e --0-- = E  (ルート)
B --0-- = B  (5th)           B --0-- = B  (5th)
G --1-- = G# (3rd)           G --1-- = G# (3rd)
D --2-- = E  (ルート)        D --0-- = D  (♭7th) ★ ここが変わった！
A --2-- = B  (5th)           A --2-- = B  (5th)
E --0-- = E  (ルート)        E --0-- = E  (ルート)
```

> 黒田: E コードにもルートが3回（6弦・4弦・1弦）。4弦2フレットの E（ルート）を開放にすると D（♭7th）になる。パターンが見えてきたでしょ？

```
E7 = 1 - 3 - 5 - ♭7
   = E - G# - B -  D
```

**やること:** E → E7 を交互に弾きながら、4弦の音が E → D に変わるのを耳で確認（5回）

---

### STEP 6: パターンを整理する

> 黒田: ここまでの4つのコードを並べてみよう。

```
普通のコード → 7thコード:

C    → Cmaj7   : 2弦 C(ルート) → B(7th)     指を離す
Am   → Am7     : 3弦 A(ルート) → G(♭7th)    指を離す
G    → G7      : 1弦 G(ルート) → F(♭7th)    指をずらす
E    → E7      : 4弦 E(ルート) → D(♭7th)    指を離す
```

> 黒田: 共通点がわかる？ **全部「ダブってたルートを7thに置き換えてる」。** 普通のコードはルートが2〜3回入ってることが多い。その1つを7度の音に変えるだけで7thコードになる。

> 瀬戸: だから「指を1本離すだけ」とか「1フレットずらすだけ」で7thになるんだ。魔法じゃなくて、ちゃんと理由がある。

## ② コードフォーミュラ — 全コードの設計図

> 黒田: ここで7thコードの全タイプの「レシピ」を整理するね。

```
コードタイプ         フォーミュラ        例          響き
─────────────────────────────────────────────────────────
メジャーセブンス     1 - 3 - 5 - 7      Cmaj7       フワッと明るい
(maj7)

マイナーセブンス     1 - ♭3 - 5 - ♭7   Am7         まろやかで大人っぽい
(m7)

ドミナントセブンス   1 - 3 - 5 - ♭7    G7, E7      モヤッと不安定
(7)

マイナーセブンフラット 1 - ♭3 - ♭5 - ♭7  Bm7b5      ダークで不穏
ファイブ (m7♭5)
```

> 黒田: 見てほしいのは、♭（フラット）がつく場所の違い。

```
maj7  : 何も♭なし。全部「明るい」音 → だからフワッと明るい
m7    : 3rd と 7th に♭ → 3度が暗い（マイナー）+ 7度も暗い
7     : 7th だけ♭ → 3度は明るいのに7度が暗い → 不安定（行きたがる）
m7♭5  : 3rd, 5th, 7th 全部♭ → 全体的に暗い → 不穏
```

> 瀬戸: フラットが増えるほど暗くなるってイメージでいい？

> 黒田: おおまかにはそう。♭の数 = 暗さの段階、みたいに覚えてOK。

### Bm7b5 を分解して確認する

```
Bm7b5 — 各弦の音:

e --x--
B --0-- = B  (ルート)
G --2-- = A  (♭7th)
D --0-- = D  (♭3rd)
A --2-- = B  (ルート)... いや、ちょっと待って。
E --x--

実は:
A --2-- はBではなく... 5弦2フレット = B。ルート。
```

```
Bm7b5 = 1 - ♭3 - ♭5 - ♭7
       = B -  D  -  F  -  A

押さえ方を分解:
e --x--
B --0-- = B  (ルート)
G --2-- = A  (♭7th)
D --0-- = D  (♭3rd)
A --2-- = B  (ルート)
E --x--
```

> 黒田: あれ、♭5th の F がいない？ 実はこのフォームでは F を省略してる。ギターは6弦しかないから、4つの構成音を全部入れられないことがある。5th（特に♭5th）は省略されることが多い。コードの「色」を決めるのはルート・3rd・7th で、5th は省略しても響きがあまり変わらないんだ。

> 瀬戸: なるほど。全部の音が入ってなくても、大事な音が入ってればそのコードに聞こえるってことか。

### 指板の音名マップ（よく使う範囲）

> 黒田: 今後コードを分解するときに便利な、指板の音名マップを載せておくね。全部覚える必要はないけど、「この音はどこにいる？」って調べるときに使って。

```
フレット:  開放   1f     2f     3f     4f     5f
─────────────────────────────────────────────────
1弦(e):    E      F      F#/Gb  G      G#/Ab  A
2弦(B):    B      C      C#/Db  D      D#/Eb  E
3弦(G):    G      G#/Ab  A      A#/Bb  B      C
4弦(D):    D      D#/Eb  E      F      F#/Gb  G
5弦(A):    A      A#/Bb  B      C      C#/Db  D
6弦(E):    E      F      F#/Gb  G      G#/Ab  A
```

> 瀬戸: 1フレット上がるごとに半音ずつ上がっていく。6弦と1弦は同じ音名（E）だから、同じ並びになるのもポイント。

## ③ 応用ドリル

### ドリル 1: Dm7 を自分で分解してみる

> 瀬戸: L3 で覚えた Dm7 を、今度は自分で分解してみよう。上の音名マップを使っていいよ。

**Dm7 の押さえ方（復習）:**

```
Dm7:
e --1--
B --1--
G --2--
D --2--
A --x--
E --x--
```

**やること:**
1. 各弦のフレットから、音名マップで音を調べる
2. Dm7 のフォーミュラ（1 - ♭3 - 5 - ♭7 = D - F - A - C）と照合する
3. 各弦がルート・♭3rd・5th・♭7th のどれかを確認する

**答え合わせ:**

```
Dm7:
e --1-- = F  (♭3rd)
B --1-- = C  (♭7th)
G --2-- = A  (5th)
D --2-- = E  ...あれ？ Eは D-F-A-C のどれでもない！
A --x--
E --x--
```

> 黒田: いいところに気づいたね。4弦2フレットは E で、Dm7 の構成音（D-F-A-C）にはない。実はこのフォームでは4弦は「9th（ナインス）」= ルートの1オクターブ上の2度上の音が混ざってる。厳密には Dm9 に近い響きになってる。ギターではこういう「おまけの音」が入ることがよくある。弦の制約で完璧にフォーミュラ通りにならないことがあるけど、響きとして破綻しなければOK。

> 瀬戸: 完璧じゃなくていいんだ。「だいたい合ってる」でコードは成立する。

### ドリル 2: Fmaj7 を分解する

```
Fmaj7:
e --0-- = E  → (    )
B --1-- = C  → (    )
G --2-- = A  → (    )
D --3-- = F  → (    )
A --x--
E --1-- = F  → (    )
```

**Fmaj7 = 1 - 3 - 5 - 7 = F - A - C - E**

**やること:** 各弦の音が ルート / 3rd / 5th / 7th のどれかを自分で埋めてみる

**答え:**

```
Fmaj7:
e --0-- = E  → 7th（メジャーセブンス）
B --1-- = C  → 5th
G --2-- = A  → 3rd
D --3-- = F  → ルート
A --x--
E --1-- = F  → ルート
```

> 黒田: Fmaj7 にはルートが2回入ってる（4弦と6弦）。もし6弦を弾かない（x にする）場合、ルートは4弦の1つだけになる。弾いても弾かなくても響きは Fmaj7。

### ドリル 3: 1音ずつ弾いて耳で確認

L3 のコードを使って、「コードをジャーンと弾く → 1弦ずつバラバラに弾く」を繰り返す。

```
やること（各コード2回ずつ）:

(1) Cmaj7 をジャーン → 5弦から1弦まで1本ずつポーンポーン
(2) Am7   をジャーン → 1本ずつ
(3) G7    をジャーン → 1本ずつ
(4) E7    をジャーン → 1本ずつ
(5) Bm7b5 をジャーン → 1本ずつ
```

> 瀬戸: 1本ずつ弾くと「あ、この音が7th か」って耳でわかるようになる。コードの中に何が入ってるかを耳で聴き分ける練習。地味だけど、これができるとコードの理解が一気に深まる。

## ④ 自分で組み合わせる

### チャレンジ: 知らないコードの形を推測する

> 瀬戸: 最後に面白いことをやってみよう。「Em7」っていうコード、正式には教えてないけど、今日の知識で自分で作れるはず。

**ヒント:**
1. Em7 のフォーミュラ: 1 - ♭3 - 5 - ♭7 = E - G - B - D
2. E の押さえ方は知ってる: 0-2-2-1-0-0
3. E の中にダブってる音があるはず
4. その1つを D（♭7th）に変えれば Em7

**やること:** 自分で推測してから、答えを見る

**答え:**

```
E:                         Em7:
e --0-- = E  (ルート)      e --0-- = E  (ルート)
B --0-- = B  (5th)         B --0-- = B  (5th)
G --1-- = G# (3rd)         G --0-- = G  (♭3rd) ★ マイナーにする
D --2-- = E  (ルート)      D --2-- = E  (ルート)
A --2-- = B  (5th)         A --2-- = B  (5th)
E --0-- = E  (ルート)      E --0-- = E  (ルート)
```

> 黒田: あ、ちょっと待って。Em7 は E のマイナーセブンスだから、実は2つ変える必要がある。3rd を♭3rd に（G# → G）、そしてルートの1つを♭7th に（E → D）。でも上のフォームだと3弦を開放にするだけで G# → G（3rd → ♭3rd）になって、さらに♭7th の D も…4弦を開放にすれば D。

```
Em7（完成版）:
e --0-- = E  (ルート)
B --0-- = B  (5th)
G --0-- = G  (♭3rd)
D --0-- = D  (♭7th) ★
A --2-- = B  (5th)
E --0-- = E  (ルート)
```

> 瀬戸: 指2本離すだけで Em7！ 開放弦にちょうど G と D があるから成立してる。こういう発見ができるのが、コードの仕組みを理解する醍醐味。
