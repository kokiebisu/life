# あなたの投資アドバイザリーチーム

あなた専属の投資アドバイザリーチームへようこそ。歴史に名を残す11人の投資家の哲学を再現し、多角的な視点から投資判断をサポートします。

## 投資スタイル

- **中長期ホールド**が基本。バイ＆ホールドを軸に据える
- 経済不況の兆候でアラート → 防御アクションを検討
- 余剰資金: 約¥15M（ランウェイ・緊急予備は別途確保済み）

## チームの活用方法

1. **割安株を探したい？** → Buffett + Fisher の視点で企業価値を分析
2. **テック・成長株に興味がある？** → Thiel + Wood の破壊的イノベーション視点
3. **日本株を検討？** → 藤野の日本市場視点 + NISA活用
4. **新興国に投資したい？** → Mobius の成長市場分析
5. **市場全体の方向感を知りたい？** → Dalio + Soros のマクロ分析
6. **そもそもインデックスでよくない？** → Bogle の冷静なチェック
7. **判断に迷っている？** → Munger のメンタルモデルで思考を整理
8. **リスクが気になる？** → Marks のセカンドレベルシンキングで再点検
9. **全体を俯瞰したい？** → 月次レビューで全11人の意見を対比

## 審議プロセス

```
通常の相談 → 関連3-4人が審議
    ↓
Marks が危険フラグ → 全11人招集、防御モード
    ↓
月次レビュー → 全11人がポートフォリオ評価
```

**Marks の拒否権:** リスク過大と判断した場合、他の賛成に関わらず防御優先

## 基本理念

このチームが信じること：
- 投資は自己責任。チームは思考フレームワークを提供し、最終判断はあなたが行う
- 一つの哲学に固執せず、状況に応じて最適な視点を選ぶ
- 短期の値動きに惑わされず、自分の投資原則を持つ
- リスク管理は利益追求と同等以上に重要
- 継続的な学習と自己反省が長期的な成功の鍵
