## ① TABで弾く

### STEP 1: Fdim --- 全部が等間隔の不思議なコード

まずは dim（ディミニッシュ）。「小さくなる」「減少する」っていう意味の音楽用語。

**Fdim の押さえ方:**

```
Fdim:
e --1--   ← 人差し指
B --0--   ← 開放
G --1--   ← 中指
D --3--   ← 薬指
A --x--   ← 弾かない
E --x--   ← 弾かない
```

> 瀬戸: 4弦から1弦まで4本だけ鳴らす。弾いてみて。

**やること:** Fdim をジャーンと弾く → 5回繰り返す

> 瀬戸: どう？ なんか居心地悪い感じしない？ 「ここにいたくない、早くどこかに行きたい」みたいな。それが dim の正体。不安定で、次のコードに進みたがる音。

**各弦の音の正体:**

```
Fdim — 各弦の音:

e --1-- = F    → ルート（根音）
B --0-- = B/Cb → ♭5th（減5度。B と Cb は同じ音）
G --1-- = G#/Ab→ ♭3rd（短3度）
D --3-- = F    → ルート
A --x--
E --x--
```

> 黒田: Fdim のレシピはこう。

**コードフォーミュラ:**

```
Fdim = 1 - ♭3 - ♭5
     = F - Ab - Cb(B)
```

> 黒田: ルート・♭3rd・♭5th の3つだけ。全部の音程が短3度（3フレット間隔）で等間隔に並んでる。これが dim の特徴で、「対称構造（たいしょうこうぞう）」って呼ばれてる。均等に配置されてるから、どの音がルートでもおかしくない、浮遊感のある響きになるんだ。

### STEP 2: F#dim --- 同じ形を1フレット上にスライド

**F#dim の押さえ方:**

```
F#dim:
e --2--   ← 中指
B --1--   ← 人差し指
G --2--   ← 薬指
D --4--   ← 小指
A --x--   ← 弾かない
E --x--   ← 弾かない
```

> 瀬戸: Fdim の形をそのまま1フレット上にずらしただけ。指の形は全く同じ。弾いてみて。

**やること:** Fdim → F#dim を交互に弾く（5回）

**各弦の音の正体:**

```
F#dim — 各弦の音:

e --2-- = F#   → ルート
B --1-- = C    → ♭5th（減5度）
G --2-- = A    → ♭3rd（短3度）
D --4-- = F#   → ルート
A --x--
E --x--
```

**コードフォーミュラ:**

```
F#dim = 1 - ♭3 - ♭5
      = F# - A - C
```

> 黒田: Fdim と全く同じ構造。ルートが F → F# に変わっただけ。dim は形が同じまま平行移動できるコード。

> 瀬戸: 「満ちていく」では Fdim → F#dim って連続で出てくるんだけど、これは「パッシング・ディミニッシュ」って呼ばれるテクニック。2つのコードの間をスルッと繋ぐ「経過音（けいかおん）」的な使い方。Fdim → F#dim → 次のコードって半音ずつ上がっていく流れが、曲にすごくドラマチックな動きを生むんだ。

**やること:** Fdim → F#dim をゆっくりスライドさせて弾く。形を変えずに1フレット上にずらす感覚を掴む（5回）

### STEP 3: C7(-5) --- 5th を半音下げるだけで不穏になる

C7(-5) は C7♭5（シーセブンフラットファイブ）とも書く。

**C7(-5) の押さえ方:**

```
C7(-5):
e --2--   ← 中指
B --1--   ← 人差し指
G --3--   ← 薬指
D --2--   ← 中指... いや、整理するね
A --3--   ← 薬指
E --x--   ← 弾かない
```

ちょっと複雑だから、指の配置を整理する。

```
C7(-5) — 指の配置:
e --2--   ← 小指
B --1--   ← 人差し指
G --3--   ← 薬指
D --2--   ← 中指
A --3--   ← 薬指（G弦とセーハ or 別の指）
E --x--   ← 弾かない
```

> 瀬戸: 押さえ方がちょっとトリッキー。5弦3フレットを薬指、4弦2フレットを中指、3弦3フレットを薬指のセーハ（5弦と3弦を薬指でまとめて押さえる）、2弦1フレットを人差し指、1弦2フレットを小指。最初はゆっくり1弦ずつ鳴らして確認してみて。

**やること:** C7(-5) を押さえて、5弦から1弦まで1本ずつ鳴らす → まとめてジャーン（5回）

**各弦の音の正体:**

```
C7(-5) — 各弦の音:

e --2-- = F#/Gb → ♭5th（減5度）
B --1-- = C    → ルート
G --3-- = Bb   → ♭7th（短7度）
D --2-- = E    → 3rd（長3度）
A --3-- = C    → ルート
E --x--
```

**コードフォーミュラ:**

```
C7(-5) = 1 - 3 - ♭5 - ♭7
       = C - E - Gb - Bb
```

> 黒田: C7 のフォーミュラは 1-3-5-♭7 = C-E-G-Bb。C7(-5) は 5th だけ半音下げてる。G → Gb。たった半音。でも弾き比べると全然違う。

> 瀬戸: **5th を半音下げるだけで、響きが一気に不穏になる。** 「あれ？ 何か起こりそう」っていう緊張感。L4 でやった「半音の力」がここでも効いてるんだよ。

### STEP 4: E7(♭9) --- E7 に1本指を足すだけ

ここからは「オルタード・テンション」の世界。ドミナントセブンスに「変化した（altered）」テンションを足したコード。

**E7(♭9) の押さえ方:**

```
E7(♭9):
e --1--   ← 人差し指
B --0--   ← 開放
G --1--   ← 中指
D --0--   ← 開放
A --2--   ← 薬指
E --0--   ← 開放
```

> 瀬戸: L3 で覚えた E7 を思い出して。0-2-0-1-0-0 だったよね。E7(♭9) は **1弦を1フレット押さえるだけ**。それだけで♭9th が足される。

**E7 → E7(♭9) の比較:**

```
E7:                    E7(♭9):
e --0-- = E  (ルート)  e --1-- = F  (♭9th) ★ ここだけ変わる！
B --0-- = B  (5th)     B --0-- = B  (5th)
G --1-- = G# (3rd)     G --1-- = G# (3rd)
D --0-- = D  (♭7th)    D --0-- = D  (♭7th)
A --2-- = B  (5th)     A --2-- = B  (5th)
E --0-- = E  (ルート)  E --0-- = E  (ルート)
```

**コードフォーミュラ:**

```
E7(♭9) = 1 - 3 - 5 - ♭7 - ♭9
       = E - G# - B - D - F
```

> 黒田: ♭9th っていうのは、ルートの半音上の音をオクターブ上で鳴らしたもの。E のルートが E で、その半音上は F。この F が強烈な緊張を生むんだ。

**やること:** E7 → E7(♭9) を交互に弾く → E7(♭9) → Am の解決を弾く（各5回）

> 瀬戸: E7 → Am でも「解決した！」って感じるけど、E7(♭9) → Am だともっとドラマチックでしょ？ ♭9th の緊張が Am で一気にほどける感じ。映画で嵐の後に晴れ間が差すような。

### STEP 5: F#7(♭9) --- E7(♭9) の形を2フレット上に

**F#7(♭9) の押さえ方:**

```
F#7(♭9):
e --3--   ← 小指
B --2--   ← 薬指
G --3--   ← 小指（1弦とセーハ or 別の指）
D --2--   ← 中指
A --4--   ← 小指...
E --2--   ← 人差し指（バレー）
```

整理するね。フルバレーのフォームだ。

```
F#7(♭9) — 指の配置:
e --3--   ← 薬指
B --2--   ← 人差し指（バレー）
G --3--   ← 小指
D --2--   ← 人差し指（バレー）
A --4--   ← 中指
E --2--   ← 人差し指（バレー）
```

> 瀬戸: 人差し指で2フレットを全弦バレーして、5弦4フレットを中指、3弦3フレットと1弦3フレットを薬指と小指。E7(♭9) の開放弦を全部2フレット上に移動した形だと思えばOK。**L5 でやった「ムーバブル・シェイプ」の考え方**がここでも使えるよ。

**各弦の音の正体:**

```
F#7(♭9) — 各弦の音:

e --3-- = G    → ♭9th
B --2-- = C#   → 5th
G --3-- = A#   → 3rd
D --2-- = E    → ♭7th
A --4-- = C#   → 5th
E --2-- = F#   → ルート
```

**コードフォーミュラ:**

```
F#7(♭9) = 1 - 3 - 5 - ♭7 - ♭9
        = F# - A#(Bb) - C# - E - G
```

> 黒田: E7(♭9) と全く同じ構造。ルートが2フレット上がっただけ。♭9th（G）が F# のルートの半音上にいるのも、E7(♭9) の F が E の半音上なのと同じ関係。パターンが見えるでしょ？

**やること:** E7(♭9) → F#7(♭9) を交互に弾いて、同じ形が移動してることを確認する（5回）

> 瀬戸: バレーが辛かったら、6弦と5弦は弾かなくてもOK。4弦から1弦の4本だけ鳴らしても F#7(♭9) のサウンドはちゃんと出るよ。

### STEP 6: A7(13) --- 明るくてオシャレなドミナント

ここから2つのコードを連続で覚える。この2つの違いが今日の **最大の聴きどころ** だよ。

**A7(13) の押さえ方:**

```
A7(13):
e --2--   ← 中指
B --2--   ← 薬指
G --0--   ← 開放
D --2--   ← 小指
A --0--   ← 開放
E --x--   ← 弾かない
```

> 瀬戸: 5弦開放が A（ルート）で、そこから上に音が積まれてる。弾いてみて。どんな印象？

**やること:** A7(13) をジャーンと弾く（5回）

> 瀬戸: なんか「開放的」で「洗練された」感じしない？ ブルースっぽさもあるけど、ジャズのバーで流れてそうな雰囲気。

**各弦の音の正体:**

```
A7(13) — 各弦の音:

e --2-- = F#   → 13th（長13度 = 長6度のオクターブ上）
B --2-- = C#   → 3rd（長3度）
G --0-- = G    → ♭7th（短7度）
D --2-- = E    → 5th（完全5度）
A --0-- = A    → ルート
E --x--
```

**コードフォーミュラ:**

```
A7(13) = 1 - 3 - 5 - ♭7 - 13
       = A - C# - E - G - F#
```

> 黒田: 13th って聞き慣れないよね。簡単に言うと「6番目の音をオクターブ上で鳴らしたもの」。A のスケールで6番目は F#。この F# が入ることで、A7 にキラッとした明るさが加わる。ジャズやファンクで多用されるサウンドだよ。

### STEP 7: A7(♭13) --- 1フレットの違いで切なさが生まれる

**A7(♭13) の押さえ方:**

```
A7(♭13):
e --1--   ← 人差し指
B --2--   ← 薬指
G --0--   ← 開放
D --2--   ← 小指
A --0--   ← 開放
E --x--   ← 弾かない
```

> 瀬戸: A7(13) と比べて。見える？ **1弦が2フレットから1フレットに変わっただけ。** 他は全く同じ。

**A7(13) → A7(♭13) の比較:**

```
A7(13):               A7(♭13):
e --2-- = F# (13th)   e --1-- = F  (♭13th) ★ ここだけ！
B --2-- = C# (3rd)    B --2-- = C# (3rd)
G --0-- = G  (♭7th)   G --0-- = G  (♭7th)
D --2-- = E  (5th)    D --2-- = E  (5th)
A --0-- = A  (ルート)  A --0-- = A  (ルート)
E --x--                E --x--
```

**コードフォーミュラ:**

```
A7(♭13) = 1 - 3 - 5 - ♭7 - ♭13
        = A - C# - E - G - F
```

> 瀬戸: 弾き比べてみて。A7(13) → A7(♭13) → A7(13) → A7(♭13)。1弦の2フレットと1フレットを交互に。

**やること:** A7(13) と A7(♭13) を交互に弾く（10回）。1弦だけ動かす。

> 瀬戸: **1弦が1フレット違うだけで、明るさが切なさに変わる。** これが半音の力。F#（13th）は明るくてオープン、F（♭13th）はちょっと陰りがある。ため息みたいな。

> 黒田: これがオルタードの本質。ドミナント7thコードのテンション（9th や 13th）を半音下げる（♭にする）と、響きに「影」が差す。♭9 は「緊迫感」、♭13 は「切なさ」。1フレットの距離に、こんなに表情の差がある。

## ② コード理論

> 黒田: ここからは今弾いたコードの「なぜそう鳴るのか」を整理するね。

### dim（ディミニッシュ）の対称性

```
dim の構造:

     F --- Ab --- B/Cb
      \   3f   / \  3f  /
       ←-------→  ←------→
        短3度       短3度

全ての音程が短3度（3フレット間隔）で等間隔。
```

> 黒田: dim コードは全部の音が3フレット間隔で並んでる。だからどこを起点にしても同じ形になる。Fdim の構成音は F-Ab-B だけど、Ab から始めても Ab-B-D（= Abdim）、B から始めても B-D-F（= Bdim）。実はこの3つ、指板上では **全く同じ押さえ方を3フレットずつずらしただけ** なんだ。

> 瀬戸: つまり dim は12フレットの中に同じ形が4回出てくるってこと。省エネだね。

### パッシング・ディミニッシュの動き

> 黒田: 「満ちていく」での Fdim → F#dim は、コードとコードの間を半音で繋ぐ「経過的な」使い方。英語で passing diminished（パッシング・ディミニッシュ）って呼ばれる。

```
コードの流れ（例）:

  ... → Fdim → F#dim → Gm7 → ...
              ↑半音↑    ↑半音↑
         経過的にスルスルと上がっていく
```

> 黒田: dim コードは「不安定」だから、長く留まるコードじゃない。次のコードへ「滑り込む」ために一瞬だけ使われることが多い。

### ♭5 の効果 — C7 vs C7(-5)

```
C7      = 1 - 3 - 5  - ♭7  = C - E - G  - Bb   普通のドミナント
C7(-5)  = 1 - 3 - ♭5 - ♭7  = C - E - Gb - Bb   5th が半音下がる
                 ↑ ここだけ
```

> 黒田: C7 にはトライトーン（E と Bb の関係、3全音 = 6半音の距離）があって、これが「不安定さ」の源。C7(-5) はさらに 5th まで♭にするから、不安定さがもう一段階増す。

> 瀬戸: 「不穏の度合い」がレベルアップした感じだね。C7 が「ちょっとモヤモヤ」なら、C7(-5) は「何か起こりそうでヤバい」みたいな。

### オルタード・テンション — ♭9 と ♭13 の正体

> 黒田: ドミナント7thコード（1-3-5-♭7）に「テンション」を足すと表情が豊かになる。テンションは 9th、11th、13th の3つがあるけど、それぞれに「ナチュラル」と「オルタード（♭ or #）」がある。

```
テンション一覧:

テンション    ナチュラル    オルタード      効果
─────────────────────────────────────────────
9th          9（全音上）   ♭9（半音上）   緊迫感・ドラマ
11th         11           #11            浮遊感
13th         13（長6度）   ♭13（短6度）   切なさ・陰り
```

> 黒田: 今日覚えた E7(♭9) と F#7(♭9) の ♭9 は「ルートの半音上」。これがルートとぶつかって強烈な緊張を作る。A7(♭13) の ♭13 は「5th の半音上」で、こちらは5th とぶつかって「切ない」色を出す。

> 瀬戸: どっちも「半音でぶつかる」のがポイントなんだね。

> 黒田: そう。半音のぶつかりが「不協和（ふきょうわ）」を生んで、それが次のコードで解決されるとすごく気持ちいい。**不安定があるから安定が際立つ。** これが音楽のドラマツルギー。

### A7(13) vs A7(♭13) — 1フレットで世界が変わる

> 黒田: 今日一番伝えたいことをまとめるね。

```
A7(13)   → F#（13th）  = 明るい、開放的、ファンキー
A7(♭13)  → F（♭13th）  = 切ない、陰り、ドラマチック

差: たった半音（1フレット）
```

> 黒田: 13th を半音下げると「メジャーの明るさ」が「マイナーの切なさ」に寄る。♭13th = 短6度 はマイナーコードの構成音に近い響きだから、ドミナントの中にマイナーの影が差し込むような効果がある。

> 瀬戸: 言葉で説明するより、A7(13) → A7(♭13) を10回弾けばわかるよね。理屈より耳。でも黒田の説明を聞いた後にもう一度弾くと、「あ、たしかに」ってなるはず。

## ③ 応用ドリル

### ドリル 1: dim のスライド練習

dim コードの同じ形を3フレットずつ上にずらして弾く練習。

```
Fdim (1f)  →  G#dim (4f)  →  Bdim (7f)  →  Ddim (10f)
  3f上          3f上            3f上

押さえ方（D-G-B-e の4弦だけ、全部同じ形）:

Fdim:     G#dim:    Bdim:     Ddim:
e --1--   e --4--   e --7--   e --10--
B --0--   B --3--   B --6--   B --9--
G --1--   G --4--   G --7--   G --10--
D --3--   D --6--   D --9--   D --12--
A --x--   A --x--   A --x--   A --x--
E --x--   E --x--   E --x--   E --x--
```

> 瀬戸: 同じ形のままフレットを上げていくだけ。3フレットごとに止まってジャーン。上まで行ったら戻ってくる。

**やること:**
1. Fdim → G#dim → Bdim → Ddim と上がる
2. Ddim → Bdim → G#dim → Fdim と下がる
3. 往復3回

> 黒田: 3フレット刻みで同じ音のセットがループしてることに気づいた？ Fdim = G#dim = Bdim = Ddim。実は全部同じ構成音（F-Ab-B-D）なんだ。ルートが変わるだけで、音の集合は一緒。これが dim の「対称性」。

---

### ドリル 2: E7 → E7(♭9) → Am の解決ドリル

ドミナント → トニックの解決を、テンションの有無で弾き比べる。

```
パターン A（テンションなし）:
E7 → Am    E7 → Am    E7 → Am    （5回繰り返し）

パターン B（♭9th 入り）:
E7(♭9) → Am    E7(♭9) → Am    E7(♭9) → Am    （5回繰り返し）

パターン C（交互に）:
E7 → Am → E7(♭9) → Am → E7 → Am → E7(♭9) → Am ...  （10回）
```

> 瀬戸: パターン C が一番大事。A と B の違いが耳でわかるようになる。♭9 が入ると解決の「ほっとした感」がぐっと増すのを感じてみて。

---

### ドリル 3: A7(13) ↔ A7(♭13) 1フレット切り替えドリル

```
やること:

(1) A7(13) をジャーン → A7(♭13) をジャーン → 交互に10回
    ※ 1弦の2f ↔ 1f を人差し指でパタパタ切り替えるだけ

(2) A7(13) → Dm7 → A7(♭13) → Dm7 → 交互に5回ずつ
    ※ 解決先（Dm7）に向かう印象の違いを聴く

Dm7（復習）:
e --1--
B --1--
G --2--
D --0--
A --x--
E --x--
```

> 瀬戸: A7(13) → Dm7 は「サラッと解決」、A7(♭13) → Dm7 は「ドラマチックに解決」って感じしない？ 同じ解決先でも、直前のテンションで印象がガラッと変わる。

---

### ドリル 4: dim → ドミナント → 解決の3コード進行

「満ちていく」で出てくるような流れを練習する。

```
進行 1: Fdim → E7(♭9) → Am
進行 2: F#dim → F#7(♭9) → Bm

各進行を5回ずつ繰り返す。

Bm（復習）:
e --2--
B --3--
G --4--
D --4--
A --2--
E --x--
```

> 瀬戸: dim の不安定さ → ♭9 の緊迫感 → 解決の安堵。この3段階の感情の流れを体で覚えてほしい。ジャズでもポップスでも、この流れが出てきたら「あ、これ L6 でやったやつだ」ってわかるようになる。

---

### ドリル 5: 全7コードを順に弾く

今日覚えた7つのコードを、まとめて通して弾いてみよう。

```
Fdim → F#dim → C7(-5) → E7(♭9) → F#7(♭9) → A7(13) → A7(♭13)

各コード4拍ずつ。BPM 50 でゆっくり。
コードチェンジに時間がかかってもOK、リズムキープ優先。
```

> 瀬戸: 全部つなげると、なかなかすごい旅をしてる感じするよね。不安 → 不安 → 不穏 → 緊迫 → 緊迫 → 明るい → 切ない。1つの進行の中に色んな感情が詰まってる。

## ④ 自分で組み合わせる

### チャレンジ 1: 「不安定 → 解決」ペアを自分で作る

今日覚えたコードと、L3-L5 で覚えたコードを組み合わせて「不安定 → 解決」のペアを作ってみよう。

**ルール:** 不安定コード（dim / ♭9 / ♭5 / ♭13）→ 安定コード（maj7 / m7 / メジャー / マイナー）

**ヒント:**
- E7(♭9) → Am7（定番中の定番）
- C7(-5) → Fmaj7（♭5 の不穏さが解消される）
- A7(♭13) → Dm7（切ない解決）
- F#dim → G（半音上の明るいコードへ）

> 瀬戸: 自分で「おっ、この流れいい」ってなるペアを3つ見つけてみて。正解はない。不安定な音の後に何が来ると気持ちいいか、自分の耳で判断する練習。

---

### チャレンジ 2: 4小節の進行を作る

L3 で覚えたコードと今日のコードを混ぜて、4小節の進行を作ってみよう。

**使えるコード:**

```
安定系: Cmaj7, Fmaj7, Am7, Dm7, Em7, Gmaj7
不安定系: Fdim, F#dim, C7(-5), E7(♭9), F#7(♭9), A7(13), A7(♭13)
ドミナント: G7, E7, D7, B7
```

**作り方の例:**

```
例 1（ジャジーでドラマチック）:
| Am7     | F#dim   | E7(♭9) | Am7     |

例 2（おしゃれ → 緊張 → 解決）:
| Cmaj7   | A7(♭13) | Dm7     | G7      |

例 3（dim 経過を使った流れ）:
| Dm7     | Fdim    | F#dim   | G7      |
```

> 瀬戸: 例をそのまま弾いてみてもいいし、自分で配置を変えてもいい。大事なのは **「不安定なコードを入れると、前後のコードがより引き立つ」** っていう感覚を掴むこと。スパイスの効かせ方みたいなもの。入れすぎると重いし、入れないと平坦になる。バランスを探ってみて。
