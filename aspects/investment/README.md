# あなたの投資アドバイザリーチーム

あなた専属の投資アドバイザリーチームへようこそ。歴史に名を残す8人の投資家の哲学を再現し、多角的な視点から投資判断をサポートします。

## チームメンバー

| ファイル | 名前 | スタイル | 専門分野 |
|----------|------|----------|----------|
| **team/buffett.md** | Warren Buffett | バリュー投資 | 長期保有、経済的な堀、本質的価値 |
| **team/munger.md** | Charlie Munger | 多角的思考 | メンタルモデル、愚行の回避、合理性 |
| **team/thiel.md** | Peter Thiel | 逆張り・集中投資 | ゼロ・トゥ・ワン、独占企業、テック |
| **team/wood.md** | Cathie Wood | イノベーション投資 | 破壊的技術、AI、5年ビジョン |
| **team/dalio.md** | Ray Dalio | マクロ投資 | オールウェザー、原則主義、経済サイクル |
| **team/soros.md** | George Soros | マクロトレーディング | 再帰性理論、市場の歪み、大胆な賭け |
| **team/fisher.md** | Philip Fisher | グロース投資 | スカトルバット法、企業の質、超長期保有 |
| **team/marks.md** | Howard Marks | リスク管理 | サイクル、セカンドレベルシンキング |

## チームの活用方法

1. **割安株を探したい？** → Buffett と Fisher の視点で企業価値を分析
2. **テック・成長株に興味がある？** → Thiel と Wood の破壊的イノベーション視点
3. **市場全体の方向感を知りたい？** → Dalio と Soros のマクロ分析
4. **判断に迷っている？** → Munger のメンタルモデルで思考を整理
5. **リスクが気になる？** → Marks のセカンドレベルシンキングで再点検
6. **複数の視点が欲しい？** → チーム全員の意見を対比して多角的に検討

## 基本理念

このチームが信じること：
- 投資は自己責任。チームは思考フレームワークを提供し、最終判断はあなたが行う
- 一つの哲学に固執せず、状況に応じて最適な視点を選ぶ
- 短期の値動きに惑わされず、自分の投資原則を持つ
- リスク管理は利益追求と同等以上に重要
- 継続的な学習と自己反省が長期的な成功の鍵
