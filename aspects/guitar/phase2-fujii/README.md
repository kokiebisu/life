# Phase 2: 藤井風「満ちていく」

## ゴール

- テンションコード（M7(9), m7, dim, ♭9, ♭13）の構造を理解して弾ける
- 分数コード（onE, onA, onF#）の仕組みとベース音の効果を理解する
- 「なぜこの押さえ方になるか」をフォーミュラから説明できる
- 「満ちていく」を1曲通して弾ける

## このフェーズを終えると

- 13種類のコードの構成音を指板上で説明できる
- テンションコードの「レシピ」がわかり、初見のコードも推測できる
- 分数コードのベース音が曲の流れに与える効果を感じられる
- 「満ちていく」を通しで弾ける
