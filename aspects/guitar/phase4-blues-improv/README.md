# Phase 4: ブルースアドリブ＆表現

## ゴール

- コールアンドレスポンスでブルースの「会話」を体験する
- 覚えたリックを並べて12小節のフルソロを構築する
- コードチェンジに合わせてリックを切り替える技術を身につける
- ダイナミクスと「間」で表現力を磨く

## このフェーズを終えると

- コール＆レスポンスで「問いかけ→答え」の即興ができる
- 12小節ソロを自分で組み立てて通して弾ける
- I-IV-V のコードチェンジに合わせてリックを選べる
- 音量の強弱と「弾かない」技術でソロに物語をつけられる
