# Phase 3: Modes & Scales（モードとスケール）

色彩豊かな音の選択を学ぶフェーズ。同じコード上でも、スケールによって全く違う雰囲気を出せる。

## Goals

このフェーズを終えると:
- ペンタトニックを自在に使える
- モードの「色」を感じられる
- ブルースフィーリングを表現できる
- シンプルなアドリブができる

## Topics

### 1. ペンタトニックスケール
- マイナーペンタトニック（最重要）
- メジャーペンタトニック
- 5つのポジションを繋げる
- John Mayerのペンタトニック使い

### 2. モード
- モードとは何か
- **Dorian** - マイナーだけど明るい（Neo Soulで多用）
- **Mixolydian** - メジャーだけどブルージー（John Mayer的）
- **Lydian** - 浮遊感のあるサウンド
- 各モードの特徴音（Characteristic Note）

### 3. ブルーススケール
- ブルーノート（b3, b5, b7）
- ブルースのフィーリング
- ペンタトニック + ブルーノート
- John Mayerのブルースフレーズ

### 4. スケールとコードの関係
- コードに対してどのスケールを選ぶか
- コードトーンを軸にしたフレージング
- アウトサイドノートの使い方

## Practice Approach

### アドリブ練習
1. バッキングトラックを流す
2. まずコードトーンだけで弾く
3. ペンタトニックを加える
4. モードの特徴音を意識する

### 耳コピ
- John Mayerのソロをフレーズ単位でコピー
- なぜその音を選んだのか分析

## Study Songs

- 「Belief」- John Mayer（ブルースフィーリング）
- 「Vultures」- John Mayer（Mixolydian的アプローチ）
- 「傘」- King Gnu（モード感のあるメロディ）

## Notes

スケールの発見やアドリブの気づきを記録。
